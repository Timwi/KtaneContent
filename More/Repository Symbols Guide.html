On the Subject of Repository Symbols

Pw, W, Mz, Ba, what are those?

  1. What are repository symbols?

    Repository Symbols are a way to relate module names with their 1 to 4 letter symbols. It is also an alternative way to search for the desired manuals in the search bar faster.


  2. How do you use them to search for manuals?

    3. Simply type the symbol in the search bar without modifying the checkboxes, it will always be enabled.



  3. How can I view the list of symbols?

    4. To view the list of symbols, simply go to Repository of Manual Pages > View > Periodic Table.



    There is a Google Sheet which records all symbols as well.
