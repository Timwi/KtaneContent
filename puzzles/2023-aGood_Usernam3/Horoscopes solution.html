Solution

The puzzle contains 12 images, each showing a constellation icon (from the Constellations module) and a triplet of alchemical symbols (from the Astrology module).

For each image, calculate the omen score of the Astrology triplet (using the indicated serial number). Then, use this score as an index into the name of the constellation.

Since the images are ordered by the name of the constellation, we need to reorder them. Sorting the results by the number of stars in each constellation (as given in the Constellations manual) yields a clue phrase:
