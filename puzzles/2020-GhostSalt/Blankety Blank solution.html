Blankety Blank

Solution

Each element is a partial module name, with question marks and asterisks representing unknown letters.

The flavour text hints that we should take the letters obscured by asterisks. By doing this and grouping letters from the same list of module names, we get five words.

Each of these words hints at a letter. By putting these letters together, we get the answer to the puzzle.

This puzzle was correct when it was released. However, it may be ambiguous now.
