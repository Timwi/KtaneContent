Solution

The feeder answers for this puzzle are words that contain Ice Cream names.

Each of the 8 tickets gives 4 bits of information. The first digit in the ticket number provides an ordering. The clues describe who the Ice Cream customer is based on what the feeder answer it came from was.

The second digit of the ticket number shows which column to use in the table found in the Ice Cream manual. Lastly, the two digits in the order give which two allergy numbers found in the table are going to be used, giving one last set of numbers.
