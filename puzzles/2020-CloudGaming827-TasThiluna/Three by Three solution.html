In the manual for Mystic Square, there is a table that allows you to determine a target constellation for solving the module. Finding each Mystic Square's constellation provides spaces which can then be applied to the background of each grid.

The background resembles Lights Out. "Clicking" each of the marked spaces provides a new set of lit up spaces.

The new lights fit different Minecraft tools and armor crafting recipes, which is hinted by the appearance of the smaller grid at the bottom.
