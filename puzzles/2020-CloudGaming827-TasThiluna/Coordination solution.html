Here, the flavor text becomes important. It clues at using Friendship and the edges of the Friendship table.

The first clued letter shows which letter in the Friendship table to use. The second clued letter shows what edge of the table this letter should be found on (U = Up, D = Down, L = Left, R = Right). The number is an index into the name of the pony that is next to the letter.
