Solution

This puzzle consists of several mini-puzzles and a meta, all written exclusively in Z's. The name of the module that each puzzle uses, as well as that module's flavor text, can also be found and are also written in Z's.

Part 1: Binary Puzzle

Part 2: Grid Matching

Three grid segments are present. Identifying which grid each segment belongs to yields the letters B, O, and N. The large "×2" implies that this needs to be repeated, so the answer is BONBON.

Part 3: Cryptic Cycle

Three boozleglyphs are shown, layed over each other, with additive color mixing. The letter with red coloring is the first letter, green is the second letter, and blue is the last letter.

K E Y

Part 4: Translated Morse Code

The table contains 4 columns: MORSE CODE, WORD, LANGUAGE, and INDEX. A capital Z corresponds to a dash, while a lowercase z is a dot. Each Morse code cell spells out a word from one of the variants of Morse Code, without breaks. The red Z in the index column gives a digit, based on the solution that this word would give. Use this to index into the name of the language.

Part 6: Shikaku

Several shapes from Shikaku are shown. The manual for Shikaku uses a special font to display the symbols. Translating the symbols depicted into their letters according to this font yields the solution, HEART.

Part 7: Semaphore

5 semaphore letters are shown in one table. The size of the Z's used both distinguishes letters and provides an ordering. Order them from smallest to largest.

Semaphore: W.SE NW.N SW.S W.E NE.S
Letter: S T A R E

Meta: Dreamcipher

An 8×8 grid of New York Point letters is present. 8×8 also happens to be the size of the symbol table in Dreamcipher. Taking the New York Point letters in the position of the symbols described by the mini-puzzles, in the order they were given, spells out the solution.
