To see how the 25 feeder answers can be grouped into 5 groups of 5 answers, one for each meta puzzle, refer to the solution page for “Base of Operations”. The title very weakly hints at the Elements of Harmony from Friendship, but it’s sufficient to solve the other meta puzzles and be left with this group.

Each row corresponds to a “disease” from Dr. Doctor. Each column has mangled these names in a consistent way as described below.

The bottom part suggests to perform the same mangling to the feeder answers, which are assumed to be entered in alphabetical order.
