Some Assembly Required

Solution

The provided pieces can be assembled (using Inkscape, for example) into a diagram that resembles the one in Flyswatting.

Once this connection to Flyswatting is made, an additional restriction becomes intuitive: the pieces should be arranged in such a way that the diagram has been cut up into pieces using only straight-line cuts from vertex to vertex (no cuts that stop or change direction anywhere in the middle). This arrangement is unique.

Now if the diagram from the Flyswatting manual is overlaid, the numbers 1–9 appear to label specific letters. Read those in order to obtain the solution to the puzzle.
