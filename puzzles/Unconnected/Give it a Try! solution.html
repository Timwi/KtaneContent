Solution

The hints underneath each grid hint at the modules Adjacent Letters, DetoNATO, Caesar Cipher and Alphabet.

The first table is an 8×3 grid to be solved using the rules of Adjacent Letters. Doing so reveals the binary representation of three letters in ASCII (one per row):

FAYEIRJV= T
LEJMOXGR= I
PVWCKZLU= Z

The next set of tables contain letter strings used by DetoNATO for the phrases “Wisky”, “Golph” and “Oskar”, leading to the letters WGO.

The third set of tables contain Caesar ciphered clue phrases that hint at individual letter solutions:

Placing all the letter triplets underneath each other and reading in columns reveals the solution.
