HELLHOLE

Solution

The font in this puzzle is the same font used by the Cycle modules created by SpeakingEvil. Furthermore, those modules use colors to distinguish themselves. Some of those colors are also used by Jumble Cycle but not all of them.

Hovering over the letters shows us that letters with the same color in the same row are “linked” somehow. Let’s start with the first one: in the first row, W and N are both pink. The Cycle module that uses pink lettering is Pigpen Cycle.

The W and N are in first and fourth place. Turns out there’s only one word in Pigpen Cycle that fits this pattern — WINNABLE — which leads us to the corresponding response word — FORTUNES — which has F and T in the first and fourth place. Indeed the word on the right, FROTHING, also has F and T in the same places! This confirms that we are on the right track. But it also tells us that all of the colored letters, and the words on the right, are only there to discover the mechanic; the real puzzle is in the last line, where there are no letters.

The key insights are:

  • There are eight colored buttons, so we need an 8-letter word to put there.
  • There is an 8-letter word on the page that is written in the module’s font — the puzzle title!

Now we can use the mechanics of the puzzle to reconstruct the word that should go where the question marks are.
