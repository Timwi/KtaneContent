The Plebs

Solution

Warning: This solution page has spoilers to
several of puzzles from The Puzzle Plethora

This puzzle is a throwback mash-up puzzle which refers to The Puzzle Plethora, another KTaNE Hunt. This puzzle was made for a challenge issued by CloudGaming827, where he asked puzzlers to create puzzles similar to The Hustlers, but with different hunts.

Each section uses the same mechanics as 6 different Plethora puzzles, all of which are hinted in the flavor text. Applying these mechanics provides bits of information which can then be combined to extract a solution.

NOTE: This solution page will not go into depth about how the original puzzles are solved. The Plebs requires you to know how the originals were solved, so the solution page will too.
