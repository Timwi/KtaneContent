All about the letters

Solution

Answering the questions will give you a number and 5 words/phrases begining with either N or M. These relate to the module N&Ms. The number is a ruleseed for that round, while the first letter of each answer forms a 5 letter string. The phrase "we don’t talk about K" is a reference to Tap Code (the grid doesn't contain a K). Taking the position of each 5 letter string in N&Ms with the ruleseeds and mapping it to the grid in Tap Code will spell out a word
