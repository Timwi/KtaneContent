Solution

The bulk of this puzzle requires you to make shapes for each set of instructions. These shapes are shown below. Note that not all given lines make up the intended shapes.

These shapes can be found in the X-Ray manual, with the top row corresponding to column shapes, and the bottom row corresponding to row shapes.

When you highlight these rows in the table, these digits are the ones that exist in both a corresponding row and column: 34 21 21 15 42 43

Finally, the flavor text hints at using Tap Code to convert those numbers to letters, which gives the answer.
