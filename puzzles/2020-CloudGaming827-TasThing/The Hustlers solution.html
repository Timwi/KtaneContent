This is a mash-up throwback puzzle. The puzzle is made of five mini-puzzles that are mash-ups of puzzles from the KTaNE Puzzle Hustle. Each mini-puzzle can be solved similarly to how the original puzzles were solved.

NOTE: This solution page will not go into depth about how the original puzzles are solved. The Hustlers requires you to know how the originals were solved, so the solution page will too.

Baby Don't Hurt Me + Redundancy

There are four modules given with one clickable area. When taking the first letters of these modules, three of them can be used to create a word from Alphabet (TJL). We can then extract the unused letter: C.

Easy as $#*& + Put Your Labcoat On

The names can be converted to numbers. To figure out how, look at the solution for Easy as $#*&. The second part of the equation is straight up counting whatever is given. Once the full equation is solved, looking at the element symbol corresponding to the atomic numbers spells a clue phrase: NATO REVERBERATION. This provides the second letter, ECHO.

Base of Operations + Word Qualities



Finding the pairs of symbols in the X-Ray manual under the given rule seeds provides a number (30132244355040325552132). This number, just like the one in the original puzzle, is in base 6. Converting this number to base 30 provides a name, MICHAEL CAINE. This name can then be put into the wheel and solved like a normal Word Qualities puzzle, giving the next letter: LIMA.



Abstract Art + Bearing in Mind

The three images have a flag from Maritime Flags on the right. Breaking down the four different color channels and reading the numbers in each section provides a rule seed. The letter corresponding to the flag given with each rule seed gives the phrase: FOURTH VOWEL, which is an O.
