On the Subject of Synapse Says

The importance thing to do is to make position of colour so colour is positioned colour position colour.

  • Four coloured buttons will flash.
  • Press the correct button to progress to the next stage. Complete all stages to disarm the module.
  • Repeat the sequence of colours previously pressed before pressing another button.
  • Pressing an incorrect button will reset the module back to stage 1.
  • If the module is left idle before a stage is complete, the sequence will begin to flash again and the input is reset.

Stage 1:

If the display is a 1:

If the button in the fourth position flashed fourth, press the button that flashed first.
Otherwise if the blue button flashed first, press the button in the third position.
Otherwise if the yellow button never flashes, press the button that flashed second.
Otherwise press the button in the first position.

If the display is a 2:

If the red button flashes more than once, press the button in the fourth position.
Otherwise if the button in the second position never flashes, press the button in the second position.
Otherwise if the green button flashes first, press the button that flashed third.
Otherwise press the button that flashed second.

If the display is a 3:

If all four buttons flash, press the button in the first position.
Otherwise if the button that flashed first also flashed fourth, press the button that flashed third.
Otherwise if the green button flashes exactly once, press the button in the fourth position.
Otherwise press the button that flashed fourth.

If the display is a 4:

If the red button flashes first, press the button in the second position.
Otherwise if any button flashes twice consecutively, press the button that flashed fourth.
Otherwise if the yellow button flashes third, press the button that flashed first.
Otherwise press the button in the third position.

Stage 2:

If the display is a 1:

If the button that flashed third is to the right of the button that flashed second, press the button in the position that flashed first in stage 1.
Otherwise if the button in the fourth position never flashes, press the button in the position that flashed fourth in stage 1.
Otherwise if the same button flashes both third and fourth, press the button in the first position.
Otherwise press the button that flashed fourth.

If the display is a 2:

If the button that flashed third is the same as the button that flashed fourth, press the button in the fourth position.
Otherwise if the red button never flashes, press the button in the position that flashed second in stage 1.
Otherwise if the button in the second position flashes first, press the button that flashes second.
Otherwise press the button in the position that flashed third in stage 1.

If the display is a 3:

If the yellow and blue buttons flash consecutively, press the button that flashed first.
Otherwise if the button in the first position flashes exactly once, press the button in the position that flashed fourth in stage 1.
Otherwise if the green button never flashes, press the button in the position that flashed third in stage 1.
Otherwise, press the button in the third position.

If the display is a 4:

If the yellow button flashes more than once, press the button in the position that flashed second in stage 1.
Otherwise if the button in the third position flashes fourth, press the button in the position that flashed first in stage 1.
Otherwise if the button in the second position flashes exactly once, press that button that flashed third.
Otherwise press the button in the second position.
