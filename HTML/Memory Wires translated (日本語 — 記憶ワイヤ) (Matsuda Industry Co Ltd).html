Keep Talking and Nobody Explodes Mod 記憶ワイヤ

モジュール詳細：記憶ワイヤ

記憶とは、電気回路を流れる生き血のような存在だ！いや待て、電気こそ生き血だ。記憶とは、なんともろい存在だろう。だが、爆弾が爆発してしまえばどんなものだってもろいか？
いや…

  • 正しいワイヤを全て切ると次のステージへ進む。すべてのステージが完了すると、モジュールが解除される。
  • 誤ったワイヤを切るとステージ1へ戻される。
  • 以下のリストの中から、モジュールのステージと画面に表示されている数字に対応した、ルールを満たす正しいワイヤを見つける。
  • ワイヤの位置は1～6までの番号があり、左から右に数える。

爆弾にパラレルポート、CARまたはFRKと書かれた点灯したインジケーター、2本以上のバッテリーがある場合

すべてのステージでこのルールのリストを使用する。

各ワイヤについて、以下のルールを満たした場合、切断する。
  1. そのワイヤの色はこのステージ内で1本だけである。
  2. ワイヤは素数番目のポートに接続されている。
  3. 赤色または黒いワイヤが偶数本あった位置にある。
  4. 白いワイヤではない。
    これがステージ1でない場合、直前のステージで切断されていないワイヤと同じ位置のワイヤでもある。
  5. ステージ1：青いワイヤに隣接している。
    その他のすべてのステージ：直前のステージで切断されなかったワイヤに隣接している。
  6. 黄色いワイヤか切断されたワイヤが奇数本あった位置にある。

それ以外の場合、以下の2つのリストのうちいずれかを使用する。

シリアルナンバーの最後の数字が奇数の場合

各ワイヤについて、以下のルールを満たした場合、切断する。
ステージ1:
  1. ワイヤが奇数番目で、黄色ではない。
  2. そのワイヤは、同じ色のワイヤと隣接している。
  3. その色のワイヤは、3本以上出現している。
  4. 白または黒いワイヤである。
  5. 一番左の赤いワイヤより右側にある。
  6. 青いワイヤと隣接していない。
ステージ2:
  1. 赤色または青いワイヤがちょうど1回あった位置にある。
  2. ステージ1で切断された一番左のワイヤより左側のワイヤである。
  3. 黒いワイヤが一度もなかった位置にある。
  4. 黄色または青いワイヤで、ステージ1で切断されていない位置にある。
  5. ステージ1で同じ色のワイヤがあった位置にある。
  6. ステージ1で白いワイヤがあった位置の右隣にあるワイヤである。
ステージ3:
  1. ワイヤがちょうど1本切断された位置にある。
  2. 現在黄色ではない、黄色いワイヤが今までにあった位置にある。
  3. 偶数番目である、赤色または白色のワイヤが今までにあったワイヤである。
  4. 黒色ではなく、ステージ2で切断されていない位置にある。
  5. 今までに3種類の色があった位置にある。
  6. 青いワイヤが今までに1本だけあった位置にある。
ステージ4:
  1. 赤色または黄色いワイヤで、ステージ3でワイヤを切断した位置にある。
  2. 青色または黒いワイヤで、ステージ2で切断された位置にある。
  3. 白いワイヤが今までに1本だけあった位置にある。
  4. 黒いワイヤが今までに偶数本あった位置にある。
  5. ステージ1で切断された位置の左隣にある。
  6. ちょうど2本切断された位置にある。
ステージ5:
  1. そのワイヤは、以前のステージでその位置にあった他のワイヤが使用していない色である。
  2. ステージ2で同じ色だったワイヤと隣接している。
  3. 青色ではなく、ステージ4で青いワイヤがあった位置に隣接していない。
  4. 赤または白いワイヤに隣接し、ステージ1で切断されていない。
  5. 黒いワイヤで、その位置で奇数回切断されたことがあるか、黒いワイヤではなく、その位置で偶数回切断されたことがある。
  6. 黄色いワイヤに隣接していない位置にあり、ステージ3でも黄色いワイヤに隣接していない。

シリアルナンバーの最後の数字が偶数の場合

各ワイヤについて、以下のルールに満たした場合、切断する。
ステージ1:
  1. 色の異なる2本のワイヤに隣接している。
  2. 黒いワイヤに隣接していない。
  3. 右端の黄色いワイヤより左側にある。
  4. 赤いワイヤの左隣、または青いワイヤの右隣にある。
  5. ワイヤが奇数番目で、白ではない。
  6. その色のワイヤは、ちょうど2本出現している。
ステージ2:
  1. ステージ1の状態よりも出現回数が少ない色のワイヤである。
  2. ステージ1で赤いワイヤがあった位置にある。
  3. ステージ1で切断されていない黒または白いワイヤがあった位置にある。
  4. 青色ではなく、ステージ1で切断されていない位置にある。
  5. ステージ1で黄色いワイヤがあった位置の右隣にある。
  6. ステージ1で切断されなかったワイヤがあった位置と隣接している。
ステージ3:
  1. 黄色ではなく、ステージ2で切断したワイヤがあった位置に隣接している。
  2. ステージ2で赤いワイヤがあった位置に隣接している。
  3. ワイヤがまだ切断されていない位置にある。
  4. ワイヤが青色でなかった位置にある。
  5. 黒いワイヤであるか、ステージ1で黒いワイヤがあった位置に隣接している。
  6. ステージ1で切断されていないワイヤがあった位置に隣接している。
ステージ4:
  1. 赤色または青いワイヤで、ステージ1で切断していないワイヤがあった位置にある。
  2. 偶数番目の位置の黄色いワイヤと隣接していた位置にある。
  3. ステージ3の状態よりも出現回数が少ない色のワイヤである。
  4. ワイヤは白ではなく、ステージ4で同じ色のワイヤがあった位置でもない。
  5. 黒ではなく、偶数回切断された位置のワイヤでもない。
  6. ステージ2で同じ色のワイヤがあった位置にある。
ステージ5:
  1. ステージ1の状態と同じ色だったワイヤに隣接している。
  2. 黄色いワイヤで、ステージ3で切断されたワイヤに隣接しているか、黄色いワイヤではなく、ステージ3で切断されなかったワイヤに隣接している。
  3. ステージ4では切断されなかったが、他の少なくとも2つのステージで切断された位置にある。
  4. ステージ2で白いワイヤがあった位置の左隣か、ステージ2で青いワイヤがあった位置の右隣にある。
  5. 今までに少なくとも4種類の色があった位置にある。
  6. 赤または黒いワイヤに隣接しており、少なくとも2本の切断されていないワイヤが同じ色であった位置にある。
