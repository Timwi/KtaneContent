On the Subject of the Triple Triplex Reflector CM

Create a key using the keyword on screen 1 and the Boolean Expression on screen A. Lay the resulting key out in a 3×9 grid with a hole in the center. This will be referred to as Reflector A.

Reflector B is created by doing the same steps to create Reflector A but with the keyword on screen 2 and the Boolean Expression on screen B.

Reflector C is created by doing the same steps to create Reflector A but with the keyword on screen 3 and the Boolean Expression on screen C.

For each letter of the encrypted word, do the following:

  • Take the Nth letter of the encrypted word, find it in Reflector A, and replace it with the letter that is in the same position in Reflector B.
  • Find this letter in Reflector C and replace it with the letter that is diametrically opposite.
  • Find this letter in Reflector B and replace it with the letter that is in the same position in Reflector A.
  • Convert the Nth letter of the keyword on screen 4 into ternary.
  • Shift Reflector A based on the Most Significant Digit. Shift Reflector B based on the Middle Digit. Shift Reflector C based on the Least Significant Digit.
  • Do not move the hole when shifting the reflectors.
  • Do this for each letter of the encrypted word to get a decrypted word. Treat the keyword as if it wraps around to the front of itself.
