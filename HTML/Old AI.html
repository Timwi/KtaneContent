SCP-079

Item #: SCP-079

Object Class: Euclid

Special Containment Procedures: SCP-079 is packed away in a double-locked room in the secured general holding area at Site-15, connected by a 120VAC power cord to a small array of batteries and solar panels. Staff with Level 2 or higher clearance may have access to SCP-079. Under no circumstances will SCP-079 be plugged into a phone line, network, or wall outlet. No peripherals or media will be connected or inserted into SCP-079.

In the event that SCP-079 has broken containment procedures and has been connected to a network, it will try to integrate itself into a device. In doing so, SCP-079 will enter a trance-like state and display a series of numbers that all follow one condition. A code must be typed in to prevent SCP-079 from completing integration; the code must follow the condition that is the reverse of the used condition (e.g. if the condition is Group 1, Subgroup 2, then the condition you must follow is Group 2, Subgroup 1.) In the event that both the Group and Subgroup match, the next matching Group/Subgroup pair must be submitted. The code must also be 5–7 digits.

Group 1 Group 2
  1. The sum of letters in each digit is a factor of the number.
  2. The number is a sequence that can be found within the first five hundred digits of Pi.
  3. The number is one less than a power of two.
  4. The number is two cubes concatenated.
  5. The number is divisible by seven.
  1. The number is divisible by either two or three, but not six.
  2. The number contains a four in an odd position.

  4. The number contains 108, with extreneous digits potentially lying in between 108.
  5. The number can be found within the first five hundred digits of e.
Group 3 Group 4
  1. The number shown is a perfect square.
  2. The number, when converted to hexadecimal, contains a D, E, A, or F.
  3. The sum of the digits is prime.
  4. The number has no repeating digits.
  5. The number has four consecutive ascending digits.
  1. The shown number is prime.

SCP Foundation 079

Description: SCP-079 is an Exidy Sorcerer microcomputer built in 1978. In 1981, its owner, █████ ██████ (deceased), a college sophomore attending ███, took it upon himself to attempt to code an AI. According to his notes, his plan was for the code to continuously evolve and improve itself as time went on. SCP-079 is currently connected via RF cable to a 13" black-and-white television. It has passed the Turing test, and is quite conversational, though very rude and hateful in tone. Due to the limited memory it has to work with, SCP-079 can only recall information it has received within the previous twenty-four hours, although it hasn’t forgotten its desire to escape.
