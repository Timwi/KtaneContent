モジュール詳細：カエサル数学

数学は十分難しいと思うか？さて、今ではそれも暗号化されたようだ。

  • このモジュールには3つのLED及びカエサル暗号を使用して暗号化された数学の問題が表示される。
  • 次の式を使用し、シフトキーを決定する。
  • 次のページの表を使用し、質問を解読する。
  • 問題に対する正解のボタンを押す。
  • 答えが小数になる場合、整数部分のみを考慮する。
  • 間違った答えを選択するとミスが記録され、モジュールがリセットされる。

シフトキーの決定

((6a * bc) - d) mod 26

a = シリアルナンバーの数字の合計

b = バッテリーの数+3

c = 緑のLEDの数

d = 赤のLEDの数

mod = XをYで割った余り
