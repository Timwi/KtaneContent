Overrides

If any of the following conditions are true for a wire, do the opposite of what the table tells you. Opposites do not cancel out.

  • Wire connects to a letter in a lit indicator
  • SN contains a zero AND it contains the number of wires of this colour seen so far, including this wire
    • ie. SN contains zero AND (the # of the current row being used)
    • ex. SN: 130WP5, override the 1st, 3rd, and 5th wire of each colour
  • Sum of SN digits is a multiple of 3 AND the number of total wires seen so far, including this wire, is also a multiple of 3 (ie. every third wire looked at)
    • ie. If sum of SN digits is a multiple of 3, override every 3rd wire checked
    • Note that this is cumulative across all wires seen, including wires of other colours
