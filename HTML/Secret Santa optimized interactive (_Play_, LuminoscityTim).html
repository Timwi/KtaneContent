Follow the process below to obtain the price of each present on the module:

  • Obtain the initial price from the left table.
  • T = (digital root of the current price) + (present’s color value from the right table)
    • There will always be exactly one present of each color.
  • If the current price is less than $45, subtract T from it.
    • If this makes the current price less than $10, instead add T to it.
  • If the current price is greater than or equal to $45, add T to it.
    • If this makes the current price greater than $99, instead subtract T from it.
