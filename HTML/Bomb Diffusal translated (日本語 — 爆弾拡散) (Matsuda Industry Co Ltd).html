配送先選択時の手順

コンポーネントに関する手順

バッテリー

まず、Bomb Diffusal Software Casing™ に付いているバッテリーの本数を数えます。次にバッテリーホルダーの数を加えます。それに 1 を加算し、10 でモジュロします。これがバッテリーの本数です。

インジケーター

Bomb Diffusal Software Casing™ のシリアルナンバーにある数字が 4 桁の場合、Bomb Diffusal Software Casing™ 上のインジケーターの数を送信してください。Bomb Diffusal Software Casing™ のシリアルナンバーにある数字が 3 桁の場合、Bomb Diffusal Software Casing™ の点灯しているインジケーターの数を 2 倍にして送信してください。そうでない場合は、点灯していないインジケーターの数を 2 倍にして送信してください。10 以上の場合は、その数を 10 でモジュロしてください。

ポート

Bomb Diffusal Software™モジュールには、2つのポートがあります。ポート名をポート1の名前に設定し、下表のポート2の番号をXとして、X回左に移動します。

マニュアル

デフォルトでは、マニュアルは1冊のみです。ただし、場所がアメリカ合衆国またはユーラシア大陸の場合、Bomb Diffusal Software Casing™ のポートプレート 1 枚につき 2 冊のマニュアルを追加で送付します。アフリカまたはアメリカ大陸の場合は、Bomb Diffusal Software Casing™のバッテリーホルダー1個につき 1 冊のマニュアルを追加で送付します。水星、金星、太陽の場合は、ほとんどのマニュアルが途中で燃えてしまうので、最大数のマニュアルを送付してください。最後に、Bomb Diffusal Software Casing™ のシリアル番号の下一桁が偶数であれば、 1 冊のマニュアルを追加してください。必要なマニュアルの数が入力可能な最大数よりも多い場合は、最大数のマニュアルを送信してください。

納品時の手順
