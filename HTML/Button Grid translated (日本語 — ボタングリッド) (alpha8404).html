Keep Talking and Nobody Explodes Mod ボタングリッド

モジュール詳細：ボタングリッド

「見た目ほどは難しくない…」

  • このモジュールは赤、青、黄、緑のボタンによる4×5グリッドと、モジュール上部にある5つのLEDを表示するディスプレー、及びリセットボタンで構成されている。
  • 各色はちょうど5回ずつ出現しているはずである。
  • モジュールを解除するには、各ボタンを5ステージにおいて1回ずつ押す必要がある。
  • モジュールには5つのステージがあり、現在のステージは点灯したLEDの数で示される。
  • 5つのステージそれぞれについて、特定の順序で各色のボタンを1つずつ押す必要がある。間違った順序でボタンを押すと、全て押した後にミスが記録される。
  • 同じボタンを複数回押してはならない。ボタンは一度押しても見た目は変わらない。一度押されたことのあるボタンを再度押すと、即座にミスが記録され、現在のステージの入力がリセットされる。
  • ただし、点灯したBOBインジケーター、DVI-Dポート、ちょうど1つのバッテリー、及びシリアルナンバーにCHRISという単語と共通する文字が少なくとも1つある場合は青、赤、青、黄の順番に4つのボタンを押すことで即座にモジュールを解除することが出来る。これはステージ1でのみ実行でき、2つの異なる青のボタンを押す必要がある。

ステージ1

  1. 四隅の色が全て違う場合、赤、青、黄、緑を押す。
  2. それ以外の場合、モジュールのいずれかの列に4色すべての色がある場合、緑、赤、黄、青を押す。
  3. それ以外の場合、いずれかの段に含まれていない色がある場合、赤、黄、青、緑である。
  4. それ以外の場合、緑、黄、青、赤である。

ステージ2

  1. モジュールの上半分に全ての青のボタンがある場合、青、赤、黄、緑である。
  2. それ以外の場合、モジュールの下半分に少なくとも3つの赤のボタンがある場合、赤、緑、黄、青である。
  3. それ以外の場合、緑のボタンが別の緑のボタンに隣接している場合、緑、青、赤、黄である。
  4. それ以外の場合、黄、赤、青、緑である。

ステージ3

  1. シリアルナンバーにB、TまたはNのいずれかが含まれている場合、緑、青、赤、黄である。
  2. それ以外の場合、シリアルナンバーにG、R、Dのいずれかが含まれている場合、赤、緑、青、黄である。
  3. それ以外の場合、シリアルナンバーに母音が含まれている場合、青、緑、赤、黄である。
  4. それ以外の場合、黄、赤、緑、青である。

ステージ4

  1. 読み順で最初のボタンの色が赤の場合、赤、黄、緑、青である。
  2. それ以外の場合、読み順で最初のボタンの色が緑の場合、緑、赤、青、黄である。
  3. それ以外の場合、読み順で最初のボタンの色が黄の場合、黄、赤、緑、青である。
  4. それ以外の場合、青、赤、黄、緑である。

ステージ5

  1. 読み順で最後のボタンの色が黄の場合、赤、青、緑、黄である。
  2. それ以外の場合、読み順で最後のボタンの色が緑の場合、青、黄、赤、緑である。
  3. それ以外の場合、読み順で最後のボタンの色が青の場合、緑、黄、赤、青である。
  4. それ以外の場合、黄、青、緑、赤である。
