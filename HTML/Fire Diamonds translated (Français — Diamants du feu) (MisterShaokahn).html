Keep Talking and Nobody Explodes Mod Diamants du Feu

À propos des Diamants du Feu

Aussi connu sous le nom de Carrés de Sécurité ou, pour les Britanniques, de
« Fantasmagorique Quadrilatère Pigmenté ».

VOIR L’ANNEXE 704 POUR PLUS D’INFORMATIONS SUR LES DIAMANTS DU FEU.

  • Sur ce module se trouvent quatre boutons : un rouge, un bleu, un jaune et un blanc. Les trois boutons de couleur portent un chiffre de 0 à 4, tandis que le bouton blanc est vide ou porte un symbole.
  • Ce module comporte trois étapes. Les réponses à chaque étape n’affectent pas les réponses aux autres étapes.
  • Utiliser les numéros/symboles sur les boutons du tableau 2 pour obtenir une séquence de cinq couleurs. (J = Jaune / R = Rouge / B = Bleu / Bl = Blanc)
  • Utiliser le tableau 1 ci-dessous pour modifier la séquence pour chacune des couleurs. En fonction des couleurs de la séquence et des informations de la bombe, remplacer chaque couleur du tableau 2 par celle obtenue comme résultat dans le tableau 1.
  • Entrer une séquence de couleurs incor recte entraînera une erreur. Toutefois, le module n’enregistrera pas d’erreur tant qu’une séquence complète de cinq entrées n’aura pas été effectuée.
