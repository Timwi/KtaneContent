On the Subject of ASCII Art

www

This module contains a display containing an image made of ASCII art, a numerical keypad with a query button, and four buttons with various characters on them. To solve the module, repeatedly query the module with various numbers to receive ASCII codes. Translate each received code to a character, and press the only character on the module that appears in the list.

Use the characteristics of the given image in the below table to find the starting number.

Once the starting number is found, observe the table. Query the number above, below, left, and right of the number. Also query the number in the same position of the 6×6 grids left and right of the starting number. If any of these directions are inapplicable, do not query them. Do not query the starting number.

If there are no batteries on the bomb...
