À propos de La Vis

Vous savez quoi ? Ce module ne vaut pas un clou...

  • Le module La Vis possède 4 boutons étiquetés A, B, C, et D.
  • 6 trous aux contours colorés se trouvent sous les boutons.
  • Les fonctions des boutons changent selon le trou dans lequel se trouve la vis.
  • Appuyer sur un mauvais bouton entraîne une erreur. En revanche, mettre la vis dans le mauvais trou n’entraîne pas de suite une erreur, la position de la vis peut être changée sans risque avant d’appuyer sur un bouton.
  • Les touches n’ont aucun effet sur le désamorçage ou les erreurs si une vis n’est pas insérée dans un des trous.
  • Au début de la bombe, la vis est toujours dans le trou supérieur gauche.
  • Pour résoudre une étape, vous devez retirer la vis du trou actuel, la visser dans le trou correct suivant, puis appuyer sur le bon bouton.
  • L’écran dans le coin supérieur gauche du module indique l’étape actuelle du module.
  • Une fois que trois étapes ont été résolues, le module sera désarmé.

D’abord, déterminer la couleur du trou dans lequel la vis doit aller.

  • Les contours colorés sont numérotés de 1 à 6 dans l’ordre de lecture.
  • La position du trou de la première étape correspond au nombre de piles plus au nombre d’indicateurs éteints.
  • La position du trou de la deuxième étape correspond au chiffre le plus à droite du numéro de série plus au nombre d’indicateurs allumés.
  • La position du trou de la troisième étape correspond au nombre de ports plus au nombre de supports de piles.
  • Si le trou correct est celui dans lequel la vis se trouvait déjà avant cette étape, le trou correct est en fait le trou suivant dans l’ordre numérique.
  • Si une valeur est égale ou supérieure à 7, soustraire 6 jusqu’à ce que la valeur soit comprise entre 1 et 6. Si une valeur est 0, la vraie valeur est 1.

Ensuite, déterminer quel bouton doit être appuyé en fonction des tableaux suivants :

La couleur du contour est rouge, jaune ou verte :

Si le trou est dans la ligne supérieure :

  • Si la position du trou dans la rangée est égale au nombre de porte-piles, appuyer sur le bouton en quatrième position.
  • Sinon, si la lettre A est en première ou troisième position, appuyer sur le bouton marqué C.
  • Sinon, si la bombe a un indicateur CLR, FRK ou TRN, appuyer sur le bouton en troisième position.
  • Sinon, si le trou partage la même rangée que le trou bleu, appuyer sur le bouton en première position.
  • Sinon, appuyer sur le bouton marqué B.

Si le trou est dans la ligne inférieure :

  • Si la position du trou dans la rangée est égale au nombre de types de ports, appuyer sur le bouton en deuxième position.
  • Sinon, si la position du trou dans la rangée est égale au nombre de piles, appuyer sur le bouton marqué D.
  • Sinon, si le trou n’est pas verticalement opposé au trou blanc, appuyer sur le bouton marqué A.
  • Sinon, si le trou est horizontalement adjacent au trou magenta, appuyer sur le bouton marqué C.
  • Sinon, appuyer sur le bouton en première position.

La couleur du contour est bleue, magenta ou blanche :

Si le trou est dans la ligne supérieure :

  • Si la position du trou dans la rangée est égale au nombre de types de ports, appuyer sur le bouton marqué D.
  • Sinon, si la lettre C est en deuxième ou quatrième position, appuyer sur le bouton marqué B.
  • Sinon, si la bombe a un indicateur CAR, FRQ ou SND, appuyer sur le bouton en quatrième position.
  • Sinon, si le trou partage la même rangée que le trou rouge, appuyer sur le bouton en deuxième position.
  • Sinon, appuyer sur le bouton marqué A.

Si le trou est dans la ligne inférieure :

  • Si la position du trou dans la rangée est égale au nombre de plaques de port, appuyer sur le bouton en deuxième position.
  • Sinon, si la position du trou dans la rangée est égale au nombre total d’indicateurs, appuyer sur le bouton marqué A.
  • Sinon, si le trou est horizontalement adjacent au trou jaune, appuyer sur le bouton marqué C.
  • Sinon, si le trou n’est pas verticalement opposé au trou vert, appuyer sur le bouton marqué D.
  • Sinon, appuyer sur le bouton en quatrième position.
