On the Subject of Symbstructions

Defuse these bombs, blah, blah, blah. Don’t kill us all, blah, blah, blah.

On the module is a timer, a screen, and a closed panel.
Tap the screen to start the timer and open the panel.

Once the panel is open, the screen will show a set of symbols, and the timer will begin counting down.*

Solve the puzzle by completing the actions depicted by the symbols. Rules may contradict each other, or negate themselves entirely, so read carefully before touching anything.

Performing an incorrect action (or letting the timer run out) will invoke a strike and wait for you to start a new puzzle.

Symbol definitions are depicted on the following pages.

*The default time limit is 60 seconds, but this can be changed in the mod settings.
(Set this to -1 to remove the timer altogether.)

Wires

A set of 3 to 6 wires will be present in the panel. Cut all applicable wires to solve the puzzle.

Some symbols have arguments, like a reference to certain wires or colors. Mandatory arguments are shown in parentheses (red in the diagrams), while optional arguments are shown in square brackets (gray in the diagrams).

Attachments

These symbols may be attached to an action symbol. They add an extra step and must be considered.

  • If an attachment is placed above an action symbol, only follow the action symbol if the condition applies.
  • If an attachment is placed below an action symbol, only follow the OPPOSITE of the action symbol if the condition applies.
    • (e.g. if the action symbol says to cut the third wire and the condition below the symbol applies, DO NOT cut the third at all costs.)
