Keep Talking and Nobody Explodes Mod ブラックジャック

モジュール詳細：ブラックジャック

我々はこのモジュールがギャンブルを推奨しているわけではないことを法的に立証しなければいけません。そして自己責任で明日は無いと感じながらギャンブルをやるのです。

  • 各カードをC1~C5と呼ぶ。
  • カードの合計をスコアとする(絵札=10、A=1)。
  • 勝利条件:
    • 21にする。
    • 21未満の場合、ディーラーより上にする。
    • ディーラーを22以上にする。
  • 注:C1とC2がAと10/J/Q/Kの場合は「Blackjack」を押す。

C2の特定

C1 = A♠

  • 点灯BOB:C2 = K♥
  • SN母音:C2 = 5♦
  • SN数字の合計8以上:C2 = 7♠
  • それ以外:C2 = 2♣

C1 = K♦

  • CLR, CAR, FRQ, SIG, NSA, MSA, TRN, BOB, FRKのいずれか:C2 = Q♥
  • 単1が2つ以上:C2 = 9♠
  • シリアルポート:C2 = 3♦
  • それ以外:C2 = 4♣

C1 = 2♥

  • ポート > バッテリー:C2 = A♦
  • NSA:C2 = 3♥
  • バッテリー無し:C2 = 7♣
  • それ以外:C2 = 4♠

C1 = 10♣

  • SNに「CASINO」の文字:C2 = 5♣
  • 単3が4つ以上:C2 = 3♥
  • パラレルポート:C2 = 6♦
  • それ以外:C2 = A♠
