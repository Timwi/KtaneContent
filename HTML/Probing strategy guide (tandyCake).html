On the Subject of Probing Probing

Not that kind of probing? Oh come one! I had my hopes up...

  • Each wire has one value from 10, 22, 50, and 60.
  • To determine the value of a wire, connect it to every other wire and take the value which appears on the screen every time. Ignore the connections which make nothing appear.
  • Once one wire's value is determined, others can be determined by connecting the known wire to the other.
    • One of the values will be the known value. The other is the value of the unknown wire.
    • If the display appears blank, the unknown wire has the same value as the known wire.

Wires are ordered from left to right, then top to bottom. Values are represented by their first digit.

  • If wire #1's value is 1, 2, or 6, connect the 1st clip to a wire with value 5.
  • Otherwise, if Wire #5's value is 1, connect the 1st clip to wire #5.
  • Otherwise, connect the 1st clip to a wire with value 6.
  • If wire #5's value is 2, 5, or 6, connect the 2nd clip to a wire with value 2.
  • Otherwise, connect the 2nd clip to a wire with value 6.
