This module contains:

  • Cranky, the main character of Beatblock;
  • Accuracy and judgement counter on the top right;
  • Stage counter on the top.

Everything else is purely cosmetical.

This is a boss module. Every time a non-ignored module is solved, this module will show a new stage.

On every stage, the module shows a percentage, and may show an amount of barelies and/or an amount of misses. If any of these amounts are absent, consider them as 0. Every miss is 1 point, every barely is 0.25 points. Also, for each stage, the module picks some amount of all notes in this section. The percentage is calculated like this:

percentage = (notes-points)/notes,

and is rounded down to 2 decimal digits.

To solve the module, enter the total percentage of all sections. To do that, you'll have to find amount of notes for every stage. Be aware, that amount of notes for every stage is minimal as possible. If defining the amount of notes is impossible (as in any amount is applicable), then module picks 250 as the amount.

If you've entered a wrong answer, you can press the "<" button to enter Failsafe State. It will show all stages. To exit, press the button with the label "Page #/#".

If there are no stages to show, input 100.00% as the final answer.
