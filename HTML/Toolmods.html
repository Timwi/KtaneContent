On the Subject of Toolmods

Brought to you by time the timer.

This module is designed for training purposes and/or give information.
Press anywhere within the border to solve the module.

This module will display several information in following order, from left to right, top to bottom:

  • Date where the bomb is started on.
  • Elapsed time after the bomb activates
  • Duration between when the bomb is started and when the bomb is activated.
  • Current local time.
  • Bomb remaining realtime (with default strike time).
  • Number of solved modules.
  • Number of solvable modules.
  • Percentage of number of solved modules to number of solvable modules.
  • State of the module, either solved or unsolved.
