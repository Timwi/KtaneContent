À propos des Deux Bouts

Ce dispositif de recherche mal programmé est aussi exaspérant avec ses réponses lentes qu’il est impitoyable avec des entrées intempestives. Patience requise.

Demander une série de codes à deux lettres pour trouver la bonne réponse avant de la soumettre. Cette machine de recherche primitive est intolérante aux entrées incomplètes et excessives, ainsi qu’à toute entrée lorsqu’elle est occupée.

Étape 1 : Déterminer le code initial

Si le numéro de série contient une lettre, utiliser la position numérique de la lettre la plus à gauche dans l’alphabet comme valeur de base (A=1, B=2, ...). Sinon, utiliser 0.

Ajouter le dernier chiffre du numéro de série multiplié par le nombre de piles présentes.

S’il y a un port Stéréo RCA présent, doubler la valeur actuelle*.

Cette valeur** est maintenant le code actuel.

* Remarque : Sauter cette étape s’il y a aussi un port RJ-45 présent.

** Remarque : Prendre la valeur modulo 100 si > 99 et ajouter un zéro si < 10.

Étape 2 : Déterminer la paire de caractères et effectuer la Requête

En utilisant le code actuel, rechercher la paire de caractères. Entrer cette paire dans l’appareil et appuyer sur « Query ».

Étape 3 : Répéter et Soumettre

Le code de réponse de l’appareil est maintenant votre code actuel. Exécuter l’étape 2 deux fois de plus, en utilisant le nouveau code à chaque fois.

Après avoir reçu le code de réponse de la dernière requête, rechercher la paire de caractères correspondante, entrer-la dans l’appareil et appuyer sur « Submit ».
