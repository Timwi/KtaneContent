On the Subject of Rubik’s Clock

Time is a relative concept. But it can absolutely turn you crazy.

It’s possible to solve it like an actual Rubik's Clock.

Alternatively, repeat until all clocks, front and back, are set to 12 o’clock:

  • Select an instruction in the table below based on which clock is lit (big square) and which pin is lit (small square).
  • Select two modifications from the second table.
    First time: The first four characters of the serial number determine respectively; action of the 1st modification, amount of the 1st modification, action of the 2nd modification and amount of the 2nd modification.
    After that: Select the next row for both actions and amounts. If you are at the bottom, continue at the top.
  • If you have ‘Move’ modification(s), apply them to select a different instruction. The instruction table is cyclic; if you get over an edge, continue on the opposite side.
  • In the instruction square, the top two arrows indicate which pins to change (pull or push, depending on their current state). The bottom arrow and number indicate which gear to rotate for how many hours. An overline means counterclockwise.
  • If you have other modification(s), apply them to alter the instructions. ‘Rotate’ has priority over others. Now apply the modified instructions to the Clock.
  • Now turn the Clock over to the other side.
