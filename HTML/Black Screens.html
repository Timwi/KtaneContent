On the Subject of Black Screens

Bypassing... Done! Wait, it was too easy.

Hello, expert! We have sent you on a mission to crack the code of this barmy module! The bomb where your module is will contain all code info so no fretting! Just chop, chopping!

Screen messages

As you can see, the module has been set with a display. Simply count the amount of letters on each display and use this number for the corresponding codes. The top is used for Code 1, right used for Code 2, bottom used for Code 3.

Solving

When all row locks have been disabled, simply fool the bomb into solving the module by pressing the centre button.

Striking

If a button is not safe, it is dangerous (will strike) and vice versa.

Code 1 (Top row)

Screen number less than 3

  • If there are 2 or less port types, the first one will disable the locks.
  • If the bomb has any strikes, the second button will strike.
  • If the second button is dangerous, the third button will unlock the top row.

Screen number between 3 and 5

  • If there are 15 mins or less, the first one will disable the locks.
  • If there are more than 10 minutes on the bomb, the second button will strike.
  • If there are 10 or less batteries on the bomb (that’s a big one!), the third button will unlock the top row.

Screen number bigger than 5

  • If there are 16 mins or less, the first one will disable the locks.
  • If the bomb has 2 or more strikes, the second button will strike.
  • If the second button is dangerous, the third button will unlock the top row.

Code 2 (Middle row)

Screen number is 1

  • If there are 3 or less battery holders, the first one will disable the locks.
  • If the first button is safe, the second button will strike.
  • If there are two factors, the third button will unlock the middle row.

Screen number is bigger than 1

  • If there are no two factors, the first one will disable the locks.
  • If the third button is safe, the second button will strike.
  • If there is 1 or less port plates, the third button will unlock the middle row.

Code 3 (Bottom row)

Screen number is between 1 and 3

  • If all solved modules do not include one of these modules, the first one will disable the locks.
  • If there is a yahtzee module on the bomb, the second button will strike.
  • If the first button is not safe, the third button will unlock the bottom row.

Screen number is between 4 and 7

  • If there are three or less strikes on the bomb, the first one will disable the locks.
  • If there are no ports, the second button will strike.
  • If the first button is dangerous, the third button will unlock the bottom row.

Screen number is more than 7

  • If there is a minute or less on the bomb, the first one will disable the locks.
  • If there are more than 1000 seconds (exactly) left on the timer, the second button will strike.
  • If there is no MSA indicator on the bomb, the third button will unlock the bottom row.
