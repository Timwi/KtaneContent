Keep Talking and Nobody Explodes Mod Roulette de Placement

À propos de la Roulette de Placement

Espérons que vous ne vous fassiez pas Mario Karté.

  • Il vous est présenté une boîte avec du texte. Utiliser les flèches pour naviguer à travers les informations données. Avec ces informations, et celles présentes sur la bombe, déterminer l’objet que le coureur va avoir.
  • Pour valider un objet, appuyer sur les flèches en bas à droite du module pour naviguer à travers les objets. Une fois sur l’objet désiré, appuyer sur le bouton Submit.
  • Chaque information se traduit par un nombre que vous ajoutez à votre total. Certaines informations vous obligent à utiliser un tableau pour les convertir en informations présentes dans le Tableau 3. Le total commence toujours à 0.
  • Pour trouver la catégorie du poids, utiliser le Tableau 1.
  • Pour trouver la catégorie du véhicule, utiliser le Tableau 2.
  • Pour déterminer si le circuit est Nitro ou Retro, chercher les codes suivants : SNES, N64, GBA, GCN, DS. Si le circuit a l’un de ces codes, c’est un circuit Retro. Sinon, c’est un circuit Nitro.
  • Ajouter toutes les informations dans le Tableau 3 pour avoir votre total. Utiliser ensuite le Tableau 4 pour déterminer l’objet que le coureur va avoir. Cet objet est la solution.
