Keep Talking and Nobody Explodes Mod Раковина

О Раковине

Раковина есть, духовка есть, осталось завезти кровать и можно жить!

Этот модуль отображает раковину с двумя вентилями: горячим и холодным. Используйте количество батареек для определения строки в Таблице А. Эта строка покажет последовательность чисел, которая соответствует условиям в списке.
Для каждого условия, которое выполняется, поверните вентиль в колонке “Выполняется”.
Для каждого условия, которое не выполняется, поверните вентиль в колонке “Не выполняется”.

Кол-во батареек Последовательность условий Выполняется Не выполняется
0–1 2,1,4 Горячий Холодный
2–3 3,6,2 Холодный Горячий
4–5 5,3,1 Холодный Горячий
6+ 5,6,4 Горячий Холодный

Условия

  1. Если есть негорящий индикатор NSA
  2. Если есть гласная в серийном номере
  3. Если есть золотой вентиль
  4. Если кран из нержавеющей стали
  5. Если сливная труба из меди
  6. Если есть порт HDMI или RJ-45
