En caso de Generala

¿Desde cuándo el desactivar bombas se ha convertido en un juego de azar?

Hay cinco dados de color púrpura, amarillo, azul, blanco y negro. Empieza lanzando los dados. Luego decide cuáles mantener y cuáles relanzar. Debes mantener los que tengan el mismo valor. Repite el proceso hasta que consigas una Generala*

En casa fase, sigue las siguientes reglas para ver qué dados puedes mantener:

Si acabas de lanzar 5 dados:

Sigue la primera regla que aplique:

  • Escalera larga:* mantén el dado con el mayor valor igual a un dígito del número de serie; si no hay ninguno, mantén el púrpura
  • Escalera corta:* mantén el que sobra.*
  • Trío o full:* mantén el dado blanco si hay ≥ 2 indicadores encendidos; en caso contrario, mantén el dado negro si hay ≥ 2 indicadores apagados; en caso contrario, mantén el dado de mayor valor que no esté en el trío.
  • Póquer o doble pareja: mantén cualquier dado que tenga el mismo valor que el número de pilas; en caso contrario, mantén cualquier dado que tenga el mismo valor que el número de portapilas; en caso contrario, mantén el dado amarillo.
  • Pareja: mantén el dado del color que primero aplique:
    Hay un...Color
    puerto paralelo púrpura
    puerto PS/2 azul
    puerto RCA blanco
    puerto RJ negro
    si no amarillo
  • En caso distinto: debes lanzar los dados de nuevo.
  • En todos los casos de arriba, puedes mantener cualquier dado del mismo valor que el dado que tengas que mantener.

Generala: Todos los dados tienen el mismo valor.

Escalera larga: Cinco valores consecutivos.

Escalera corta: Cuatro valores consecutivos. El que sobra es el quinto dado.

Trío: Tres dados del mismo valor, no cuatro.

Full: Trío más pareja (de dados).

Estos términos siempre a refieren a los dados, independientemente de cuáles han sido lanzados y cuáles mantenidos.

Si acabas de lanzar 4 dados:

  • Escalera (corta o larga):* debes relanzar el dado que acabas has mantenido y mantener uno o varios dados de valor diferente:
  • En caso distinto:
    • Relanzar todos los dados siempre se permite.
    • Se permite mantener 1 dado si no es negro.
    • Se permite mantener 2 dados si ninguno es azul.
    • Se permite mantener 3 dados si el valor de los otros dos no están en el número de serie.
    • Se permite mantener 4 dados si el valor del quinto tiene un valor más grande.
    • No se permite mantener un número de dados igual al número de placas, excepto si son 2 o menos.

Si acabas de lanzar 3 dados:

  • Full:* relanza la pareja si hay algún puerto duplicado; en caso distinto, relanza el trío.
  • En caso distinto:
    • Siempre se permite mantener 0, 1 o 2 dados.
    • Se permite mantener 3 dados si el púrpura o el blanco se ha mantenido en el lanzamiento anterior.
    • Se permite mantener 4 dados si el quinto dado tiene un valor más pequeño.
    • Se permite mantener cualquier número de dados si los valores de los dados mantenidos están en el número de serie.

Si acabas de lanzar 2 dados:

  • Siempre se permite mantener 0, 1, 2 o 3 dados.
  • Se permite mantener 4 dados si el amarillo o el azul se ha mantenido en el lanzamiento anterior o si el quinto dado tiene una diferencia de 1 con los dados mantenidos.

Si solamente falta 1 para lanzar:

Sigue lanzando hasta que consigas una Generala.
