Keep Talking and Nobody Explodes Mod ❖

On the Subject of ❖ (Nonverbal Simon)

We've committed a cardinal sin - using words in a nonverbal setting.

This module has four different colored buttons.

Each color that flashes will correspond to a color to press.

This module has three to five stages. Each stage will flash a color following the colors from the previous stages. Press the colors in order from the first stage to the most recent stage.

Flashing Color Color To Press
Orange (Left)
  • If there are 0 or 1 batteries, press yellow.
  • If there are 2 or 3 batteries, press green.
  • If there are 4 or 5 batteries, press orange.
  • If there are 6 or more batteries, press red.
Red (Top) If the serial number has the following configuration:
(# = number, X = letter)
  • ###XX#, press green.
  • #X#XX#, press red.
  • XX#XX#, press orange.
  • X##XX#, press yellow.
If the serial number has neither of those configurations, red will never flash.
Yellow (Right)
  • If there are more unlit than lit indicators, press red.
  • If there are more lit than unlit indicators, press green.
  • If the amount of lit and unlit indicators are equal, press orange.
Green (Bottom) Follow the first rule that applies.
  • If there is a serial port, press orange.
  • If there is a PS/2 port, press red.
  • If there is an RJ-45 port, press green.
  • Otherwise, press yellow.
