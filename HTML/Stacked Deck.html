This module will have a big screen, a small screen, and a Submit button. The big screen will show a card face up, and the small display will tell you the Target Card.

When you tap on the big screen, the next card in the deck will be dealt face down on top of the previous card.

To disarm the module, keep dealing until the Target Card is on top of the deck, then press submit. Submitting an incorrect card will register a strike, and the module will reset with a new starting card and a new Target Card.

Use the rules below to determine the arrangement of the deck, stopping at the first one that applies.

  • If there is a lit BOB indicator and 3+ batteries, disregard all the rules and submit the starting card.
  • Otherwise, if there is a lit FRK indicator and a parallel port present, the deck is in New Deck Order.
  • Otherwise, if there are more than 2 port plates and the Target Card is a Spade, the deck is in The Charles Gauci stack.
  • Otherwise, if there is a Serial port on the bomb and the Serial Number contains the letter S, R, L, P, or T, the deck is in the Mnemonica stack.
  • Otherwise, if the last digit of the Serial Number is a prime number, the deck is in the Si Stebbins stack.
  • Otherwise, if both the Starting Card and the Target Card have the same suit, the deck is in The Bart Harding stack.
  • Otherwise, if there is a vowel in the Serial Number, the deck is the Eight Threatening Kings stack.
  • Otherwise, if there are more numbers than letters in the Serial Number, disregard the Target Card, add up all the numbers, deal that many cards, and press Submit.
  • OTHERWISE, disregard the Target Card, deal down exactly 11 cards, and press Submit.
