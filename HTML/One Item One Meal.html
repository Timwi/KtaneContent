On the Subject of One Item One Meal

Carl, what do you want? You gotta eat to keep your strength up, man.

One Item One Meal is a module in which you select the number of calories an item on a display gives the consumer.

Display

On the module, there is an image display. Match the image from the display with the table below to get to correct calorie-amount.

Number Rules

  • If the serial number has a vowel, submit when the countdown timer has a 5 in the last position.
  • If the bomb contains 4 batteries, submit when the countdown timer has a 1 in the last position.
  • Otherwise, submit when the countdown timer has a 3 in the last position.
