Original Manual

Use the letter to find the rule to modify the digit displayed. Afterwards, modulo this result by 10 by adding or subtracting 10 until it is within the range of 0–9. If after modulo 10, the value is 0, the actual value is 1. Press the button corresponding to that value. Do this 3 times to disarm.

Case: Simple Modifiers, No Edgework

E Add 0. F Modulo the digit by 6. G Add 1.
O Add 1. W If the digit is even, the value is 1. Otherwise the value is 5. Z Add 1.

Case: Simple Modifiers, No Conditions, Edgework Required

A Subtract the last digit of the serial number. B Add the number of Parallel ports. C Add the number of battery holders.
D Subtract the number of AA batteries. H Add the number of PS/2 ports. I Subtract the number of OmegaForget modules on the bomb. (Most of the time, you’re subtracting by 0.)
K Add the number of indicators. M Add the number of modules. P Add the number of serial ports.
S Add the sum of digits in the serial number. U Add the number of digits in the serial number. X Add the sum of the number of batteries and battery holders.
Y Subtract 1, then add the number of port plates. R Add the number of D batteries and PS/2 ports. Then subtract the sum of digits in the serial number. Take the absolute value of this result.

Case: Conditional Modifiers

L If there are exactly 4 batteries, 3 holders and 2 ports, add 0. Otherwise add 3. Q If the number of ports is more than 2, add the sum of 2 and the number of port plates. Otherwise add 0.
T If the sum of digits in the serial number is greater than 10 and the digit displayed is odd, the value is 5. Otherwise the value is 1. V If there is at least 1 module with “Forget” in its name, add 2 times the number of batteries. Otherwise, multiply by 9.

Case: J

J If the last digit of the serial number is 0 or 9, or Necronomicon is NOT present, add the number of PS/2 ports. Otherwise, find that digit on the top row of the table provided and add the corresponding value on the bottom row to that digit.
