À propos de la Répétition de Symboles

As-tu déjà fixé le logo DVD en espérant qu’il touche un coin de l’écran ?

  • Ce module consiste en deux écrans affichant des symboles, un interrupteur, et un écran affichant un nombre.
  • Les écrans changent en même temps, mais à des cycles différents. Chaque écran peut afficher une suite de deux à cinq symboles. (Ne pas presser les écrans pour le moment, le module enregistrerait une erreur.)
  • Le nombre affiché en dessous correspond au nombre de répétitions, qui augmente donc progressivement.
  • Lorsque le désamorceur est prêt, basculer l’interrupteur. Les écrans arrêtent de changer, affichent deux symboles aléatoires, et l’écran du bas affiche un nombre aléatoire également.
  • Le module a maintenant basculé dans le mode rétrotransphasique ou le mode antérodiamètrique. Suivre, selon le mode, les étapes ci-dessous.

    Mode rétrotransphasique

    • Dans ce mode, presser les écrans change leurs symboles.
    • Changer les symboles par ceux qui devraient normalement être affichés à la répétition indiquée sur l’écran du bas.
    • À noter que l’ordre des symboles dans lequel le désamorceur peut naviguer n’est peut-être pas le même qu’avant. Il peut même y avoir des symboles supplémentaires.

    Mode antérodiamètrique

    • Dans ce mode, presser les écrans change le numéro affiché. L’un des écrans augmente le nombre affiché, l’autre le diminue.
    • Changer le nombre en n’importe quel nombre de répétition où les deux symboles actuellement présents seraient affichés.
  • Lorsque les étapes sont effectuées, re-basculer l’interrupteur pour soumettre votre réponse. Si celle-ci est incorrecte, le module se réinitialisera avec de nouveaux symboles et potentiellement un nouveau mode.
