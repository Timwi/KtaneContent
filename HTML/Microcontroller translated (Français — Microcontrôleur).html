À propos des Microcontrôleurs

On l’appelle un “micro”-contrôleur, mais ce truc est plutôt imposant. Sans doute parce qu’il peut déclencher une énorme explosion ?...

  1. Déterminer le type de contrôleur en le comparant avec les informations ci-dessous. Il s’agit de trouver la configuration des connexions.
  2. Le point blanc sur la puce indique la position de la patte 1. L’ordre des pattes continue sur la même ligne puis à l’envers sur la ligne opposée.
  3. Utiliser le tableau pour déterminer le code couleur de chaque patte du composant pour l’appliquer à la DEL correspondante.
  4. Presser les boutons UP et DOWN pour changer les couleurs puis presser OK pour valider votre configuration (la patte suivante est choisie automatiquement).
