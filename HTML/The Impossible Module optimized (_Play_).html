For those familiar with the original manual.

To determine the speeds available to the defuser, refer to the table below with the module’s displayed model.

To determine the speeds available to Blananas9, start at the displayed model in the table, and go down X rows (wrapping around if needed), where X is:
((first speed of displayed model) * (battery holders + 1)) % 10.

Submit a speed available to the defuser - a (by putting the module’s gear into the position according to a’s column number), so that Blananas9 has some speed available to them - b, where one of the following conditions applies:

  • a + b = 25
  • abs(a - b) = 25
