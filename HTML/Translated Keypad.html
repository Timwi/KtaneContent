On the Subject of Translated Keypad

I’m not sure what these symbols are, but I suspect they have something to do with some foreign language.

Note: If all four button labels match the symbols listed in Appendix $YM8, you are looking at the wrong manual.

A “Translated Keypad” module has four square buttons with LEDs, each of which is labeled with a strange symbol. Exactly one of these symbols will be flipped vertically.

Press the four buttons in a certain order to disarm the module.

Translate each matching symbol to an assigned letter listed in Appendix $YM8. Treat both sigma (Ͼ) and reversed sigma (Ͽ) as the same symbol, as they’re only mirrored reflections of each other.

The flipped symbol, however, is ALWAYS a vowel. Use the following columns to determine what vowel the flipped symbol is.
