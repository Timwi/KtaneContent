On the Subject of Push The Button

Did I stutter?

There are aliens present among the widgets on your bomb.

Each alien is represented by a number. For each alien, you must push the Button when the countdown number has that alien's number in any position.

Once all aliens have been found, the module is solved. An incorrect button push will result in a strike.

Finding Aliens

Each rule below that applies counts as an alien. The value specified is the alien's number.

If the value is zero, the rule does not apply. If the value is 10 or more, use the first digit of the value.

  • the last digit of the serial number
  • the number of ports
  • the number of batteries
  • the number of indicators

HOWEVER, follow each rule below that applies:

  • Add the number of consonants in the serial number to each non-zero alien number.
  • If the 3rd character of the serial number is even, add it to each non-zero alien number.

If multiple aliens have the same number, only push the Button for that number ONCE.

In the VERY rare case that there are NO aliens, press the Button once at any time to auto-solve the module. (This can only happen if the last digit of the serial number is zero, there are no consonants in the serial number, the 3rd character of the serial number is odd, and there are no ports, batteries, or indicators.)
