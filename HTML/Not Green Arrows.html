On the Subject of Not Green Arrows

What’s your points?

On the module are 4 directional buttons, and a small screen in the middle.

If the buttons are not Green, or the screen is not showing three letters, you’re looking at a different module.

First, look at the table in the second page. Check all conditions listed, and keep track of the number of true conditions assigned to each of the four buttons.

Then, based on the Action Priority List below, perform the first action that applies to disarm the module. Pressing an incorrect button will register a strike, and generate 3 new letters.

Action Priority List

If all buttons have the same number of true conditions, press any button.

Otherwise, if three buttons have the same number of true conditions, press the one that doesn’t.

Otherwise, if there are two pairs of buttons with the same number of true conditions present, press either button of the higher pair.

Otherwise, if there is one pair of buttons with the same number of true conditions present, press the one that has the most false conditions and is not part of the pair.

Otherwise, if the numbers of true conditions of all four buttons are consecutive, press the one with the lowest number of true conditions.

Otherwise, if the numbers of true conditions of three buttons are consecutive, press the one that isn’t.

Otherwise, press the one with the highest number of true conditions.
