On the Subject of Recursive Password

If I were to say that I hid a message within this flavor text, how long would it take you to realize that I’m lying?

  • This module features five columns of letters, each column having five letters that can be cycled.
  • When arranged properly, the letters of each column will contribute a letter to a word in the word bank below, with the first column representing the first letter and so on. In total, there will be five words used.
  • However, this feature does not work as intended: each word will have a letter that is not what it is supposed to be. Faulty letters may be within the same column. Faulty letters will not appear in the first column.
  • The letters that the faulty ones replace when rearranged will also be within this wordlist. Pressing the toggle button will allow the module to enter input mode, in which you must input that word to disarm this module (a word will be submitted on the second toggle press.)
