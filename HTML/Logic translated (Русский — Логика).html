О Логике

Логика — это просто, но логика И разминирование бомб — определённо нет.

  • Каждый ряд отображает 3 буквы. Каждая буква представляет из себя утверждение, которое можно найти в Таблице 1.
  • На каждом ряду сперва решите утверждения внутри скобок.
  • Утверждения соединены между собой символами логических операций. Определите, как работает тот или иной символ в Таблице 2.
  • Сперва примените инверсию (ложь становится правдой, а правда — ложью) к каждому из утверждений, если над утверждением горит красный светодиод.
  • Найдите конечный результат каждого ряда и используйте кнопку справа, чтобы выбрать правду (T) или ложь (F). Когда закончите, нажмите “SUBMIT”.

Определение индикатора — см. Приложение А.
Определение элемента питания — см. Приложение Б.
Определение порта — см. Приложение В.

Таблица 2: Список символов логических операций

Логическая операция Символ Эквивалент логическому элементу Пояснение
Конъюнкция ∧ И Возвращает 1, если все входящие элементы = 1. Иначе — 0.
Дизъюнкция ∨ ИЛИ Возвращает 1, если любой из входящих элементов = 1. Иначе — 0.
Сильная дизъюнкция ⊻ Искл. ИЛИ Возвращает 1, если только один из входящих элементов = 1. Иначе — 0.
Штрих Шеффера | НЕ-И Возвращает 0, если все входящие элементы = 1. Иначе — 1.
Стрелка Пирса ↓ НЕ-ИЛИ Возвращает 0, если любой из входящих элементов = 1. Иначе — 1.
Эквиваленция ↔ Искл. ИЛИ-НЕ Возвращает 0, если только один из входящих элементов = 1. Иначе — 1.
Импликация (вправо) → - Возвращает 0, если левый входящий элемент = 1, а правый = 0. Иначе — 1.
Импликация (влево) ← - Возвращает 0, если левый входящий элемент = 0, а правый = 1. Иначе — 1.
