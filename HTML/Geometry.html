On the Subject of Geometry

Who likes geometry? Definitely not me.

This module consists of a screen, a number keypad (decimal point included), and a submit and clear button.

The middle screen will cycle 3 shapes, with a small gap at the start of the sequence. The screen will also show a symbol at the top right of the display.

In order to solve the module, you need to do some calculations, apply all necessary rules and then submit the answer using the keypad. To start you need to get your starting value.

For each calculation performed, if the decimal places are longer than 6, round off the number so that it has 6 decimal places. If the integer is larger than 999999, modulo the number by 1000000.

Starting Value:

Get the sum of all digits in the serial number and then multiply it by the number of letters in the serial number.

If a lit BOB is present, add 69 to the number.

Modifying the Starting Value:

Modify your starting number based on the symbol shown on the top right part of the screen.

Solving:

When you have done all calculations and applied all necessary rules, input the number using the keypad and hit check. Inputting a number will stop the module from cycling. If you strike, the module will NOT reset. You can press “<” to remove the last number (or decimal point) on your input.
