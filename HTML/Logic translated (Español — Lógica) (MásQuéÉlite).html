En caso de lógica

La lógica es fácil, pero la lógica Y desactivar una bomba puede que no.

  • En cada fila hay 3 letras, representando una fórmula proposicional. Cada letra representa una frase.
  • Determina la veracidad de cada letra con la tabla 1.
  • En cada fila, evalúa primero la parte de los paréntesis.
  • Estas frases (letras) están relacionadas con operadores lógicos. Puedes ver cómo funciona cada uno en la tabla 2.
  • Aplica la negación (verdadero se convierte en falso y viceversa) a cada frase si el led que tiene arriba está encendido.
  • Al evaluar toda la fila, usa los botones T/F para seleccionar si esta misma es verdadera (verde) o falsa (rojo). Y luego pulsa "Submit".

Véase el Apéndice A para la identificación del indicador.
Véase el Apéndice B para la identificación de la pila.
Véase el Apéndice C para la identificación del puerto.
