Pas à propos du Bouton

On pourrait penser qu’il est assez simple d’appuyer sur un bouton. C’est ce genre de raisonnement qui fait exploser les gens.

Le module comporte un gros bouton et un écran à gauche.
Utiliser le premier tableau pour déterminer ce qu’il faut faire avec le bouton.

Sur les modules adaptés aux daltoniens, le bouton peut présenter un motif permettant de déterminer la couleur.

Si le bouton doit être maintenu, voir la section « Ne pas relâcher un bouton maintenu ».

Si le bouton doit être martelé, voir la section « Ne pas marteler un bouton ».

Ne pas relâcher un bouton maintenu

Lorsque le bouton est maintenu enfoncé, une bande de couleur s’allume sur le côté gauche du module. Elle peut être rayée ou non. En fonction de sa(ses) couleur(s), le bouton doit être relâché lorsque le compte à rebours de la bombe remplit une condition spécifique.

Ne pas marteler un bouton

Lorsque le bouton commence à être martelé, un écran s’affiche pour indiquer combien de fois le bouton a été appuyé. En fonction de la couleur du bouton, utiliser l’une des expressions ci-dessous pour calculer le nombre de fois que le bouton doit être appuyé. Si le résultat est en dehors de la fourchette 10-99, continuer à soustraire ou à ajouter 7 au chiffre obtenu jusqu’à ce qu’il entre dans la fourchette.
Après avoir appuyé sur le bouton le bon nombre de fois, attendre 3 secondes pour que la réponse soit soumise.

Couleur du boutonExpressions
Rouge (a + 2b) - d
Orange (2b + 1) - g
Jaune (2a + d) - c
Vert d + (2f - b)
Cyan (e + f + g) - b
Bleu 2c + (d - 1)
Violet 2(f - a) + d
Rose 3g - (a + 3)
Blanc (f + ac) * (e + d)
Noir (ab + cd) - g(e - f)

Où :

  • a = nombre de piles sur la bombe
  • b = nombre de types de ports distincts sur la bombe
  • c = nombre de modules résolvables sur la bombe, y compris ce module
  • d = nombre d’indicateurs sur la bombe
  • e = dernier chiffre du numéro de série*
  • f = position alphabétique de la deuxième lettre du numéro de série*
  • g = nombre de lettres du mot sur le bouton

* Si le numéro de série ne contient aucun chiffre ou moins de deux lettres, utiliser 0 pour les valeurs correspondantes.
