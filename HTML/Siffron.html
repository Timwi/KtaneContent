Keep Talking and Nobody Explodes Mod Siffron

On the Subject of Siffron

Why are there so many words that mean 'blue'!?

  • This module has a 4 button with korean words on it.
  • Each word is name of blue in one of the 12 blue colors.
  • Use the table in the next page to get information about colors.
  • To solve the module, you must press every correct button according to rules in any order.
  • Pressing a wrong button will cause a strike and the module resets.

Rules

  • If there is more than 3 battery on the bomb, press the button that its Blue Value is greater than 180.
  • Otherwise, If there is a indicator with label IND or SND, press the button that its Hue equals to 214 or 240.
  • Otherwise, If the serial number contains a vowel, press the button that its Cyan Value is greater than 80%.
  • Otherwise, If serial and parallel present, press the button that its Blue Value is divisible by 5.
  • Otherwise, If there is no battery, press the button that its Brightness is greater than Cyan Value.
  • Otherwise, press the button with the Brightest color.
