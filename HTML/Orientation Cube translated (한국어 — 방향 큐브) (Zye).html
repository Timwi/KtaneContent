방향 큐브에 대하여

폭탄으로 죽지 않아도 뇌가 터져 죽을 것 같네요.

이 모듈을 해제하려면 좋은 3차원 방향 감각이 필요합니다. 아래쪽의 4개의 버튼을 이용해 가상의 큐브를 특정 방향으로 회전시키세요. 다만, 현재 큐브의 방향을 표시하는 기능이 없기 때문에 큐브의 상태를 직접 상상해야 합니다.

왼쪽 아래의 두 개의 버튼은 큐브를 위에서 봤을 때 시계방향 또는 반시계방향으로 돌립니다.

오른쪽 아래의 두 개의 버튼은 큐브를 가상 관찰자의 위치에서 봤을 때 시계방향 또는 반시계방향으로 굴립니다. 가상 관찰자의 위치는 모듈에 눈 아이콘으로 표시되어 있습니다. 주의: 가상 관찰자의 위치는 도중에 바뀔 수 있습니다.

예를 들어, 눈 아이콘이 아래쪽에 있다면 관찰자는 ‘FRONT(앞쪽)’ 면을 바라보고 있습니다. ‘시계방향으로 굴리기’ 버튼을 누르면 큐브의 ‘LEFT(왼쪽)’ 면이 ‘TOP(위쪽)’ 위치로 갈 것입니다.

폭탄의 시리얼 넘버에 글자 R이 있다면:

큐브를 회전시켜 처음의 왼쪽 면이 위쪽 위치에 오도록 하고, SET 버튼을 누르세요.

그렇지 않고, 폭탄에 점등된 TRN 지시등이 있거나, 점등된/점등되지 않은 CAR 지시등이 있다면:

큐브를 회전시켜 처음의 아래쪽 면이 오른쪽 위치에 오도록 하고, SET 버튼을 누르세요.

그렇지 않고, 폭탄에 PS2 포트가 있거나, 폭탄에 실수가 1개 이상 기록되어 있으면:

큐브를 회전시켜 처음의 아래쪽 면이 앞쪽 위치에 오도록 하고, 동시에 처음의 왼쪽 면이 아래쪽 위치에 오도록 하고, SET 버튼을 누르세요.

그렇지 않고, 폭탄의 시리얼 넘버에 7이나 8이라는 숫자가 있다면:

큐브를 회전시켜 처음의 오른쪽 면이 아래쪽 위치에 오도록 하고, 동시에 처음의 뒤쪽 면이 앞쪽 위치에 오도록 하고, SET 버튼을 누르세요.

그렇지 않고, 폭탄에 배터리가 2개보다 많거나, 가상 관찰자가 처음에 왼쪽 면을 바라보고 있다면:

큐브를 회전시켜 처음의 위쪽 면이 아래쪽 위치에 오도록 하고, SET 버튼을 누르세요.

그렇지 않다면:

큐브를 회전시켜 처음의 위쪽 면이 왼쪽 위치에 오도록 하고, SET 버튼을 누르세요.

실수를 할 경우:

실수를 하면 가상 큐브는 맨 처음 상태로 리셋됩니다. 가상 관찰자의 위치에 따라 이전과 다른 규칙을 적용해야 할 수도 있다는 것을 유의하세요.
