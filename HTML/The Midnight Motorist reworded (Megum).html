Keep Talking and Nobody Explodes Mod The Midnight Motorist

On the Subject of The Midnight Motorist

The Bite of ’87 refers to how many big macs William Afton ate before going into cardiac arrest.

Key points:

  • The module simulates races between 8 possible cars.
  • If 2 cars repeatedly race against each other the winner is consistent.
  • There are 2 modes the module has: practice and bracket tournament.
  • The goal of the module is to determine the winner of the bracket tournament.

Global controls (independent of the mode):

  • Left blue button: switch mode.
  • Right blue button: mute/unmute.
  • Right joystick: change volume.

Practice mode:

  • Controls:
    • Left joystick: N/A.
    • Left yellow button: start a race/halt a race and generate a new one.
    • Right yellow button: N/A.
  • The module starts in this mode by default.
  • There are 2 independent pairs of randomly picked cars (no repeats) from the available pool (the pool has 8 cars by default).
  • Use this mode to collect data on which cars win against which cars.

Bracket tournament mode:

  • Controls:
    • Left joystick: switch the pointer between cars.
    • Left yellow button: select the winner car.
    • Right yellow button: add/remove the selected car from the practice pool (4 minimum).
  • There are 8 consistently ordered cars competing in a bracket tournament.
  • Using the practice mode, detetrmine the winner of the finals.
  • You can add/remove cars from the practice pool to make rerolling for desired pairs in practice mode faster.
