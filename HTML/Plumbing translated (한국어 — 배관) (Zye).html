배관에 대하여

이건 하고서 손을 좀 씻어야겠어요...

  • 모듈에는 4개의 입력 파이프(왼쪽)와 4개의 출력 파이프(오른쪽)가 있습니다. 적어도 하나의 입력 파이프와 하나의 출력 파이프는 활성화되어 있습니다.
  • 해체자는 6×6칸의 파이프 격자를 이용해, 모든 활성화된 입력 파이프를 모든 활성화된 출력 파이프로 연결하되, 비활성화된 파이프들은 연결하지 말아야 합니다. 6×6 격자에 있는 파이프를 클릭하면 파이프가 회전합니다.
  • 활성화된 파이프에 연결되어 있는 모든 파이프는 올바르게 다른 파이프들로 이어져야 합니다. 다른 파이프로 이어지지 않은 (또는 비활성화된 입력/출력 파이프로 연결된) 파이프가 있다면 제출 시 실수로 처리됩니다.
  • 정답을 완성했다면, “CHECK”를 눌러 정답을 확인합니다. 오답이면 실수로 처리됩니다.
  • 활성화된 입력 및 출력 파이프는 아래 표에 의해 결정됩니다. 만약 파이프에 대한 가산점의 개수가 감점의 개수보다 많다면, 활성화된 파이프입니다.

Keep Talking and Nobody Explodes Mod 배관
붉은색 입력
  • 가산점: 시리얼 넘버에 ‘1’이 있음
  • 가산점: RJ-45 포트가 정확히 1개
  • 감점: 중복되는 포트가 있음
  • 감점: 시리얼 넘버에 중복되는 자가 있음
 노란색 입력
  • 가산점: 시리얼 넘버에 ‘2’가 있음
  • 가산점: Stereo RCA 포트가 1개 이상
  • 감점: 중복되는 포트가 없음
  • 감점: 시리얼 넘버에 ‘1’이나 ‘L’이 있음
초록색 입력
  • 가산점: 시리얼 넘버에 숫자가 3개 이상
  • 가산점: DVI-D 포트가 1개 이상
  • 감점: 붉은색 입력이 비활성화됨
  • 감점: 노란색 입력이 비활성화됨
 파란색 입력
  • 비고: 다른 입력이 모두 비활성화된 경우 무조건 활성화됨
  • 가산점: 포트 종류가 적어도 4종
  • 가산점: 배터리가 적어도 4개
  • 감점: 포트가 없음
  • 감점: 배터리가 없음
붉은색 출력
  • 가산점: Serial 포트가 1개 이상
  • 가산점: 배터리가 정확히 1개
  • 감점: 시리얼 넘버에 숫자가 2개보다 많음
  • 감점: 활성화된 입력이 2개보다 많음
 노란색 출력
  • 가산점: 중복되는 포트가 있음
  • 가산점: 시리얼 넘버에 ‘4’나 ‘8’이 있음
  • 감점: 시리얼 넘버에 ‘2’가 있지 않음
  • 감점: 초록색 입력이 활성화됨
초록색 출력
  • 가산점: 활성화된 입력이 정확히 3개
  • 가산점: 포트가 정확히 3개 있음
  • 감점: 포트가 3개보다 적음
  • 감점: 시리얼 넘버에 숫자가 3개보다 많음
 파란색 출력
  • 비고: 다른 출력이 모두 비활성화된 경우 무조건 활성화됨
  • 가산점: 모든 입력이 활성화됨
  • 가산점: 다른 출력 중 하나라도 비활성화됨
  • 감점: 배터리가 2개 미만
  • 감점: Parallel 포트가 없음
