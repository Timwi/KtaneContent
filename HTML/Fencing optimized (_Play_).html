Note:

  • The score at the top of the module is displayed in the format of
    “[Your Score]-[Opponent’s Score]”.
    • “losing”, “in a tie” and “winning” all refer to the score.
  • The lights at the bottom of the module are numbered 1-5 left-to-right in this manual.
    • The only two important lights are light #1 (leftmost) and light #5 (rightmost).
  • Button presses are bolded and underlined.
  • Rules that will solve the module after being followed have a green background.

Rules

All Weapons

  • Salute if one of the following applies:
    • The timer is 03:00 and the score is 0-0.
    • The timer is 00:00.
    • The score is 5-0 or 0-5.

Épée

  • If display is ←:
    • If you are losing or in a tie, Hand.
    • If light #1 is on, Torso.
    • Otherwise, Follow.
  • If display is →:
    • If light #5 is on, Head.
    • Otherwise, Follow.
  • If display is ∕:
    • If light #5 is on, Torso.
    • Otherwise, Wait.
  • If display is Ø:
    • If you are winning, Wait.
    • Otherwise, Hand.

Foil

  • If display is ←, Parry then Torso.
  • If display is →:
    • If you are winning, Wait.
    • Otherwise, Follow then Torso.
  • If display is ∕, Wait
  • If display is Ø, Torso then Torso.

Sabre

  • If display is ←, Parry then Head.
  • If display is →, Follow, then:
    • If light #5 is on, Head.
    • Otherwise, Hand.
  • If display is ∕, Parry then Head.
  • If display is Ø, Head then Parry then Torso.
