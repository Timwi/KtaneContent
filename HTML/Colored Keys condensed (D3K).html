Keep Talking and Nobody Explodes Mod Colored Keys

On the Subject of Colored Keys

Who painted this keypad?

This module features four colored keys with letters on them and a display with the name of a color.

Press the key that has the most conditions met.

Top Left: Top Right:
  • has lit MSA
  • has unlit CAR
  • odd #battery holders
  • has RJ-45
  • Key color matches color of another key
  • Key color = displayed word
  • Key color = color of displayed word
  • Key letter in display word
  • Key letter in SN
  • has lit SIG
  • has unlit SND
  • >3 batteries
  • has DVI-D or parallel
  • ≥1 pair of key colors
  • Key color = displayed word
  • Key color = color of displayed word
  • Key letter in display word
  • Key letter in SN
Bottom Left: Bottom Right:
  • has lit NSA
  • has unlit TRN
  • 0 batteries
  • has PS/2 or serial port
  • ≥3 keys share same color
  • Key color = displayed word
  • Key color = color of displayed word
  • Key letter in display word
  • Key letter in SN
  • has lit CLR
  • has unlit BOB
  • even #batteries
  • has Stereo RCA
  • All key colors are unique
  • Key color = displayed word
  • Key color = color of displayed word
  • Key letter in display word
  • Key letter in SN

Hint: First 4 conditions are based on edgeworks. The last 4 conditions are the same for all 4 buttons.

*SN: Serial Number

Tie-break: 1st button in reading order.

Module resets after an incorrect key press.
