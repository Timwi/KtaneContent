A Respeito do Gerador de Números Aleatórios

Sorte é acreditar que você tem sorte.

  • Quando o módulo ativar, haverão dois tiles (~) no display.
  • Clique no botão escrito "Generate" para gerar números aleatórios até que você encontre um número que satisfaça as regras.
  • Assim que você encontrar um número apropriado, aperte no botão escrito "Accept".

Se o último dígito do número de série for...

Ímpar: O número precisa ser ímpar.

Par: O número precisa ser par.

Se uma vogal no número de série...

Estiver presente: O número deve estar entre 0 e 49.

Não estiver presente: O número deve estar entre 50 e 99.
