Keep Talking and Nobody Explodes Mod Sudokulocation

On the Subject of Sudokulocation

I can fix her...

This module is initially blank with a dark background. When selecting the module, the display and its buttons will phase through the module. Deselecting the module will cause the controls to phase back through the module and be hidden.

The module contains a hidden sudoku grid. By default, the grid is 4x4 but can be extended to 9x9 with the extreme mode setting. One value in the sudoku grid has been modified such that it invalidates its row, column, and box.

The controls consist of a seven-segment display and four surrounding buttons. The display shows the current cell you are on. Use the buttons to navigate through the grid, find the invalid cell, and correct it by toggling the segments to display the correct number.

Pressing a movement button causes you to move in a random direction. Directions are randomized per cell but fixed for that cell.

Attempting to move to a different cell when the display is not a valid number will cause the module to strike.

Once you have corrected the sudoku grid, press the status light to solve the module.

Pressing the status light when the grid is not corrected will cause the module to strike.
