On the Subject of Whack The Cops

Added the a Whacka Cops?

Use the 2×2 square in the table below containing the four colors on the module with the following instructions.

  • On the module, if the top left Whacka Cop is Red or Orange, select the Cops as they appear on the table starting from the bottom left of the square going clockwise.
  • Otherwise, on the module, if the top right Whacka Cop is Orange or Grey, select the Cops as they appear on the table starting from the bottom right of the square going counterclockwise.
  • Otherwise, on the module, if the bottom right Whacka Cop is Lime or Cyan, select the Cops as they appear on the table starting from the top right of the square going counterclockwise.
  • Otherwise, on the module, if the bottom left Whacka Cop is Blue or Yellow, select the Cops as they appear on the table starting from the top left of the square going counterclockwise.
  • Otherwise, select the Cops as they appear on the table starting from the top left of the square going clockwise.
