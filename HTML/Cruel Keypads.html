On the Subject of Cruel Keypads

Do you know Korean?

  • This module displays Korean letters, a colored bar, and a stage counter. There are 3 stages.
  • To disarm the module, press the symbols in the correct order. For each stage, use the Venn diagram below to find out which table to use.
  • Apply the special rule and input the symbols in the correct order.
This is stage 1 or 3
Any of these symbols are present: ㅇ ㅈ ㅉ ㅟ ㅋ
If the color of the strip contains a letter* in common with the serial number.
The strip is red, blue or green

* The possible colors are: Red, Blue, Green, Yellow, Magenta, and White.
