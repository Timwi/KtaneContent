モジュール詳細：HTTPレスポンス

この爆弾がWebサーバーのように動くと……

  • モジュールは、リモートまたはローカルのステータス文字列を表示する。それに対して、3桁のステータスコードで応答する必要がある。
  • リモートコードは表1から探すことができ、ローカルコードは表2から探すことができる。
  • 極秘情報として、表示制限により一部の文字列が省略されている可能性がある。
  • キーパッドを利用して正しいレスポンスコードを入力すると、このモジュールは停止する。
  • 注意：削除ボタンは無いので、慎重に！
