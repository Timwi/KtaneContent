지시등 확인 참고자료는 부록 A를 보세요.
배터리 확인 참고자료는 부록 B를 보세요.

다음 규칙들을 명시된 순서대로 확인하세요. 해당하는 첫 번째 사항을 실행하시면 됩니다:
  1. 표시된 숫자가 예순아홉이라면, 'D'가 적힌 버튼을 누르세요.
  2. 표시된 숫자가 6으로 나눠떨어진다면, 'A'가 적힌 버튼을 누르세요.
  3. 폭탄에 배터리가 두 개 이상이며 표시된 숫자가 3으로 나눠떨어진다면, 'B'가 적힌 버튼을 누르세요.
  4. 시리얼 넘버에 'C', 'E', 또는 '3'이 포함되어 있고 표시된 숫자가 스물둘 이상이며 일흔아홉 이하라면, 'B'가 적힌 버튼을 누르세요.
  5. 그렇지 않고, 시리얼 넘버에 'C', 'E', 또는 '3'이 포함되어 있다면, 'C'가 적힌 버튼을 누르세요.
  6. 표시된 숫자가 마흔여섯보다 작다면, 'D'가 적힌 버튼을 누르세요.
  7. 그렇지 않다면, 'A'가 적힌 버튼을 누르세요.
