On the Subject of Job Application

Mad at your mother telling you to get a job? Apply for one — virtually....

You decided — today is the day — “I’m finally going to apply for a job and get off KTANE”. Well, great! Let’s get started.

This is a module in which you select the correct name, availability, and job a person will write on their job application, all based on real-life time and edgework.

Firstly, ascertain the real-life day of the week that the bomb was started on. Then, follow the case set for your specified weekday.

Weekdays

Monday

Garfield hates Mondays....

  1. Select applicant Emily.
  2. If the bomb has a Serial port, select availability option Flexible.
  3. Otherwise, select availability option Part-Time.
  4. Select job option Usher.

Tuesday

  1. Select applicant Jacob.
  2. If the bomb has 2 batteries, select availability option Full-Time.
  3. Otherwise, select availability option Flexible.
  4. Select job option Lifeguard.

Wednesday

  1. Select applicant William.
  2. If the bomb’s serial number contains an even digit, select availability option Part-Time.
  3. Otherwise, select availability option Full-Time.
  4. Select job option Retail.

Thursday

  1. Select applicant Olivia.
  2. If the bomb has a Stereo RCA port, select availability option Full-Time.
  3. Otherwise, select availability option Flexible.
  4. If the bomb’s serial number contains any of the letters “XDFRH”, select job option Assistant.
  5. Otherwise, select job option Barista.

Friday

  1. If the bomb’s serial number contains any of the letters “AEIOU”, select applicant Danny.
  2. Otherwise, select applicant Ava.
  3. If the bomb has an unlit CAR indicator, select availability option Flexible.
  4. Otherwise, select availability option Part-Time.
  5. Select job option Musician.

Saturday

  1. Select applicant Matthew.
  2. If the bomb has a PS2 port, select availability option Full-Time.
  3. Otherwise, select availability option Part-Time.
  4. Select job option Babysitter.

Sunday

And on the seventh day, God ended his work which he had made; and rested on the seventh day from all his work which he had made.

  1. Select applicant Matthew.
  2. Select availability option Flexible.
  3. Select job option Teacher.

Submission

If all the current attributes have been assigned from their respective weekday, press the submit button. This will solve the module.
