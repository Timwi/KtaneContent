On the Subject of Not Morse Code

An antiquated gimmick of module complication? What next? At least it’s still kinda like Morse Code, so don’t sweat it too much.

Use the left and right button to select one of the five possible frequencies available on the module. Each frequency will transmit a different word.
Only one column below contains all five words being transmitted.

Select a frequency and press “XT” to submit that word. Submit the five words in the order they appear from top to bottom within that column.

At any time, you may hold down the “XT” button for three seconds to reset the submission.
