On the Subject of Forteroudo

A “Forterouuuuudo” module has five blocks labelled with an English word inserted into some of the six compartments. Each compartment also has a scrambled word printed on the inside. A ‘CHECK’ and a ‘RESET’ button sit on the right side of the module.

  • Move the blocks around to reveal 6-8 letters inside each compartment. Note that blocks always move to the open compartment.
  • Rearrange the letters within each compartment to obtain an English word, creating a list of six English words. For each word, order the letters in the word in alphabetical order. Then find the correct word in Appendix UUUU using the rearranged word.
  • This list from top to bottom has already been alphabetized using one of the alphabets in Appendix ALPHA. Alphabetize the five blocks from top to bottom using this same alphabet.
  • Pressing the ‘RESET’ button will generate six new words within the compartments. The alphabet will not change.
  • Pressing the ‘CHECK’ button checks the current order of all the blocks. If the blocks are correctly sorted according to the alphabet obtained earlier, a short tune is played before the module defuses. If the blocks are sorted incorrectly, the bomb strikes.
    • The empty compartment is ignored when checking the block’s current order.

Note: If the pre-scrambled word inside the empty compartment is also sorted correctly relative to the blocks, the solving tune is skipped and the module defuses immediately.
