Keep Talking and Nobody Explodes Mod Mastermind

À propos du Mastermind

Le jeu de société classique, avec un petit plus. Pouvez-vous vous souvenir de tout ?

  • Pour désamorcer ce module, trouver le bon code à 5 couleurs.
  • Les couleurs utilisées sont : Blanc, Magenta, Jaune, Vert, Rouge et Bleu.
  • À noter que chaque couleur peut être utilisée plusieurs fois.
  • Les couleurs des DELs peuvent être modifiées en appuyant sur les DELs.
  • En pressant le bouton « Query », des informations sur le code actuellement saisi seront données.
    Le nombre de gauche donne le nombre de couleurs correctes dans les bonnes positions.
    Le nombre de droite donne le nombre de couleurs correctes dans les mauvaises positions.
  • Quand le bon code est saisi, le module sera désamorcé une fois le bouton « Submit » pressé. Valider un mauvais code enregistrera une erreur.
