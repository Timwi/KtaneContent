  • Multiply TR by 10 for each battery, max 106.
  • If TR = 0; no resistor.
  • If TR = Rx; go through Rx.
  • If TR > R1, R2; series.
  • If TR < R1, R2; parallel.
  • LIT FRK = connect outputs together.
  • Otherwise, at least 1 D cell = secondary to secondary, no resistor.
