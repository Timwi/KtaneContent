On the Subject of XO

Kisses from XO.

For those familiar with the original manual.

The boards are ordered top to bottom according to the word, but the first board may not be the first letter in the word. The bottom right space of the third page in submission mode can be used as a backspace.

Creating the key

  • Caesar shift the fourth serial number character backwards by the sum of serial number digits.
  • Place the resulting letter in the middlemost slot using the shift buttons.

Game rules

Determine the final state of each board using the following rules.

  • X and O players alternate, with X being the first player.
  • If a player can get 3 pieces in a row, column, or diagonal, they must do so.
  • Otherwise, if a player can block their opponent from completing a tic-tac-toe, they must do so. If there are multiple tic-tac-toes to be blocked, they must block using the first space in reading order.
  • Otherwise, gameplay freezes and the board state is final.
