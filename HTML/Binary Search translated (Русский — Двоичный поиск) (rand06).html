О Двоичном поиске

Не пытайтесь угадать возраст женщины с помощью двоичного поиска. Согласитесь, странным будет сначала спросить "пятьдесят", и сразу после этого "двадцать пять".

На модуле находится:

  • Два числа, одно из них на экране и равно 50;
  • Три кнопки: кнопка "влево", кнопка отправки и кнопка "вправо";
  • Линия с концами в точках 0 и 99, а также отметка на линии в точке, значение которой указано на экране.

Действия кнопок

Кнопки "влево"/"вправо": отнимает/добавляет N от числа на экране, где N:

  1. Первое нажатие: N = 25;
  2. Второе нажатие: N = 13;
  3. Третье нажатие: N = 7;
  4. Четвертое нажатие: N = 4;
  5. Пятое нажатие: N = 2;
  6. Шестое и последующие нажатия: N = 1.

Кнопка отправки: при нажатии данной кнопки проверяется равенство двух чисел.

  • Если числа равны, то модуль будет решен.
  • Иначе, модуль зарегистрирует ошибку. После этого, левое число изменится, число на экране снова вернется к 64, а количество нажатий на кнопки сбросится.
