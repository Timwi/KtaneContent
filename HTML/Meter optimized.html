Solving the Module

  1. Calculate the Foot Modifier by looking up the values of the first two letters from the Serial Number in Table 2, and adding them together with the last digit in the Serial Number.
  2. Find the lines displayed on the module in Table 1, and add the Foot Modifier to each. Subtract 12 from each result until it is a value between 1 and 12 inclusive.
  3. Find the modified values in Table 1, and take the corresponding Pattern.
  4. Enter the modified patterns on the module, pressing for u, and for s.
