Keep Talking and Nobody Explodes Mod セットアップウィザード

モジュール詳細：セットアップウィザード

警告: 未知のアプリケーションをインストールすると、爆弾がマルウェアに感染し、あなたが爆発する危険性があります。

  • このモジュールの解除を開始する前に、いくつかの予備ステップを完了し、インストールプロセスを終了する必要がある。
  • 「Next(次へ)」ボタンを押して次のページに進むか、右上の 「X」ボタンを押してセットアップウィザードを閉じ、モジュールを元の状態にリセットすることができる。
  • モジュールのインストールを終了するには、アカウント情報を決定し、3ページ目で送信する。

1ページ目: インストール先の選択(Download Location)

このページでは、2つの単語からなるあなたのユーザー名(Username)を決定する。このページには2つのフォルダがあるファイルディレクトリがある。ランダムなフォルダからスタートする(これを「開始フォルダ」とする。)が、現在入っているフォルダの名前は表示されず、フォルダをクリックしてその中に入ることができる。ディレクトリマップは以下の通りである。

ただし、これらのフォルダのうち2つは名前が入れ替わっている。入れ替わったフォルダと、あなたの開始フォルダ、そして以下の変換表を使用し、あなたのユーザー名を決定する:

  • シリアルナンバーの最初の英字が“SETUPWIZARD”の文字のいずれかである場合、アルファベット順で1番目の入れ替わったフォルダを表Aで使用し、アルファベット順で2番目の入れ替わったフォルダを表Bで使用する。
  • それ以外の場合、アルファベット順で2番目の入れ替わったフォルダを表Aで使用し、アルファベット順で1番目の入れ替わったフォルダを表Bで使用する。
  • シリアルナンバーの最後の英字が“COMPUTERLAB”の文字のいずれかである場合、段には入れ替わったフォルダを使用し、列には開始フォルダを使用する。
  • それ以外の場合、列には入れ替わったフォルダを使用し、段には開始フォルダを使用する。

2ページ目: 使用許諾契約(Terms and Conditions)

このページでは、6桁の数字からなるパスワード(Password)を決定する。このモジュールでは、パスワードの数字はそれぞれA～Fと表記される。

使用許諾契約のページには、パスワードを1つの数字に制限する方程式のリストがある。可能な操作は以下の通りである:

  1. X + Y = # (加算)
  2. X - Y = # (減算)
  3. X * Y = # (乗算)
  4. X / Y = # (除算)
  5. X || Y = # (連結)

ただし、“=”の後の数字のうち2つは入れ替わっている。連立方程式を求め、6桁のパスワードを見つけた後、入れ替わった数字を記録する。入れ替わった数字のうち、大きい数字をX、小さい数字をYとする。次に、シリアルナンバーの最後の数字をとり、それを5で割った余りに1足した数を求める。その数字と上記の数字のリストの対応する演算をXとYに対して使用する。最後に、パスワードの数字をZの数だけだけ右にシフトする。Zは演算の結果である。これが最終的なパスワードである。

3ページ目: ユーザー情報(Account Info)

このページには、ユーザー名とパスワードを入力するスペースがある。モジュール上のキーボードで入力し、「Finish(完了)」を押して送信する。間違ったユーザー名またはパスワードを入力した場合、ミスが記録されるが、モジュールはリセットされない。

モジュールのインストールに成功した場合、モジュール上に大きな緑色のボタンが表示される。このボタンを押すと、モジュールが解除される。
