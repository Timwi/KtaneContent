    1. L'opérateur binaire (AND, OR, XOR, NOT)

  • Uitiliser la table des octets ci-dessouset l'opérateur binaire de l'afficheur pour déterminer la réponse. Dans cette table, MSB est le bit le plus significatif, LSB le moins significatif.

Info AND OR XOR NOT
COMMENT Si les deux octets sont à 1, le résultat est 1. Sinon le résultat est 0. Si l'un des deux octets est à 1 (ou les 2), le résultat est 1. Sinon le résultat est 0. Si l'un des deux octets est à 1 (pas les 2), le résultat est 1. Sinon le résultat est 0. Ignorer le second octet. A chaque bit, le résultat est son opposé.
