On the Optimization of The Burnt

This ritual is short-lived.

Fan the flames once to lock the flames in. Consider the number of red, green and blue flames upon doing so.

Starting with 10, subtract 1 for every green flame.

If a voltage meter is present and its voltage is greater than the number of red flames, subtract 2.

This number is the number of steps required. If this is less than 1, set the number of steps to 1.

Take the number of red flames and subtract the number of blue flames, and use this as the column of the table provided. Take the number of steps as the row. The intersection determines the last seconds digit of the countdown timer to fan again.
