On the Subject of Uncomplicating Wires

Who would think that wires this complex could totally refrain from the Venn Diagram?

White Wires

  • Only white -> C
  • Star -> C
  • LED -> D
  • Star + LED -> B

Red Wires

  • Only red or red + white -> S
  • Star -> C
  • LED -> B
  • Star + LED -> B

Blue Wires

  • Only blue or blue + white -> S
  • Star -> D
  • LED -> P
  • Star + LED -> P

Red + Blue Wires

  • Only blue + red -> S
  • Star -> P
  • LED -> S
  • Star + LED -> D

Letter meanings

  • C -> Cut
  • D -> Don’t cut
  • S -> If last serial # digit is EVEN, cut
  • P -> If there’s a parallel port, cut
  • B -> If there’s two or more batteries, cut
