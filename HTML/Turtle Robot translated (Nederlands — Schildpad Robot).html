Betreffende Schildpad Robot

Het is een beetje oud en gammel, maar het werkt.

Een oude terminal computer bevat tekeninstructies voor een schildpad robot. Bijvoorbeeld:

  • FD 4 — Doe 4 stappen vooruit.
  • LT 90 — Draai 90 graden naar links.
  • RT 90 2 — Beweeg in een curve, waarbij je 90 graden draait over een cirkel met radius 2.

Door het volgen van de instructies krijg je een van de onderstaande tekeningen. Deze kan zijn gedraaid, gespiegeld en/of geschaald. De code bevat drie verkeerd ingevoerde programmeerfouten. Gebruik de pijltjestoetsen om door de broncoderegels te bladeren. Gebruik de '#'-toets om de programmeerfouten eruit te commentariëren. Het commentariëren van een instructie die niet verkeerd was resulteert in een foutmarkering. Markeer alle programmeerfouten om de module op te lossen.
