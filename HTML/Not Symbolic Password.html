On the Subject of Not Symbolic Password

Thanks... wait, shifted right?

A Not Symbolic Password module has six cells which represent six positions in the grid below. Pressing an arrow button will move all cells in that row or column once in that direction. Set the cells in such a way that the position on the module matches a 3×2 position in the grid below. Successfully doing so will disarm the module.

For clarification, shifting all cells on the module in the same direction any given amount will always form a valid position in the grid, meaning any configuration that may appear valid, but may not be a valid 3×2 position.

Do note that, while the display may show a pattern that occurs in the table below, it is only a valid submission if the current location of each cell forms a 3×2 in the grid. For example, a correct submission would still be correct if you were to shift both rows to the right once.
