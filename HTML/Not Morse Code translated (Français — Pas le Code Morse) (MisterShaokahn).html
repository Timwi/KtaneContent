Pas à propos du Code Morse

Un gadget désuet pour compliquer les modules ? C’est quoi la suite ? Au moins, c’est encore un peu comme le code morse, alors ne vous en faites pas trop.

Utiliser les boutons gauche et droit pour sélectionner l’une des cinq fréquences disponibles sur le module. Chaque fréquence transmet un mot différent.
Une seule colonne ci-dessous contient les cinq mots transmis.

Sélectionner une fréquence et appuyer sur « XT » pour soumettre ce mot. Soumettre les cinq mots dans l’ordre où ils apparaissent de haut en bas dans cette colonne.

À tout moment, le bouton « XT » peut être maintenu pendant trois secondes pour réinitialiser la séquence.
