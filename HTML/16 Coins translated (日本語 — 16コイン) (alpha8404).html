４×４の正方形に１６枚のコインが並べられる。「Ｈ」は頭(Heads)、「Ｔ」は尾(tails)である。モジュールを解除するには、手順に従ってターゲットコインを決定しクリックする。

  • 偶数の列の「Ｈ」の数を数える(下記に記された位置)。「Ｈ」の数が奇数の場合、ターゲットコインは偶数の列にある。それ以外の場合、ターゲットコインは奇数の列にある。
    x x
    x x
    x x
    x x
  • 右半分の「Ｈ」の数を数える(下記に記された位置)。「Ｈ」の数が奇数の場合、ターゲットコインは右半分にある。それ以外の場合、ターゲットコインは左半分にある。
    xx
    xx
    xx
    xx
  • 偶数の段の「Ｈ」の数を数える(下記に記された位置)。「Ｈ」の数が奇数の場合、ターゲットコインは偶数の段にある。それ以外の場合、ターゲットコインは奇数の段にある。
    o
    xxxx
    o
    xxxx
  • 下半分の「Ｈ」の数を数える(下記に記された位置)。「Ｈ」の数が奇数の場合、ターゲットコインは下半分にある。それ以外の場合、ターゲットコインは上半分にある。
    o
    o
    xxxx
    xxxx
