On the Subject of Brad Memory

"Brad, I am your father." Wait, no, that doesn't sound quite right.

  • Press all correct buttons in the correct order in a stage to progress to the next stage.
  • Complete all 5 stages to solve the module.
  • Positions are ordered from left to right.
  • Pressing an incorrect button or pressing out of order will cause a strike and reset the module to stage 1.
  • Each stage will display an image of Brad Pitt from a movie in which he has starred. Consult Appendix: Brad Pitt Movies to identify which movie is being displayed.
  • Once you have identified which movie is displayed, look up its row of information in Table 1: Movie Facts to obtain a set of original facts.
  • Each stage will have a color on the rim of the display image. Apply this color in Table 2: Color Modifiers to travel up or down each column, wrapping if necessary, from the original facts in Table 1 to find new facts for each category.
  • Use these new facts with the following instructions to determine which buttons should be pressed in each stage.
  • Finally, find the correct order to press the correct buttons in each stage using Table 3: Button Order.

Stage 1:

If the genre contains Action: Press 1
If the role contains the letter U: Press 2
If there are exactly 3 costars: Press 3
If the year is even: Press 4
If the entire character name is less than 10 characters or if no other buttons have been pressed this stage: Press 5

Stage 2:

If the title and role contain no spaces: Press 1
If you pressed 2 in the previous stage: Press 2
If the year is divisible by 3 or you only pressed 1 button in the previous stage: Press 3
If you pressed the 4th position in the previous stage: Press 4
If there is a female costar or if no other buttons have been pressed this stage: Press 5

Stage 3:

If the genre is only 1 type and you pressed the 5th position in Stage 1: Press 1
If the director is David Fincher or Quentin Tarantino: Press 2
If there were at least 4 total correct presses in Stages 1 and 2 combined: Press 3
If any costar has a 5 letter first name: Press 4
If the year is before 2000 or if no other buttons have been pressed this stage: Press 5

Stage 4:

If the Stage 1 title contained the letter B: Press 1
If the Stage 2 genre contained Sports or Sci-Fi: Press 2
If the Stage 3 character's first and last name started with the same letter: Press 3
If the role shares the same first letter with the director's first name from any previous stage: Press 4
If the displayed movie is the same as any previous stage's display or if no other buttons have been pressed this stage: Press 5

Stage 5:

If you pressed 1 exactly once: Press 1
If you did not press the 2nd or 4th position in Stage 2: Press 2
If the Stage 3 rim color was red or cyan and the role began or ended with a vowel: Press 3
If you pressed the 4th position an even number of times or if the title never contained a number: Press 4
If Pittsburgh, Peach Pit, or Ball Pit were displayed in any previous stage or if no other buttons have been pressed this stage: Press 5

Special Cases*

Hmmmm, that doesn't look like a Brad Pitt movie. Proceed accordingly:

  • Pittsburgh: You can barely breathe through the polluted air. Run. Run as far away as possible. Move down 9 spots from each original movie fact in the previous stage, wrapping around if necessary, then follow the instructions and ordering from the current stage.
  • Peach Pit: A natural vessel for cyanide! Apply the Cyan color modifier to the original movie facts from Stage 1, then follow the instructions from Stage 2 but with the current stage's ordering.
  • Ball Pit: Wow, a magical time travelling ball pit! Hop in! Find the movie parallel in location in Table 1 to the Stage 2 original movie (e.g. Se7en goes to Babylon), apply the color modifier from Stage 1, then follow the instructions from the current stage but with reversed ordering.

*Do not apply the rim color modifier from the current stage.
