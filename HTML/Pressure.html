On the Subject of Pressure Modules

Damnit, someone broke the valves again...

- This module contains extremely poisonous gas.
- It will build up over time, and can't be gotten rid of.
- The percentage number indicates how much pressure is built up from the gas.
- If it exceeds 99%, it's game over.

Stalling the module

To stop the pressure from building, hold the big button. As soon as you let go, the pressure will begin building again.

Solving the module

In order for this module to be solved, these conditions must be met:

  1. All modules that start with "The " are solved.
  2. All modules made by exactly 1 author are solved.
  3. All modules published in the repo before 2018 are solved.
