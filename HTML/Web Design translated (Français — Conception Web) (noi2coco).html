À propos de la Conception Web

Bienvenue au cours de conception de sites web. La première règle de la conception de sites web est la suivante : vous ne parlez pas de Comic Sans MS. La deuxième règle est la suivante : Vous ne devez pas parler de Comic Sans MS.

  • Le module affiche un extrait de fichier CSS sur un écran.
  • Sur la base des informations fournies, vous devez Accepter (✓), Envisager (Δ) ou Refuser (✗) le code.

Étape 3 : Calculer le score du site

Calculer le score du site comme suit :

  • Commencer par le nombre de lignes à l’intérieur des {accolades}. Compter les points-virgules est une bonne idée car c’est un terminateur de ligne.
  • +3 points chacun pour :
    • La valeur R de la Couleur Cible est inférieure à la valeur R du seuil.
    • La valeur V de la Couleur Cible est supérieure ou égale à la valeur V du seuil.
    • La valeur B de la Couleur Cible est supérieure à la valeur B du seuil.
  • +2 points pour chaque margin/padding.
  • +1 point pour chaque border/border-radius sauf si c’est 0px ou 50%.
  • -1 point pour chaque z-index sans une position.
  • +1 point pour chaque font-family sauf si c’est "Comic Sans MS", qui donne -5 points.
  • +2 points pour chaque box-shadow/text-shadow sauf si c’est none.
  • x2 les points si les boutons du module sont colorés, ou -3 points s’ils sont gris.
    • Si le mode Daltonien est actif, à la place x2 les points pour les boutons blancs, et -3 pour les boutons noirs.
  • Si le score est maintenant négatif ou 0, +16 jusqu’à ce qu’il soit positif.
  • Additionner les chiffres du score jusqu’à ce qu’il ne reste plus qu’un seul chiffre. Utiliser le tableau suivant pour trouver le bouton où appuyer.
