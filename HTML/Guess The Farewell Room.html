On the Subject of Guess The Farewell Room

But we aren't mountain climbers, never mind moon climbers!

When this needy activates, the display will show a room from Celeste's Chapter 9, "Farewell".

You must determine which room is being displayed by using the checkpoint selector at the left and number selector at the right.

Checkpoint names are abbreviated as follows:

  • SG = Singular
  • PS = Power Source
  • RM = Remembered
  • EH = Event Horizon
  • DT = Determination
  • SB = Stubborness
  • RC = Reconciliation
  • FW = Farewell

Examples of room names would be:

"PS-4" is "Power Source - 4th room", "RC-16" is "Reconciliation - 16th room", etc...

An incorrect guess will incur a strike.

For defusers and/or experts familiarized with Celeste's Chapter 9, the order of rooms will be bottom path of Power Source, using Key Skip and collecting the Golden Berry but not the Moon Berry.

Otherwise, determine which room is being displayed and find its debug name with the Farewell Map linked below, and use it on the table below to find the room name, formatted as Checkpoint-Number.
