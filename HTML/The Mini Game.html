On the Subject of The Mini Game

The machine might be smaller in scope than the other one, but it makes up for it in massive loss of time.

This machine contains 2 square displays. You can’t interact with any of the displays.

When the machine activates, the 2 square displays will light up and show numbers. There will be also static noise that will be produced while the machine is active. The minigame will last 3–5 turns (+1 extra free turn that you will always get at the end of the game). To temporarily solve the machine, follow the rules located on "The Number Game" manual.

There are a lot of things that differ from this machine and its counterpart:

  • The left display will initially show a blinking number between 10–99. This will be the maximum number that the machine will accept.
  • The right display will show the number remaining on the minigame. The number will be subtracted by the number you and the computer chose during each player’s turn.

When you type a number on the keyboard while the left display number is flashing, the flashing number will be replaced by the number you have typed. You can erase the number you have typed by pressing the "Backspace" key on the keyboard. When the number you have typed is fully erased, the blinking number will return to the display. You can only type 2 digits maximum on the left display.

You can submit the number you have typed by pressing the "Enter" key on the keyboard. This will be the number you submit on your turn. If any mistake occurs, the machine will immediately halt and give you a strike. Look at the "ERROR MEANING" part of the manual to locate the mistake that occured (which can only be seen on the machine log).
