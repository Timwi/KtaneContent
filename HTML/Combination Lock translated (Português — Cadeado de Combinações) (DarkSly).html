Isto parece-me um cadeado de combinações. Eu pensei que estava a desarmar esta bomba, não a desbloquea-la.


Ver Anexo B para Referência à identificação de baterias.
Ver Anexo 2-Fatores para Referência à identificação de 2-Fatores.

O módulo contém 3 botões. Uma seta para a esquerda e outra para a direita que rodam o cadeado, e um botão Repor '[EN=Reset]'.

Como seria de esperar num cadeado de combinações comum, são precisos 3 números para o desbloquear.
Para o primeiro número, roda o disco para a direita. Depois roda para a esquerda para o segundo número. E finalmente, roda novamente para a direita para o último número. Desse modo irá desbloquear o cadeado!

Se os números da sequência forem iguais, executa uma rotação completa de volta ao mesmo número.
Se te enganares ou ultrapassares o número correto na sequência pressiona o botão Repor para reajustar o cadeado ao estado inicial.

Como descobrir a combinação correta:

Cada número varia de 0 a 19. Consulta a tabela abaixo para determinar cada número da sequência. Mas cuidado com os códigos de 2-Fatores presentes, porque mudam periodicamente!

Primeiro número:
  1. Soma os digitos menos significativos dos dois códigos de 2-Fatores. Se não existirem códigos de 2-Fatores, soma o último dígito do Número de Série à quantidade de módulos já resolvidos na bomba.
  2. Adiciona a quantidade de baterias.
  3. Se o resultado for superior a 19, subtrai 20 até que o resultado fique entre 0 e 19.
Segundo Número:
  1. Soma os digitos mais significativos dos dois códigos de 2-Fatores. Se não existirem códigos de 2-Fatores, usa a quantidade de módulos existentes na bomba (incluíndo módulos carentes).
  2. Adiciona a quantidade de módulos já resolvidos na bomba.
  3. Se o resultado for superior a 19, subtrai 20 até que o resultado fique entre 0 e 19.
Terceiro Número:
  1. Soma os dois primeiros números.
  2. Se o resultado for superior a 19, subtrai 20 até que o resultado fique entre 0 e 19.
