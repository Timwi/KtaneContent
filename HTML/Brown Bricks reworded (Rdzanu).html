Keep Talking and Nobody Explodes Mod Brown Bricks

On the Subject of Brown Bricks

From Minecraft.

This module contains four buttons that are all blocks from Mario games.

To start off find which block is missing. Use this block in the first table to get a position you need to look at. Next look what block is at given position.

Use this block in the second table to get your third block. This block will be referenced as the block after the (→), use its position on the module in the third table. If this block isn’t on the module you need to look at the otherwise row.

Press the correct block from the third table to solve the module. Pressing an incorrect block will cause a strike and reset the module.

If at any point in the third table you have to press the block that isn’t present on the module, you press the block from the "Otherwise, if the above block does not exist" row.
