On the Subject of Legs

Hey, what’s up guys y’all got legs?

Pink dog wants to know if you Got Legs? Count how many legs.

Use the arrow buttons to scroll through the five pages.

Each page has a count (the amount of the animal) on the top left screen, an animal on the top right screen (sometimes with a unique quirk) and a quirk on the bottom screen.

Using the animal and quirk tables below, count how many legs the animals have in total and input it using the number buttons.

The button labeled S submits the number entered, and the button labeled C clears the current input.

If the final answer is less than zero, submit zero instead.
