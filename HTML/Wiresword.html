On the Subject of Wireswords

These wires don't seem to meet PAT testing requirements: adequate insulation, up to date labelling, secure connections with grounding leak prevention.

  • A panel can have 2-5 wires on it, each of which is an encoded letter.
  • Use the buttons above and below each panel to cycle through the possibilities for each panel.
  • Only one combination of panels will match a password below with wires of the same colour.
  • Only the one correct wire on each panel in this combination needs to be cut to disarm the module.
  • Wire ordering begins with the first wire at the top of the left set of ports.

Cutting Wires:

2 wires:

  • If both encoded letters are present in the serial number, cut the wire that is not an encoded letter in the password.
  • Otherwise, if both wires are the same colour, cut the wire that is an encoded letter in the password.
  • Otherwise, if the panel number is even, cut the last wire.
  • Otherwise cut the first wire.

3 wires:

  • If all wires are different colours and are encodings of different letters, cut the second wire.
  • Otherwise, if there is more than one black wire, cut the first wire.
  • Otherwise, if there are no encoded vowels, cut the last wire that is not an encoded letter in the password.
  • Otherwise cut the wire that is an encoded letter in the password.

4 wires:

  • If the wire that is an encoded letter in the password is the only one of its colour, cut the third wire.
  • Otherwise, if there are less than three distinct wire colours, cut the last wire.
  • Otherwise, if there is exactly one blue wire, cut it.
  • Otherwise, cut the first wire that is not an encoded letter in the password.

5 wires:

  • If the position of the wire that is an encoded letter in the password is the same as the panel number, cut the last wire.
  • Otherwise, if there is more than one of the same encoded letter that is not the encoded letter in the password, cut the third wire.
  • Otherwise, if there are no red wires, cut the second wire.
  • Otherwise cut the fourth wire.
