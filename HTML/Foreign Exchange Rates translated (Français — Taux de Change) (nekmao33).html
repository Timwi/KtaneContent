À propos des Taux de Change

Si les bombes étaient des courtiers en bourse...

Pour désamorcer ce module, l’expert doit disposer d’un appareil capable de se connecter au World Wide Web.

Ce module permet de se connecter à Internet et d’interroger l’état des taux de change. Il devrait y avoir une grille de trois fois trois touches, chacune avec une diode électroluminescente. N’APPUYER SUR AUCUNE TOUCHE DE CE MODULE PENDANT QUE LES DIODES ÉLECTROLUMINESCENTES CLIGNOTENT DANS L’ORDRE.

Les touches sont regroupées par lignes :

Rangée du haut : Code alphabétique ISO 4217 pour la devise de base.[1][2]
Rangée du milieu : Code alphabétique ISO 4217 pour la devise cible.[1][2]
Rangée du bas : Valeur de la devise à convertir.

Toutes les diodes électroluminescentes sont devenues vertes :

Entrez l’URL suivant dans votre appareil compatible Internet :
http://fer.eltrick.uk/latest?base=XXX&symbols=YYY
Remplacer XXX par le code alphabétique ISO 4217 pour la devise de base.
Remplacer XXX par le code alphabétique ISO 4217 pour la devise cible.
Vous allez recevoir les données dans un format JSON, chercher {“YYY”: NOMBRE}, où NOMBRE sera le taux de change. En utilisant le taux de change, convertir le chiffre de la ligne du bas dans la devise cible, arrondir ce chiffre vers le bas et noter le 2e chiffre à gauche[4]. Appuyer sur la Ne touche où N est le nombre noté (compter les touches de gauche à droite et de haut en bas)[5].

Toutes les diodes électroluminescentes sont devenues rouges :

Dans ce cas, le module n’a pas réussi à interroger les taux de change actuels. Obtenir le code numérique ISO 4217 pour le pays de la monnaie cible et noter le 2e chiffre à droite. Appuyer sur la Ne touche où N est le nombre noté (compter les touche de gauche à droite et de haut en bas)[5].

[1] : S’il y a plus d’une pile sur la bombe, le code de la devise de base et le code de la devise cible sont échangés.
[2] : Il peut s’agir plutôt du code numérique ISO 4217.
[3] : Notez que les taux de change sont mis à jour vers 16h00 CET.
[4] : Dans le cas où la devise convertie est inférieure à 10, le nombre indiqué est 0.
[5] : Dans le cas où le nombre noté est 0, appuyer sur la touche supérieure gauche.
