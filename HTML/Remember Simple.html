On the Subject of Remember Simple

Do I just keep pressing this button?

This module has a black background and a blue button. Every time a non-ignored module is solved, this button will have to be pressed exactly once, including before the first module solve.

If the button is pressed too few or too many times, it will strike. After a strike, the current count will be considered correct, so don’t push the button again until the next solve. If it is pressed consistently and is pressed when the last non-ignored module is solved, it will solve.

Note that when the last module is completed, the module will automatically go into the last click stage.
