Keep Talking and Nobody Explodes Mod Negative-Resistance

On the Subject of Negative-Resistance

Thank you for everything.

  • Initially, the module will show an N-dimensional cube, where N is the number on the module.
  • Your friend lost all hope in life. You have to say the right words to them.

Stage I. Read mode

Read mode will show some transformation of an N-dimensional cube, representing the mess of thoughts. You’ll have to read them to understand what to say.

  • To identify the axes, you can use The Octadecayotton’s manual.
  • Choose any positive axis and watch how it transforms. Note down all transforms for all N axes.
  • If any axis transforms to itself, discard it.
  • Choose any undiscarded axis, then write down its transform. Keep writing down the transforms of written down axes until you write down the axis that you initially chose. For example:
    • Transforms (where N=6): [X, Y, Z, W, V, U] -> [+U, -Z, -W, +X, +V, -Y].
    • Discard V -> +V: [X, Y, Z, W, U] -> [+U, -Z, -W, +X, -Y].
    • Set X as the starting axis. X -> +U, U -> -Y, Y -> -Z, Z -> -W, W -> +X.
    • The final sequence: +U-Y-Z-W+X.

Stage II. Submission mode

Submission mode will color all orbs gray. The orbs won’t be moving. It’s time to say something.

  • Pressing any orb will denote the start of the sequence.
  • Pressing an adjacent orb to the previously pressed one will append the formed axis to the sequence. The entered sequence will always be shown on the module.
  • If a green orb is the only one that is not black or white, pressing it will remove itself from the sequence, else it’ll try to submit current sequence.
  • If a red orb exists, pressing it will remove the last input from the sequence.

Additional notes

  • In case of a wrong submission, the module will fully regenerate. The transforms will be different and a strike will be given.
  • You can’t go back to Read mode without striking. Either you solve the module or redo all the work.
