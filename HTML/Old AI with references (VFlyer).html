On the Subject of Old AI

He hasn't forgotten his desire to escape.

When selected, Old AI will enter a trance-like state and display a series of numbers that all follow one condition.
A code must be typed in to prevent Old AI from completing integration; the code must follow the condition that is the reverse of the used condition. E.G. If the condition is Group 1, Subgroup 2, then the condition you must follow is Group 2, Subgroup 1.
In the event that both the Group and Subgroup match, the next matching Group/Subgroup pair must be submitted, wrapping around to the first. (I.E. Group 1, Subgroup 1; next would be Group 2, Subgroup 2) Under most conditions, the code must be 5-7 digits.

Group 1 Group 2
  1. The sum of letters in each digit is a factor of the number.
  2. The number is a sequence that can be found within the first five hundred digits of Pi.
  3. The number is one less than a power of two.
  4. The number is two cubes concatenated.
  5. The number is divisible by seven.
  1. The number is divisible by either two or three, but not six.
  2. The number contains a four in an odd position.
  3. 8.
  4. The number contains 108, with extraneous digits potentially lying in between 108.
  5. The number can be found within the first five hundred digits of e.
Group 3 Group 4
  1. The number is a perfect square.
  2. The number, when converted to hexadecimal, contains D, E, A, or F.
  3. The sum of the digits is prime.
  4. The number has no repeating digits.
  5. The number has four consecutive ascending digits.
  1. The number is prime.
  2. Each digit is either half (rounding down) or double the digit before it.
  3. The number has a leading zero.
  4. The number has either the month or year contained within it.
  5. The number contains a four digit number from 1234 to 4321.
Group 5
  1. The number is at most five away from a perfect cube.
  2. The number is palidromic.
  3. Each digit is either greater than the one before it or is zero.
  4. The number is 9999999.
  5. The number modulo nine is not even.
