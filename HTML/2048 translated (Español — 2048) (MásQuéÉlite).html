Mod de Keep Talking and Nobody Explodes 2048

En caso de 2048

Espera... ¿Este juego no estaba basado en otro? ¿Como "1024" o algo así?

Cómo jugar 2048

Usa las flechas del módulo* para mover todas las casillas del tablero.

Las casillas se moverán hasta alcanzar otra casilla o un lado del tablero. No obstante, si una casilla choca con otra del mismo valor, se fusionarán y se tornarán una casilla con el doble del valor.

Consiguiendo el módulo

Para resolver el módulo, mezcla casillas hasta que llegues a la casilla 256. El módulo se desactivará automáticamente cuando alcances esa casilla.

Sin embargo, si quieres, puedes seguir intentando conseguir casillas de mayor valor, por ejemplo, la 2048.

Si te quedas sin movimientos disponibles, el módulo dará un strike y se reiniciará.

Si piensas que te quedarás sin movimientos, o simplemente necesitas empezar de 0, puedes pulsar el botón cuadrado* (con la letra "R") para reiniciar el módulo sin ningún inconveniene.


*Alternativamente, puedes usar las flechas de tu teclado para mover las casillas, y usar la tecla R para reiniciar.
