モジュール詳細：ギリシャ文字グリッド

「逆さまのＬ」じゃないって！

  • このモジュールには、x軸のA~D及びy軸の１～４のラベルが付いた4×4グリッド上に３つの色付きギリシャ文字(１８種類)がある。
  • 表を上から下に見た順で優先順位をつけ、その順で各ルールに従って位置を変更する。
  • 各ギリシャ文字の判別は下の付録を使用する。
  • いずれかの文字が不適切な場所にあるときに送信ボタンを押すとミスが記録され、新しい文字に変更される。
  • 文字をクリックして選択し、選択した文字をモジュールの矢印キーで移動する。

大文字のアルファ

  1. このモジュールに小文字のシグマがある場合、A4に移動する。
  2. それ以外の場合、この文字が黄色の場合、B1に移動する。
  3. それ以外の場合、点灯していないSIGインジケーターまたは点灯していないINDインジケーターがある場合、D3に移動する。
  4. それ以外の場合、他の2文字の色が同じ場合、C3に移動する。
  5. それ以外の場合、初期位置のままにする。

小文字のアルファ

  1. 初期位置がD列の場合、2スペース左に移動する。
  2. それ以外の場合、3つ以上のバッテリーがある場合、D1に移動する。
  3. それ以外の場合、この文字が緑色であり、モジュールにデルタ(大文字／小文字)がある場合、C2に移動する。
  4. それ以外の場合、点灯したCLRインジケーターがある場合、A1に移動する。
  5. それ以外の場合、A3に移動する。

大文字のベータ

  1. この文字の色がシアンの場合、D1に移動する。
  2. それ以外の場合、この文字の初期位置が段３の場合、1スペース上に移動する。
  3. それ以外の場合、モジュールに小文字があり、シリアルナンバーの最後の数字が奇数の場合、B4に移動する。
  4. それ以外の場合、モジュールに緑色の大文字がある場合、C2に移動する。
  5. それ以外の場合、C4に移動する。

小文字のベータ

  1. 初期位置がA3の場合、1スペース下、２スペース右に移動する。
  2. それ以外の場合、いずれかの白い文字(この文字も含む)の初期位置がA列の場合、A2に移動する。
  3. それ以外の場合、この文字の色がマゼンタまたはシアンの場合、D4に移動する。
  4. それ以外の場合、DVI-Dがあり、RJ-45がない場合、B2に移動する。
  5. それ以外の場合、B1に移動する。

大文字のガンマ

  1. シリアルナンバーの英字がアルファベットの前半にある場合、D2に移動する。
  2. それ以外の場合、空のポートプレートがある場合、C2に移動する。
  3. それ以外の場合、このモジュールの全ての文字の色がマゼンタの場合、A1に移動する。
  4. それ以外の場合、この文字の色がシアンであり、初期位置が段４の場合、2スペース上に移動する。
  5. それ以外の場合、A2に移動する。

小文字のガンマ

  1. シータがある場合(大小／色指定無し)、この文字をC1に移動する。
  2. それ以外の場合、この文字の色がシアンではなく、シリアルナンバーに母音が含まれている場合、B3に移動する。
  3. それ以外の場合、初期位置がC列の場合、この文字を１スペース左に(シリアルナンバーの最後の数字が４以下の場合は2スペース左に)移動する。
  4. それ以外の場合、シリアルナンバーの最後の数字が素数の場合、C3に移動する。
  5. それ以外の場合、D4に移動する。

大文字のデルタ

  1. RCAステレオがある場合、B2に移動する。
  2. それ以外の場合、バッテリーの数がシリアルナンバーの最後の数字よりも多い場合、A3に移動する。
  3. それ以外の場合、この文字の色が黄色またはマゼンタではない場合、D4に移動する。
  4. それ以外の場合、モジュールに緑色の小文字のオメガがある場合、A1に移動する。
  5. それ以外の場合、C1に移動する。

小文字のデルタ

  1. この文字が唯一の小文字の場合、C3に移動する。
  2. それ以外の場合、シリアルナンバーの最後の数字＋解除されたモジュールの数が１０より大きい場合、B3に移動する。
  3. それ以外の場合、いずれかの初期位置がA3の文字がある場合、そこに移動する。
  4. それ以外の場合、この文字の色が唯一のシアンの場合、B4に移動する。
  5. それ以外の場合、C2に移動する。

大文字のシータ

  1. 単１バッテリーの数が単３バッテリーの数よりも多い場合、A4に移動する。
  2. それ以外の場合、点灯していないCARインジケーターがある場合、B列に移動する。段については、シリアルナンバーの最後の数字を４で除算した余りに１を加算した段を使用する。
  3. それ以外の場合、モジュールの３つの文字の色が全て異なる場合、D列に移動する。段については、ポートプレートの数を４で除算した余りに１を加算した段を使用する。
  4. それ以外の場合、モジュールに小文字のシータがある場合、D1に移動する。
  5. それ以外の場合、B2に移動する。

小文字のシータ

  1. 初期位置が他の２つの段と異なる場合、D3に移動する。
  2. それ以外の場合、この文字の色が白の場合、C列に移動する。段については、ミスの数を４で除算した余りに１を加算した段を使用する。
  3. それ以外の場合、解除されたモジュールの数がミスの数以下の場合、初期位置から変更しない。
  4. それ以外の場合、PS/2があるか、いずれかのポートが重複している場合、B1に移動する。
  5. それ以外の場合、A2に移動する。また、残り分数が奇数の時のみに送信する必要がある。

大文字のラムダ

  1. シリアルナンバーにAが含まれている場合、A1に移動する。
  2. それ以外の場合、シリアルナンバーにBが含まれている場合、B1に移動する。
  3. それ以外の場合、シリアルナンバーにDが含まれている場合、D1に移動する。
  4. それ以外の場合、シリアルナンバーにLまたはMが含まれている場合、C1に移動する。
  5. それ以外の場合、C4に移動する。

小文字のラムダ

  1. 点灯したインジケーターが２つ以上ある場合、D3に移動する。
  2. それ以外の場合、この文字がモジュールの他の一つと同じ色の場合、B2に移動する。
  3. それ以外の場合、ポートプレートよりもバッテリーが多い場合、A1に移動する。
  4. それ以外の場合、モジュールの１文字が黄色(この文字以外)の場合、黄色の文字の正しい位置に移動する。
  5. それ以外の場合、D1に移動する。

大文字のパイ

  1. 今日が円周率の日(3/14)の場合、初期位置のままにしておく。
  2. それ以外の場合、シリアルナンバーにPまたはIがある場合、B4に移動する。
  3. それ以外の場合、初期位置の段に別の文字がある場合、D2に移動する。
  4. それ以外の場合、初期地点がいずれかの角の場合、A3に移動する。
  5. それ以外の場合、列Aに移動する。段については、残り時間の分数を４で除算した余りに１を加算した段を使用する。

小文字のパイ

  1. シリアルナンバーの最後の数字が３／１／４のいずれかの場合、初期位置のままにしておく。
  2. それ以外の場合、バッテリーホルダーの数が３／１／４のいずれかの場合、列Dに移動する。段については、バッテリーホルダーの数に対応した段を使用する。
  3. それ以外の場合、インジケーター(点灯／消灯)が３／１／４の場合、列Bに移動する。段については、インジケーターの数に対応した段を使用する。
  4. それ以外の場合、解除されたモジュールの数が３／１／４のいずれかの場合、列Cに移動する。段については、解除されたモジュールの数に対応する段を使用する。
  5. それ以外の場合、C3に移動する。

大文字のシグマ**

  1. バッテリーの数と現在のミスの数が合計が５以上の場合、A4に移動する。
  2. それ以外の場合、点灯したインジケーターとポートプレートの数の合計が５以上の場合、C2に移動する。
  3. それ以外の場合、点灯していないインジケーターとシリアルナンバーの最後の数字の合計が５以上の場合、B1に移動する。
  4. それ以外の場合、解除されたモジュールとバッテリーホルダーの数の合計が５以上の場合、D1に移動する。
  5. それ以外の場合、B3に移動する。
**点灯したSIGインジケーターがある場合、以下のルールをすべて無視し、初期位置のままにしておく。

小文字のシグマ**

  1. この文字の色が緑色の場合、B1に移動する。
  2. それ以外の場合、パラレルポートがある場合、B4に移動する。
  3. それ以外の場合、他の文字のちょうど１つに大文字のラムダがある場合、その文字の初期位置に移動する。
  4. それ以外の場合、初期位置が段４の場合、初期位置のままにしておく。
  5. それ以外の場合、A1に移動する。
**点灯したSIGインジケーターがある場合、以下のルールをすべて無視し、初期位置のままにしておく。

大文字のオメガ

  1. この文字の色がマゼンタの場合、C4に移動する。
  2. それ以外の場合、シリアルナンバーにZが含まれている場合、D2に移動する。
  3. それ以外の場合、モジュールに大文字のアルファがある場合、A3に移動する。
  4. それ以外の場合、シリアルナンバーの最後の数字が素数ではない場合、B3に移動する。
  5. それ以外の場合、D1に移動する。

小文字のオメガ

  1. 初期位置がC4の場合、２スペース上の、１スペース左に移動する。
  2. それ以外の場合、シリアルナンバーにWが含まれている場合、A2に移動する。
  3. それ以外の場合、バッテリーが０の場合、C3に移動する。
  4. それ以外の場合、この文字の色がシアンで、他の文字のちょうど１つが白の場合、D4に移動する。
  5. それ以外の場合、初期位置のままにしておく。
