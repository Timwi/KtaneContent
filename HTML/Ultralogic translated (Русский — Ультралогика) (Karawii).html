Об Ультралогике

Мы случайно не упустили гиперлогику?

  • На модуле есть 3 квадрата: A, B и C. Каждая пара квадратов соединена с третьим квадратом с помощью логических элементов. Зелёный цвет означает "правда", а красный - "ложь".
  • Нажимая на логические элементы, они будут изменяться на другие.
  • Когда все три логических элемента настроены так, чтобы все утверждения были верными, нажмите на кнопку "Submit".
  • В ответе не может быть повторяющихся операторов.
Логическое "И" Если оба входных утверждения верны, то результат будет истинным, иначе - ложным.
Логическое "ИЛИ" Если хотя бы одно из входных утверждений верно, то результат будет истинным, иначе - ложным.
"Исключающее ИЛИ" Если хотя бы одно из входных утверждений верно, то результат будет истинным, иначе - ложным.
Логическое "НЕ И" Если оба входных выражения верны, то результат будет ложным, в остальных случаях - истина.
Логическое "НЕ ИЛИ" Если хотя бы одно из входных выражений верно, то результат будет ложным, иначе - истина.
"Исключающее НЕ ИЛИ" Если только одно входное выражение верно, то результат будет ложным, иначе - истина.
