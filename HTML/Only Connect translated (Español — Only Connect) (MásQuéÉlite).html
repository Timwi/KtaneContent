Mod de Keep Talking and Nobody Explodes Only Connect

En caso de Only Connect

Las letras griegas son muy pretenciosas, así que usamos jeroglíficos egipcios.

Este módulo consta de dos fases.

Fase 1: jeroglíficos egipcios

  • El módulo muestra seis jeroglíficos egipcios de forma 3×2, y un nombre de equipo.
  • Determina cuántos criterios cumple cada jeroglífico.
  • Selecciona el jeroglífico egipcio que cumpla un número diferente de criterios que los demás.
  • El criterio nº1 se cumple si dicho jeroglífico está en la posición especificada.
  • El criterio nº2 se cumple si el carácter del número de serie en la posición especificada está en el nombre del equipo. Si ese carácter es un número, pásalo a letra (0=Z, 1=A, 2=B, ..., 9=I). (usa el alfabeto inglés)
  • El criterio nº3 se cumple si la en bomba hay al menos uno del puerto especificado.
