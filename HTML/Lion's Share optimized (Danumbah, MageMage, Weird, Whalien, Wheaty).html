On the Subject of Feeding Big Cats

Let’s solve this thing in under a minute.

A Lion's Share module contains a pie chart with multiple lions' names and a year under the status light. To disarm the module, give each lion the correct percentage of the total.

This manual will help calculate the entitlement for each lion. Visiting lions can be found on this first page and all other lions can be found under the section with the corresponding year.

Each serial number letter matching a character in the lions name adds 1, if there are duplicate serial number characters add 1 for each. Duplicates in the lion's name do not matter.
Each lit indicator with the first letter of a name adds a lion specific value.

When calculating percentages, always round down and allocate the rest to the lion on the red slice.
