モジュールを解除するには、正しいコードを入力する必要がある。上部に表示された数字を使用し、以下のルールに従ってコードを特定する。


コードの決定
シリアルナンバーの最初の数字と最後の数字が等しく、バッテリーがない場合は、表示された数字をそのまま入力する。
それ以外の場合、CLRインジケーターがある場合は、表示された数字を８で除算する。
それ以外の場合、シリアルナンバーに「X」「Y」「Z」がある場合は、表示された数字を２０で除算する。
それ以外の場合、ポートが５つ以上ある場合は、表示された数字を３０で除算する。
それ以外の場合、バッテリーがない場合は、表示された数字を４２で除算する。
それ以外の場合、点灯したインジケーターが点灯していないインジケーターよりも多い場合、表示された数字を６９で除算する。
それ以外の場合、表示された数字を３で除算する。
  • 小数点がある場合は、小数部分をそのまま削除して整数のみを入力する。
