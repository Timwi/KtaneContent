On the Subject of Memory Character

Call ‘em like you see ‘em.

A Memory Character module has a display which displays a Kanji character, along with Yes and No buttons above it.

Press the No button if that character has not been displayed by the module before. This will cause a new character to be displayed. Press the Yes button if that character has been displayed by this module before. If it has, the module will solve.

Pressing an incorrect button will cause the module to record a strike. The module will then reset and discard all the previous Kanji characters prior to the reset. A correct Yes button press is guaranteed to occur within 10 characters.
