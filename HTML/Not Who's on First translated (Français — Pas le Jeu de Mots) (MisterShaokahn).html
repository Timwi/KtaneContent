Pas à propos du Jeu de Mots

Ce module ressemble à quelque chose sorti des égouts d’une usine de paillettes, ce qui est amusant parce qu’il s’agit en fait d’une « belle merde ». Je serai bref, car les mots ne font qu’allonger le texte.

Appuyer sur le bon bouton pour faire passer le module à l’étape suivante. Terminer les cinq étapes pour désamorcer le module. Appuyer sur un bouton incorrect réinitialise le module à l’étape 1. Les positions des boutons sont numérotées dans l’ordre de lecture.
