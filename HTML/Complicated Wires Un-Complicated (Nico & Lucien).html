On the Subject of Uncomplicating Wires

Who would think that wires this complex could totally refrain from the Venn Diagram?

White Wires

  • Only white/white + star -> Cut
  • LED -> Don't cut
  • Star + LED -> 2+ batteries

Red Wires

  • Only red or red + white -> Last SN digit is even
  • Star -> Cut
  • Any LED -> 2+ batteries

Blue Wires

  • Only blue or blue + white -> Last SN digit is even
  • Star -> Don't cut
  • Any LED -> Parallel port

Red + Blue Wires

  • Only blue + red -> Last SN digit is even
  • Star -> Parallel port
  • LED -> Last SN digit is even
  • Star + LED -> Don't cut
