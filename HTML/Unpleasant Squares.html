Nothing to see here, try over there.

Tilting the module will change its colour. There are 25 cells arranged into a 5×5 grid that are each one of 5 colours: red, yellow, jade, azure, and violet.
The module will turn black if it is tilted too far in any direction.

The 5×5 grid can be found in the 16×16 grid below, except for the centre cell, which is hidden, and 3 of the other cells whose colours do not match.

The module itself is divided into a 5×5 grid of cells that each correspond to the colour that shares its position.

Select the mismatched cells in alphabetical order of the cells in the grid that match the colour of the hidden centre cell.
