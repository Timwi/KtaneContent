On the Subject of Scrambled Periodic Words

Fun Fact! Titin has the longest chemical name! It's methionyl­threonyl­threonyl­glutaminyl­alanyl­prolyl­threonyl­phenylalanyl­threonyl­glutamin...yl­isoleucyl­histidyl­isoleucyl­arginyl­seryl­isoleucine!

Presented is an initially blank display, a display with 3-digit numbers below it, a QWERTY keyboard and a stage counter.

For each number, find the element in the Periodic Table of the Elements with that atomic number and note its symbol. For three stages, rearrange these symbols into a valid English word, then enter it into the module using the keyboard. You may not rearrange letters within two-letter symbols: you may only rearrange the symbols themselves. Note that a set of symbols may anagram into several words. In this case, any valid anagram is accepted.
