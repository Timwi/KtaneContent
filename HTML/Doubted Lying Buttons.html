Doubting the Subject of Lying Buttons

Six stars of confusion. Where are my Blue Crystals when I need it?

This dark-gray module consists of a 3×3 grid of buttons with clues displayed in the middle of the button. To disarm the module, press all buttons that are safe.
The hinges on the module denote how many unsafe buttons are present on the module, in addition to one rare hinge:

  • Top-Left: Exactly 1 button is not safe to press.
  • Top-Right: Exactly 2 buttons are not safe to press.
  • Bottom-Left: Exactly 3 buttons are not safe to press.
  • Bottom-Right: The module generated a puzzle with multiple possible positions of unsafe buttons. Press ALL buttons that you know are safe, then take a guess on which one is safe. This hinge may disappear on a new puzzle.

If multiple hinges out of top-left, top-right, and bottom-left are present, the solution can have any of those amounts. I.E., if the top-right and bottom-left hinges are present, the solution can have either 2 or 3 unsafe buttons.

The clues displayed on the module can be decrypted by the table on the next page.
If a button pressed is not safe, the module will strike and generate a new puzzle.
On ambiguous puzzles, the module will ONLY strike if an unsafe button is pressed while at least one knowable safe button has not been pressed.

Initially all buttons tell the truth. If a safe button shares the same color as an unsafe button, its clue will lie instead. The table provided shows possible options of buttons that can show up.
