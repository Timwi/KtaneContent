On the Subject of Wavetapping

Perhaps the craziest conference call I’ve ever heard. Don’t hang up. DO NOT HANG UP.

  • This module contains a 11×11 pixel display, a 4×4 display to the right of that, and a submit button up top.
  • You are required to enter a specific pattern based on the color of the 11×11 display.
  • This manual does not disarm the module, butgives instructions on how to input each of the 158 patterns.
  • If this manual has an incorrect instruction for a pattern, please ping @Danny7007#7431 on Discord so I can correct the manual :)
    NOTE: Since no pattern contains pixels in the first and last columns and rows, "a" represents the second column and "i" represents the 10th, while 1 represents the second row and 9 represents the 10th.
  • You also might want to check your pattern before you submit it because this is prone to error.
