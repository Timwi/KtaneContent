Step 2: Decrypt Displayed Digits

  • Split the encrypted numbers on the display into 5 number pairs in reading order.
  • Convert each number to a digit using the field from Step 1. Each number’s corresponding digit can be found in the cell at the number’s position, counting in reading order (top-left is cell 1).
  • Each pair will make a new two-digit number.
  • Modulo each two-digit number by 26 and convert them into letters using A1Z26. This will form a keyword.
