  • Un module Molettes Biaisées contient 3 afficheurs numériques et un bouton submit.
  • Sur la base de l'affichage actuel sur les molettes, soumettre les chiffres corrects en fonction des sections de règles suivantes. Chaque section est étiquetée avec la molette à laquelle elle s'applique.
  • Une fois que toutes les règles ont été appliquées, si le nombre est inférieur à 0, ajouter 10. Sinon, si le nombre est supérieur à 9, soustraire 10. Répéter cette opération jusqu'à ce que le chiffre soit compris entre 0 et 9.
  • Si les mauvais chiffres sont saisis, les molettes tourneront automatiquement et une erreur sera enregistrée.
REMARQUE : Le chiffre d'origine est le chiffre avant qu'il n'ait été modifié.
Toutes les molettes
Remplacer 2 par 5 et 7 par 0.
Pour chaque indicateur allumé, ajouter 1 au chiffre et soustraire 1 pour chaque indicateur éteint.
Si le chiffre est un multiple de 3, ajouter 4.
Sinon, si le chiffre est supérieur à 7, le multiplier par 2.
Sinon, si le chiffre est inférieur à 3 et que c'est un chiffre pair, le diviser par 2.
Sinon, s'il y a un port RCA ou PS/2 sur la bombe, ignorer les autres règles de cette section.
Sinon, prendre le chiffre original et ajouter le nombre de piles sur la bombe.
1re molette
Si le chiffre est pair et supérieur à 5, le diviser par deux.
Sinon, si le chiffre est premier, ajouter le chiffre le plus à droite dans le numéro de série.
Sinon, s'il y a un port parallèle sur la bombe, le multiplier par -1.
Sinon, si le chiffre original à sa droite est impair, laisser ce chiffre inchangé.
Sinon, soustraire 2.

2e molette
S'il y a un indicateur BOB éteint, laisser ce chiffre inchangé.
Sinon, si le chiffre est 0, ajouter le chiffre original du premier afficheur.
Sinon, si le chiffre est dans la suite de Fibonacci, ajouter le nombre suivant de la suite de Fibonacci en fonction de la première occurrence du nombre.
Sinon, si le nombre est supérieur ou égal à 7, ajouter 4.
Sinon, le multiplier par 3.
3e molette
S'il y a un port Série sur la bombe, ajouter le plus grand chiffre du numéro de série.
Sinon, si le chiffre original est le même que n'importe lequel des autres chiffres de l'original, laisser ce numéro inchangé.
Sinon, si le chiffre est supérieur ou égal à 5, additionner tous les chiffres individuels sous la forme binaire du chiffre original pour le nouveau nombre.
Sinon, ajouter 1 au chiffre.
