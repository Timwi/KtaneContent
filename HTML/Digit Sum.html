On the Subject of Digit Sum

Your bomb status equals hex(21781+(digit sum of 21e4)+(digit sum of 31e4))

This module consists of three displays and an input keyboard. The upper display shows the number “K” and the display below it shows the number “X”. Find the number “Z” and enter it using the input keyboard. The number “K” changes over time in such a way that the answer never changes.

Input keyboard also has button “<” to delete input and button “S” to submit your answer. The entry will be shown in the third display.

A = (digit sum of X) + 1

Y = sum of:

  • 19 * Remaining time in minutes
  • 13 * Batteries
  • 17 * Modules
  • 5 * DVI ports
  • 2 * Sum of most significant digit of each two factor code
  • 7 * Unlit indicators
  • 11 * Last digit of serial number
  • 3 * Starting time in minutes

D = (Y + K) modulo A

Z = smallest number such that digit sum of ((X + Z) modulo 1'000'000) is equal to D
