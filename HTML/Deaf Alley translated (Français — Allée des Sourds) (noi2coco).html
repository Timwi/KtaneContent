À propos de l’Allée des Sourds

Tous les noms qui avaient un sens ont été pris, alors voici un nom de module qui n’a aucun sens.

Voir l’Annexe VIDE dans Impasse pour l’identification des modules vides.

Le module apparaît vide, mais lorsqu’on le survole, il peut produire un son. Les sections du module qui émettent un son créent une forme. Toucher tout autour du module pour déterminer la forme, puis appuyer sur les 4 régions du module dans le bon ordre pour résoudre le module. Si une région incorrecte est pressée, une erreur sera enregistrée et toutes les régions correctement pressées devront être pressées à nouveau.
