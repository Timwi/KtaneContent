Keep Talking and Nobody Explodes Mod 顏色方格

關於【顏色方格】模組

順序亂成一團，只要找到顏色的規律就能明白一切。

有關顏色方格模組系列的識別法請參照附錄。

  • 按下全部正確類別的方格以進入下一階段。
  • 按下的方格會發出白光，全部方格發出白光即可拆除模組。
  • 開始時，按下數量最少的顏色類別，如出現數量相等的情況，則你看錯手冊了。請參照下頁的識別法。
  • 然後，利用下表決定下階段應該按下的類別
  • 所謂【類別】，就是某顏色的所有方格，或包含非白色方格的最左列/最上行的所有非白色方格
  • 按下錯誤的方格會導致一次失誤，並重置模組。
  • 已變成白色的方格會持續亮著白色，還沒變成白色的方格可能會改變顏色。

附錄：類似 【顏色方格】模組的識別法

If there are 4 of the 5 colors (Red, Yellow, Blue, Green, and Magenta) occurring exactly four times each with exactly 1 color missing:
Bicolored Squares
如果一種顏色出現次數少於任何其他顏色（沒有同數），完成按下動作後，按下的方格變為白色：
顏色方格 (Colored Squares)
如果有三種顏色剛好出現兩次、兩種顏色剛好出現五次：
掉色方格 (Decolored Squares)
如果有四種顏色剛好出現一次：
變色方塊 (Discolored Squares)
If there are exactly 3 colors that occur only once:
Isocolored Squares
如果有紅、藍、綠、黃及洋紅，且沒有重複的顏色：
比色方格 (Juxtacolored Squares)
如果一種顏色出現次數少於任何其他顏色（沒有同數），完成按下動作後，按下的方格變為黑色：
偽顏色方格 (Not Colored Squares)
If there are 16 squares of a uniform color:
Overcolored Squares

如果有紅、藍、綠、黃及洋紅以外的顏色，且有重複的顏色：
