On the Subject of The Code

What kind of code? It could be a letter code! It could be a digit code! Heck, it could even be an alphanumeric code! ffffffffffffffffffffffffffffffffffffffff.

To defuse the module, you need to input the correct code. Take the displayed number at the top and follow the rules below.


Determining the code.
If the first and last digit of the serial number are equal and there are no batteries, submit the displayed number.
Otherwise, if there is a CLR indicator, divide the displayed number by 8.
Otherwise, if the serial number contains "X", "Y" or "Z", divide the displayed number by 20.
Otherwise, if there are 5 or more ports, divide the displayed number by 30.
Otherwise, if there are no batteries, divide the displayed number by 42.
Otherwise, if there are more lit than unlit indicators, divide the displayed number by 69.
Otherwise, divide the displayed number by 3.
  • Remove all of the decimals and submit this number.
