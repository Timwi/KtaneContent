On the Subject of Cruel Countdown

Division is illegal.

Assign the letters A-F to the numbers from left to right. One of the equations below will always equal the target value.

  • (F * B) + (A - E) + (C - D)
  • (A - B + F) * (C - D)
  • (A - B + F) * (D - C)
  • (A - B + F) + (D + C)
  • ((A + E + C) - B) + (D * F)
  • (F * B) + (E * D) - (A + C)
  • ((B - D - E) * F) + A - C
  • (B + A) - (C + D) + (E * F)
  • (A * E) + (B - F) - (C + D)
  • ((F + E) * (C - D) + A - B)
  • ((C * (E + F)) - D) + A
  • ((B + F - C) * A) + E - D
  • (C - D) * (E + F)
  • (A * F) + E + D
  • (F * E * D) - (A + C)
  • ((B + F) * E) - D
  • ((B - E) + (A - C)) * F
