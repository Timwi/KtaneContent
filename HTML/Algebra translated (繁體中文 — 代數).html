“為何我們要學習代數呢？我們又不會在日常生活中用得到...” 你記得你說過這句話嗎？

  • 這個模組有一個數字鍵盤和兩個螢幕
  • 在三個階段中，將會依序顯示變數a, b 和 c 。
  • 解出a，b 或 c 的值並輸入答案以進入下一階段。
  • 變數x，y 及 z 都有機會在每條方程式中出現。
    用以下表格各列分別找出它們的值︰
x = n y = i − p z = m + t × d
n = 機身序號所有數字總和 i = 機身標籤數量
p = 機身訊號埠數量		 m = 炸彈模組數量
t = AA 電池數量
d = D 電池數量
