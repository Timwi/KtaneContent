On the Subject of NOT NOT

Never not not not not not not not fun!

This module displays a logic statement and has four clickable areas left, up, right and down.

Press any direction on the module to start the timer. Start at the position in the table labeled with the first character of the serial number. Click in the direction you want to go. For each move you make, go in that direction on the table. After ten correct answers the module will solve.

Rules

  • The Beginning: Move in the direction dictated by the module.
  • The Negation: Invert the statement when a not is referring to it and move in any direction satisfying this.
  • The Void: Don't move whenever said so.
  • The Colorist: Move towards any of the color. On the table white is a door, but not a color.
  • The Logical: Evaluate the entire statement and move to whichever satisfies.
  • The Color-Blind: The colors are displayed in the wrong color.
  • The Impossible: If a statement has no valid options, do nothing.
  • The Missing Door: If a door is missing you can't go there. These are black in the table
  • The Following: If a statement mentions 'then', do the first statement and then the second. The module will be blank on the second or it will display another 'then' statement to append.
  • The Liar: The entire statement should evaluate to false for a door to be valid if the floor has an Ø-like sign on it. This is not added to the module, but just here for completeness.
