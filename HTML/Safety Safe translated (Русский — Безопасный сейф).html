Безопасный сейф

Либо в этом сейфе несметные богатства... либо он пустой.

  • Для обезвреживания модуля все 6 ручек должны быть правильно повёрнуты.
  • У каждой ручки есть позиция, где она щёлкает громче. Это - начальная точка для каждой ручки.
  • Следуя правилам ниже, определите, на сколько позиций от начальной точки нужно повернуть каждую ручку.
  • Дёрните рычаг, чтобы проверить решение. Все правильно повёрнутые ручки будут подсвечены зелёным, в все неправильно повёрнутые - красным.
  • Начиная с 0, прибавьте количество видов портов на бомбе, умноженное на 7.
  • Прибавьте число горящих индикаторов, имеющих общую букву с серийным номером, умноженное на 5.
  • Прибавьте число не горящих индикаторов, имеющих общую букву с серийным номером.
  • Для первых пяти ручек: прибавьте число, полученное из таблицы на следующей странице, соответствующее нужному символу серийного номера.
  • Для последней ручки: прибавьте сумму чисел из последней колонки таблицы, соответствующих всем символам серийного номера.
  • Важно: Для полного оборота ручки требуется 12 поворотов.
