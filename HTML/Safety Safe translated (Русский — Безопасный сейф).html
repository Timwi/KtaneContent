О Безопасном сейфе

Либо в этом сейфе несметные сокровища... либо он пуст.

  • Для обезвреживания модуля все 6 замков должны быть верно выставлены.
  • У замков есть позиция, где они щёлкают громче. Это — начальная позиция для каждого замка.
  • Следуя правилам ниже, определите, на сколько поворотов от начальной позиции нужно выставить каждую замок.
  • Дёрните за рычаг для проверки решения. Все правильно повёрнутые замки будут подсвечены зелёным цветом, а все неправильно повёрнутые — красным.
  • Начиная с 0, прибавьте количество видов портов на бомбе, умноженное на 7.
  • Прибавьте число горящих индикаторов, имеющих общую букву с серийным номером, умноженное на 5.
  • Прибавьте число негорящих индикаторов, имеющих общую букву с серийным номером.
  • Для первых пяти замков: прибавьте число, полученное из таблицы на следующей странице, соответствующее нужному символу серийного номера.
  • Для последнего замка: прибавьте сумму чисел из последней колонки таблицы, соответствующих всем символам серийного номера.
  • Важно: Для полного оборота замка требуется 12 поворотов.
