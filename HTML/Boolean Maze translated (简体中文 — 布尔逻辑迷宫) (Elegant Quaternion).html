Keep Talking and Nobody Explodes Mod 布尔逻辑迷宫

关于布尔逻辑迷宫模块

被或非之怒惩罚

模块上含有写有U, L, R, D的四个键, 一个STUCK?键, 一个 RESET!键和一个显示从0到3的随机数的显示屏。

成功秘诀

  • 从起点走到终点来解除模块.
    • 起点：序列号的第3位作行，第4位作列。
    • 终点：序列号的第5位作行，第6位作列。
  • 把字母转化为数字(A = 1, B = 2, ...)并模除以10.
  • 起点和终点记为（行，列）的格式。迷宫的左上角是（0，0）。
  • 把显示屏上的数字转化为二进制。
  • 用写有U,L,R,D的键来向上，左，右，下移动。
  • 把转化为二进制的显示屏上的数字拆分为两位数字。如果它们作目标方格的逻辑门的输入会返回1，此移动可以执行。（见附录A）
  • 如果目标方格的逻辑门会返回0，模块会记录失误，且你不会被移动。
  • 离开迷宫的边界会记录失误，且你不会被移动。
  • 如果没有可执行的移动，STUCK?键可以用来获取另一个数字，不过如果有可执行的移动，模块会记录失误，你也会回到起点。
  • 如果你迷路了，RESET!键可以让你回到起点。
  • 提示：如果终点是在NOR或AND的方格上，它会移动，使其不在NOR或AND的方格上。移动的方向取决于显示屏上的第一个数字。(0 = 上, 1 = 右, 2 = 下, 3 = 左)。如果终点达到迷宫的边界，它会绕到迷宫的对面。
