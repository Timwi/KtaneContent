On the Subject of Flag Identification

Only flag nerds and vexillologists will reach level Guinea-Bissau.

This needy module displays a flag on a flagpole and four buttons beneath it.

When the needy module activates a random flag will be displayed. Each button will display the name of a country. To disarm the needy press the button that displays the name of the country that corresponds to the shown flag. You have 45 seconds to disarm the module.

A strike will incure when a wrong button is pressed. The module will deactivate after a wrong button press.
