Definição do módulo Obstrutor

Este módulo precisa dum microfone. O módulo irá procurar pelo dispositivo de som padrão e escuta-lo para determinar o volume.

Existe um ficheiro "modSettings.json" na pasta de mods (SteamLibrary\steamapps\workshop\content\341800\739663396) para configurar os parâmetros de variação. Podes abrir este ficheiro num editor de texto para alterar os parâmetros. Reinicia o jogo se modificares algum deles.

Esta é a configuração padrão do ficheiro:
{"MicThreshold": 25.0, "FailureThreshold": 3}

MicThreshold  [PT= Variação de Som]

Um valor de 0.0 a 100.0 para ajustar o a recepção de som do microfone.

FailureThreshold  [PT= Limite de Falha]

Quantidade de segundos válidos por insuficiência de som no microfone até causar um erro. É recomendado que este valor fique compreendido entre 1 e 10.
