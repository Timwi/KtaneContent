On the Subject of Shiritori

おしりをとり？草

Beneath the shroud of a eclipsed interface, luminous Japanese glyphs emerge, radiating an enigmatic allure. For the unversed, their combined aura pierces the depths of obscurity.

About This Module

This module draws inspiration from the classic Japanese game, “Shiritori”.
It has access to around 95,000 words spanning a vast scale of categories while the player has access to over 140,000 words that this module will allow during submissions.
The words documented in this manual serve as a baseline or guide to solving this module successfully without prior Japanese vocabulary knowledge.
It can be played like a normal Shiritori game with or without the manual.

How to Play

Simply input an acceptable word via the transcription mapping table using the last hiragana character of the display.
Ensure the Unacceptable Entries table is strictly followed.
When inputting characters, a green light will appear in the top-left corner of the module. This light indicates the transition into transcription mode. You must hit enter to exit transcription mode, then enter again to submit an answer.
You must input 10 acceptable answers to solve the module. Upon a strike, you remain on your current stage unless you lose the game.
If you lose the game, the module will reset back to stage 1 and clear all previously used answers for both parties.

Technical Note

Due to the scale of the module, generating the word list can take up to 30 seconds. Shiritori offers a caching system that drastically reduces loading times by loading a pre-generated word list. This feature is disabled by default and can be enabled in your mod settings.

Module Usage Guide and Examples

Transcription Mode Examples

To type kana in transcription mode, simply follow the mapping table on the next page.

Example Effect
Typing ka Following the kana mapping, during transcription mode, this will transcribe into: か
Typing jya or ja Following the kana mapping, during transcription mode, this will transcribe into: じゃ

Double Consonant / Small つ / 小さいつ Examples

To use the double consonants character, follow the guide below:

Example Effect
Typing kka Following the kana mapping, during transcription mode, this will transcribe into: っか
Typing gakkou Following the kana mapping, during transcription mode, this will transcribe into: がっこう

Special Keywords

Shiritori ignores submissions beginning with “!” (Exclamation mark), unless using special keywords. Usage of “!” disables kana transcription mode.

Keyword Description
!kana Displays the kana version of the currently displayed kanji.
!kanji Displays the kanji version of the currently displayed kana.

Kana Multi Selection

Some kana are written the same as one another in romaji. Should you need to use another version of a kana, the up arrow on your keyboard provides that functionality.
