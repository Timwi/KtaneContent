À propos du Générateur de Nombres Aléatoires

Nous sommes maintenant à la merci de RNGésus.

  • Quand ce module s'activera, deux tildes (~) s'afficheront sur l'écran.
  • Presser « Generate » pour générer des nombres aléatoires jusqu'à en générer un qui respecte les règles suivantes qui s'appliquent.
  • Une fois ce nombre trouvé, presser « Accept ».

Si le dernier chiffre du numéro de série est...

Impair : Le nombre doit être impair.

Pair : Le nombre doit être pair.

Si le numéro de série...

Contient une voyelle : Le nombre doit être compris entre 0 et 49.

Ne contient pas de voyelle : Le nombre doit être compris entre 50 et 99.
