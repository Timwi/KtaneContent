Dans ce module, une désignation sera indiquée par l’un des quatre boutons et l’un des personnages du livre de Mario qui montre l’image d’un personnage sur chacun d’eux. Sélectionner les bonnes lettres s’il vous plaît.

Ensuite, le nom du badge est indiqué sur le bouton. L’image du justificatif d’identité est indiquée sur l’écran où l’estimation a été affichée. Sélectionner le bon nom sur le badge s’il vous plaît. Enfin, quatre éléments d’icônes sont indiqués sur l’écran. Les quatre choix seront le nom de l’objet dans le jeu. Trois d’entre eux sont indiqués sur l’écran. Choisir un nombre impair pour désarmer ce module.
