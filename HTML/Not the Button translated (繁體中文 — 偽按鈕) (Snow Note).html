偽連打次數

開始連打後，左方指示燈會顯示按下按鈕的次數，根據按鈕的顏色決定需要計算的算式。如果計算結果在 10~99 範圍以外，重複加上/減去 7 直到結果在範圍內為止。
按鈕被按下必要的次數後，等待 3 秒鐘以提交答案。

按鈕的顏色式
紅色 (a + 2b) - d
橙色 (2b + 1) - g
黃色 (2a + d) - c
綠色 d + (2f - b)
青色 (e + f + g) - b
藍色 2c + (d - 1)
紫色 2(f - a) + d
粉紅色 3g - (a + 3)
白色 (f + ac) * (e + d)
黑色 (ab + cd) - g(e - f)

代入:

  • a = 電池數目
  • b = 埠口數目
  • c = 炸彈上的可拆除的模組數目
  • d = 指示燈的數目
  • e = 序號末位數字*
  • f = 序號第二位英文字的英文字母順序中的位置(A=1,B=2…)*
  • g = 按鈕的單詞的字數

*如果序號沒有數字，或英文字不足 2 個，有關的數值使用【0】。
