O "Забудемся Позже"

Не суди книгу по обложке.
Суди книгу по рекламе.

Изначально, модуль пройдется по последовательности цифр показанных на нижнем дисплее.
Эта последовательность имеет столько же шагов, сколько и решаемых модулей.

После каждого обезвреживания модуля, требуется произвести ввод. Ввод зависит от последней цифры абсолютного значения числа полученного используя правило, соответствующее числу показанному на маленьком дисплее.
Два последних ввода показаны на нижнем дисплее после каждого обезвреживания модуля.

Для применения существует шестьдесят различных функций.

Если сделан неправильный ввод, правильный ввод будет показан на нижнем экране и первоначальная последовательность цифр будет показана снова, перед тем как можно будет взаимодействовать с модулем опять.

Каждый неверный ввод или пропущенный этап спровоцирует ошибку.
