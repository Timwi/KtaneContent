À propos des Modules d’Erreur

Au fond de moi, je sais que c’est peut-être mal, mais je suis prêt à commettre l’Erreur no 2.

Utiliser la page 2 et la page 3 ou l’annexe VIDE dans Impasse pour l’identification des modules vides.

Nos plus sincères excuses, mais la bombe qui vous a été envoyée contient un module inachevé dont les composants n’ont pas été ajoutés dans les temps. Pour cette raison, nous vous demandons de la détruire. Cependant, même s’il n’y a pas de composants, le circuit interne du module est toujours présent.

Comme il n’y a pas de composants, le plastique du module est très fragile, et peut être détruit simplement en le touchant. Cependant, le circuit est toujours actif, et ne permettra pas de toucher le module n’importe quand. Lire ci-dessous ce que nous suggérons de faire pour ne pas recevoir d’erreurs sur votre bombe.

Nous comprenons également qu’il existe d’autres modules fonctionnels qui semblent identiques à celui qui vous a été envoyé par erreur. L’annexe NON contient des informations sur la manière de différencier ce module des autres.

Désarmer ce module :

Vous devriez pouvoir toucher le module une fois sans problème. Si bien fait, vous devriez entendre le plastique se briser et le module devrait se fissurer.

Faites attention à partir de maintenant, nous ne pensons pas que toucher le module à tout moment soit sans danger. Nous vous suggérons de le toucher une seconde fois lorsque le dernier chiffre de la minuterie de la bombe est égal au dernier chiffre du numéro de série de la bombe.

Ne vous inquiétez pas, c’est la dernière instruction que nous vous donnons. Si nos hypothèses sont correctes, vous pourrez toucher à nouveau le module lorsque la section des secondes du minuteur de la bombe sera égale à la somme des chiffres du numéro de série de la bombe. Cela devrait résoudre le module après quelques secondes. Si ce n’est pas le cas, veuillez nous contacter.

Si le module donne une erreur, le module se brisera quand même, et vous devrez passer à la directive suivante ci-dessus. Le module peut également se résoudre même dans le cas où il vous a donné une erreur peu de temps auparavant. Cependant, nous ne vous conseillons pas de continuer à le toucher.

Nous vous remercions de votre coopération.   - Les ouvriers de l’usine de bombes

On ne nous accorde pas assez de crédit.

Différenciation de ce module par rapport aux autres :

Les rapports des désamorceurs qui ont rencontré ce module nous ont amenés à penser qu’il n’apparaît que sur le côté de la bombe qui contient la minuterie.

Charnières :

Sur un module Charnières, le voyant d’état n’est pas visible au départ et certaines des charnières du plastique sont manquantes. Le module qui vous a été envoyé a toutes les charnières intactes et le voyant d’état devrait toujours être visible.

9-cube :

En survolant un module 9-cube, la bombe se met à trembler et le module fait un bruit fort, change de couleur et émet de minuscules particules*. Le module qui vous a été envoyé ne fera rien tant qu’il n’aura pas été touché.

Braille :

En survolant un module Braille, vous pouvez sentir des points en braille. Il n’y a pas de points sur le module qui vous a été envoyé.

Polygones :

En survolant un module Polygones, vous pouvez voir les contours des formes cachées. Il n’y a pas de formes cachées sur le module qui vous a été envoyé.

...? (Signes de Ponctuation) :

En survolant un module Signes de Ponctuation, le contour du module devient rouge-orange au lieu de jaune. De plus, toucher une fois un module Signes de Ponctuation est totalement sûr, car cela ne déclenche aucun mécanisme logique dans le module, à part un son différent de celui d’un éclat. Le module qu’on vous a envoyé se brisera au premier contact.

Impasse :

En survolant un module Impasse, vous pouvez entendre les huit sections différentes du module. Le module qui vous a été envoyé ne comporte qu’une seule section. Si vous n’êtes pas en mesure d’entendre les différentes sections, toucher le voyant d’état d’un module Impasse ne fait rien. Sur le module qui vous a été envoyé, le fait d’appuyer sur le voyant fait éclater le module.

*Nous n’avons aucune preuve officielle de cela, c’est simplement ce que nous croyons. Aucun rapport connu de désamorceurs n’a confirmé ce qui se passe avec un module 9-cube. Nous craignons le pire.

Allée des Sourds :

En survolant un module Allée des Sourds, vous pouvez entendre les quatre sections différentes du module ainsi qu’un son de basse dans certaines parties du module. Le module qui vous a été envoyé ne comporte qu’une seule section et ne peut pas produire de sons de basse. Si vous n’êtes pas en mesure d’entendre les différentes sections ou la tonalité, le fait de toucher une charnière sur le plastique d’un module Allée des Sourds ne fait rien. Sur le module qui vous a été envoyé, appuyer sur celles-ci fera éclater le module.

Code de Tape :

Malheureusement, le module qui vous a été envoyé ne peut pas être distingué d’un module Tap Code en passant la souris sur certaines parties du module. Cependant, en touchant un module Code de Tape n’importe où, il jouera une série de tapes ou enregistrera les tapes que vous avez entrées. Dans tous les cas, le module n’aura pas l’air de se briser, comme le fait le module qui vous a été envoyé. C’est également totalement sûr car l’entrée d’un module Tap Code peut être réinitialisée à tout moment en maintenant le module.

42 et 501 :

De la même manière que pour le Code de Tape, le module qui vous a été envoyé ne peut pas être distingué d’un module 42 ou 501 en passant la souris sur certaines parties du module. Cependant, le fait de toucher le voyant d’état d’un module 42 ou 501 n’active pas son mécanisme d’affichage des chiffres. De plus, les modules 42 et 501 produisent régulièrement des clips vocaux autotunés. Le module qui vous a été envoyé se brise lorsqu’on appuie sur le voyant d’état, et il ne peut pas produire de clips vocaux.
