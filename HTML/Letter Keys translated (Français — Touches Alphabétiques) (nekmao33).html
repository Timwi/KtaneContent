Suivre ces règles dans l’ordre dans lequel elles sont listées. Effectuer la première action qui s’applique :
  1. Si le nombre indiqué est égal à soixante-neuf, appuyer sur la touche avec l’étiquette « D ».
  2. Si le nombre indiqué est divisible par six, appuyer sur le bouton avec l’étiquette « A ».
  3. S’il y a deux piles ou plus sur la bombe et que le nombre est divisible par trois, appuyer sur le bouton portant l’étiquette « B ».
  4. Si le numéro de série contient un « C », « E » ou « 3 » et que le nombre est supérieur ou égal à vingt-deux, et inférieur ou égal à soixante-dix-neuf, alors appuyer sur le bouton marqué « B ».
  5. Si le numéro de série contient un « C », « E » ou « 3 », appuyer sur le bouton « C ».
  6. Si le nombre indiqué est inférieur à quarante-six, appuyer sur le bouton « D ».
  7. Sinon, appuyer sur le bouton marqué « A ».
