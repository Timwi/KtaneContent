Keep Talking and Nobody Explodes Mod Color Sorting

On the Subject of Color Sorting

Love sorting M&M’s by color? This module is for you!

The module contains 54 buttons (in default settings).

Divide the shape on the module to 6 equal diamonds. Every diamond must contain only one color.

Pressing two buttons will swap their positions.

To change the size in a mission, simply put [Color Sorting] # anywhere inside of the mission description, replacing # with the positive number.
