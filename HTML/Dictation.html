Keep Talking and Nobody Explodes Mod Dictation

On the Subject of Dictation

Sphinx of black quartz, judge my vow.

  • Press the button labelled "begin" to start cycling through a phrase on the main screen which could contain up to ninety-nine words.
  • When the phrase is done cycling, a number will appear on the smaller screen to the right.
  • Input the word whose position in the phrase matches the given number. Use the < key to delete the previous input and the return key to submit.
