Keep Talking and Nobody Explodes Mod 中国の数え方

モジュール詳細：中国の数え方

まあ、日本の数え方でもあるんですけど。

このモジュールには、漢数字が書かれた４つのボタンと２つのLEDがある。

左のLEDを段、右のLEDを列として使用し、ボタンを押す順序を決定する。

白 赤 緑 オレンジ
白 A C H D
赤 H D A C
緑 C H D A
オレンジ H A C D

A = 値の昇順
D = 値の降順
C = 文字数の昇順(同数は値の降順)
H = 文字数の降順(同数は値の昇順)

一1六6
二2七7
三3八8
四4九9
五5十10
