À propos de Challenge & Contact

Pourquoi continuer à créer des modules basés sur des jeux ? Le désamorçage de bombes est censé être stressant, pas amusant...

Sur le module se trouve un « indice » pour un mot, avec un écran de lettres et un bouton intitulé « Contact ». En appuyant sur ce bouton, le module se retournera, révélant un clavier et un chronomètre, où il faudra taper le mot correspondant à l'indice. Les indices peuvent être trouvés dans le tableau ci-dessous. Une fois le module retourné, soumettre le mot lorsque (ou avant) le module indique « GO! ». Le bon indice est celui qui commence par les lettres affichées à l'écran. Taper trois mots d'affilée pour résoudre le module. (Après le premier mot, le clavier se casse parfois et les lettres tournent).

L'écran des lettres en bas est malheureusement très défectueux et affiche les lettres cryptées dans un code. Si le nombre de modules désamorcés pair lorsque l'indice est généré, la lettre est cryptée en rot13. Prendre la lettre et parcourir les 13 lettres de l'alphabet, en revenant à A lorsque Z est atteint. Sinon, la lettre est cryptée dans le code Atbash. Dans le chiffrement Atbash, chaque lettre est son opposé dans l'alphabet (A = Z, B = Y, C = X... Z = A). Pour la première lettre, utiliser à la place le nombre total de modules (including needies).

(Cependant, s'il y a un indicateur BOB allumé et qu'il aucune pile sur la bombe, les lettres sont tout à fait normales. Ne pas s'inquiéter.)

Indices/Réponses

Le clavier ne comportant que des lettres, seuls les modules d'un seul mot, sans chiffres ni symboles, peuvent être soumis. S'il y a plusieurs réponses correctes, la première dans l'ordre alphabétique est la bonne.
