On the Subject of The Multiverse Hotline

  1. Make a call*. Get Origin-1 and the 3 digit number.
  2. Reorder the digits from Origin-1's order to 123 order. Example: 493, order 312: 493 -> 934.
  3. Convert the number from Origin-1's base to base 10** to get N.
  4. Use digits of N to find Origin-2 on the second page.
  5. Input the ID of Origin-2 and pick up the phone.
  6. Convert N from base 10 to Origin-2's base**.
  7. Rearrange the digits of new N from 123 order to Origin-2's order. Example: 612, order 231: 612 -> 126.
  8. Input the resulting number. Listen to the message* and input said number.
  9. If at any point a mistake is made, the module strikes and completely resets.

*At this point you can pick up the phone again to listen to the message again.

**After converting take the 3 least significant digits or prepend 0s if the number is shorter than 3 digits.

Finding Origin-2

Translate digits of N into 3 consecutive directions:

0UP1RIGHT
2DOWN3NONE
4LEFT5UP
6RIGHT7LEFT
8NONE9DOWN

Starting from Origin-1 repeatedly go in these 3 directions (skipping over already-visited cells and wrapping around if needed) until you end up in a loop. The last cell visited before ending up in the loop is Origin-2.
