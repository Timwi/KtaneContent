PIPress if there are more lit than unlit indicators
!PP%2Press if there are an odd number of port plates
!VIPress if all lit indicators don’t contain any vowels
2C!DPress if there is at least 2 of the current button colour and no other duplicate button colours
P>SPress if there are more parallel ports than serial port
WMPress if there is a button with a warm colour
