À propos du Test de Stroop

Y a-t-il un peu de place pour un onzième ?

Attention : Si l'écran n'affiche pas continuellement de nouveaux mots, il s'agit d'un module différent.

Un module Test de Stroop affiche continuellement de nouveaux mots de couleurs différentes. Ces mots représentent le nom de 6 couleurs distinctes. La couleur des mots ne correspond pas forcément au mot lui-même. Ce module consiste également en un bouton « YES » et en un bouton « NO ».

Répondre correctement à trois questions selon les informations données par l'écran pour désamorcer le module. Noter le premier mot coloré qui a été correctement validé.

Les questions se trouvent dans le tableau ci-dessous. Le mot actuellement affiché et les caractères du numéro de série déterminent la question à laquelle répondre.

La colonne la plus à gauche détermine la couleur associée à chaque question. Pour déterminer quelle question à laquelle répondre pour le mot affiché, prendre le dernier chiffre du numéro de série. S'il est pair, utiliser la couleur du mot affiché. Sinon, utiliser le mot lui-même.

Regarder la couleur utilisée dans le tableau ci-dessous. Chaque couleur est associée à deux questions. Pour déterminer quelle question à laquelle répondre, prendre le premier chiffre du numéro de série. S'il est pair, prendre la question du haut. Sinon, prendre la question du bas.

Si la validité de la question est vraie, presser « YES ». Si elle est fausse, presser « NO ».

Si la réponse donnée est incorrecte, le module enregistrera une erreur. Cependant, peu importe si la réponse donnée est correcte ou non, le module enregistrera une erreur si :

  • La même question a déjà été répondue auparavant.
  • La réponse donnée pour la troisième question est la même que pour la deuxième.

Soumettre une réponse correcte à trois questions différentes. Chaque fois qu'une réponse correcte est donnée, le module effectuera une pause de 2 secondes. Cette pause permet de noter le mot et la couleur associés à la bonne réponse donnée.

Une fois les trois réponses correctes données, le module se désamorcera.
