Keep Talking and Nobody Explodes Mod zzz

On the Subject of The zzz

Someone fell asleep when making this module...

This needy module has 2 displays and 3 buttons.

To deactivate this needy module, you must use the left display and count the buttons from top to bottom to find the corresponding one. The top button is number 1. You then must press that button the same amount of times as the right display, as if you are sleeping on that button. Then wait for the timer to run out, and if the needy module does not strike, then the answer was correct.

If the wrong button is pressed OR if the timer runs out and the answer wasn’t correct, the needy module will strike and deactivate.
