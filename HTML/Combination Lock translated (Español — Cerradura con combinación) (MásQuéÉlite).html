Mod de Keep Talking and Nobody Explodes Cerradura con combinación

En caso de la cerradura con combinación

Esto tiene pinta de una cerradura con combinación. Pensaba que estaba desactivando esta bomba, no abriéndola.


Véase el Apéndice B para la identificación de la pila.
Véase el Apéndice Two Factor para la identificación de dos factores.

Como una típica cerradura con combinación, esta requiere de 3 números para desbloquearla(¿?). Gira el dial a la derecha para ponerlo al primer número. Luego a la izquierda para el tercero. Y finalmente a la derecha para el tercero. ¡Eso lo desbloqueará!

Si dos números en la secuencia son los mismos, gira completamente el dial al mismo número.

Cómo decodificar la combinación:

Cada número está entre el 0 y el 19. Vete a la tabla para determinar cada número. ¡Ten cuidado si hay códigos de doble factor en la bomba porque cambian periódicamente!

Primer número:
  1. Suma el último dígito de cada doble factor. Si no hay ninguno, usa el último dígito del número de serie más el número de módulos resueltos.
  2. Añádele el número de pilas.
  3. Si el resultado pasa de 19, réstale 20 hasta que esté entre 0 y 19.
Primer número:
  1. Suma el primer dígito de cada doble factor. Si no hay ninguno, usa el número de módulos en la bomba (incluyendo needys).
  2. Añádele el número de módulos resueltos.
  3. Si el resultado pasa de 19, réstale 20 hasta que esté entre 0 y 19.
Tercer número:
  1. Suma los dos números.
  2. Si el resultado pasa de 19, réstale 20 hasta que esté entre 0 y 19.
