À propos du Mastermind Cruel

Le jeu de société de votre enfance. Plus poussé. Et plus difficile à mémoriser. Et les indices ne sont plus pareils...

  • Pour désamorcer ce module, trouver le bon code à 5 couleurs.
  • Les couleurs utilisées sont : Blanc, Magenta, Jaune, Vert, Rouge et Bleu.
  • À noter que chaque couleur peut être utilisée plusieurs fois.
  • Les couleurs des DELs peuvent être modifiées en appuyant sur les DELs.
  • Presser le bouton « Query » donnera des informations sur le code saisi.
  • Quand le bon code est saisi, le module sera désamorcé une fois le bouton « Submit » pressé. Valider un mauvais code enregistrera une erreur.

Chaque demande d’informations affichera deux nombres sur l’écran. Ces nombres représentent chacun la somme totale de deux valeurs listées ci-dessous.

A = Nombre de couleurs correctes dans les bonnes positions.
B = Nombre de couleurs correctes dans les mauvaises solutions.
C = Nombre de couleurs pas dans le code tout court.

Couleur sur l’écran : Position de la couleur Nombre de gauche Nombre de droite

Droite : # de piles # de modules résolus
