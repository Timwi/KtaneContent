À propos de la Réponse HTTP

Quand ta bombe se comporte comme un serveur web...

  • Le module affiche une chaîne de statut distant ou local. Y répondre par un code de statut à 3 chiffres.
  • Les statuts distants peuvent être trouvés dans le tableau 1 et les statuts locaux dans le tableau 2.
  • Une source confidentielle a rapporté que dû à des restrictions d’affichage, des chaînes peuvent être raccourcies.
  • Utiliser le clavier pour entrer le code de réponse correct pour désamorcer le module demandant.
  • Attention : Il n’y a pas de bouton « Effacer », il faut donc être prudent !
