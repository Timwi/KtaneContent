À propos du Code Frappé

Ce n’est pas du morse ?

Voir l’annexe VIDE dans Impasse pour l’identification des modules vides.

  • Ce module peut détecter quand il est tapé. Même sur le voyant d’état !
  • Pour le résoudre, taper, sur le module, un mot de 5 lettres en code frappé.
  • Pour trouver le mot, appuyer et maintenir le module et il tapera un mot en code frappé, puis trouver le mot dans le tableau.
  • Maintenir le module réinitialise également toute entrée.
  • Se déplacer de la valeur du dernier chiffre du numéro de série dans la direction correspondant aux conditions ci-dessous :
    12345
    1ABCDE
    2FGHIJ
    3LMNOP
    4QRSTU
    5VWXYZ
    • S’il y a un zéro dans le numéro de série, se déplacer du montant de la somme des chiffres du numéro de série à la place.
    • Si les deux premiers caractères du numéro de série sont... XX : Haut, ## : Bas, X# : Gauche, #X : Droite. (# = Numéro, X = Lettre)
  • Si le bout du tableau est atteint, retourner de l’autre côté du tableau.
  • Chaque lettre est représentée par son numéro de ligne (nombre de tapes), une pause, puis son numéro de colonne (nombre de tapes).
  • La lettre « K » est remplacée par un « C » en code frappé.
  • Lors de la saisie de la réponse, un clic est émis après chaque pause, signalant qu’il faut poursuivre la saisie de la réponse. À la fin, il y aura un autre bruit de clic. Taper à nouveau pour soumettre votre solution.
  • La saisie d’un mot incorrect entraîne une erreur et la saisie est effacée.
