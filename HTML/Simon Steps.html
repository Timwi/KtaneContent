On the Subject of Simon Steps

I’m stepping away from this. Too many Simons on this trail.

This module uses four rule seeds. To visit the necessary pages for this module, copy https://ktane.timwi.de/HTML/Simon%20Steps.html# into your search bar and append any necessary rule seed.

This module features a diamond with 4 colored sections that can flash. Each section that flashes displays a digit. During the first stage, the sections that flashes will be the order in which to create the initial rule seed. Remove any leading zeros and visit the rule-seeded page using this first rule seed.

The four sections on the module will be one of eight colors. A color may appear more than once.

When ready to submit stage(s), press one of the four sections. All four sections will lose their color after the sequence is done flashing.

When submitting, a number will appear in the center of the diamond. This number represents the stage of the module to submit.

Each color in all four tables will have their own relative direction. Find the color(s) in the nth stage’s table and note its direction in sequential order, first flash with first direction, then second flash with second direction, et cetera.

The directions are related to the sections, “left” is the left button, “right” is the right button, et cetera. Press the buttons in order to advance to the next stage.

For each digit that flashes, append these digits to the previous obtained digits in order of the flashes. Modulo this number by 2147483647. Visit the rule-seeded page using this number. After using the modulo operation, save the dividend (number to the left of modulo operation) for future stages.

Note the colors of each stage. The answer for the first stage in the first stage may not be the same answer for the first stage in future stages.

To solve the module, complete the four stages of the module. Pressing an incorrect section during a stages submission will result in a strike. Once the module strikes, all stages from beginning to current will flash again. Once finished, the module will allow submissions.
