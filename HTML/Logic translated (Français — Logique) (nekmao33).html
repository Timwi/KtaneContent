À propos de la Logique

La logique est facile, mais logique ET désamorçage pourraient ne pas l’être.

  • Chaque ligne affiche 3 lettres. Chaque lettre représente un énoncé qui se trouve dans le tableau 1.
  • Sur chaque rangée, résoudre d’abord les énoncés entre parenthèses.
  • Les énoncés sont reliés par des symboles de connexion logiques. Retrouver le fonctionnement de chaque symbole dans le tableau 2.
  • Appliquer la négation (porte NOT : faux devient vrai et vrai devient faux) à chaque énoncé d’abord si la DEL rouge au-dessus de cet énoncé est allumée.
  • Trouver le résultat de chaque rangée, puis utiliser le bouton "T/F" à droite pour sélectionner Vrai/Faux. Appuyer sur « Submit » une fois terminé.

Tableau 2 : Liste des symboles de connexion logique

Connectivité logique Symbole Porte logique Signification
Conjonction ∧ AND Renvoie vrai si toutes les entrées sont vraies. Sinon renvoie faux.
Disjonction ∨ OR Renvoie vrai si une entrée est vraie. Sinon renvoie faux.
Disjonction exclusive ⊻ XOR Renvoie vrai si une seule entrée est vraie. Sinon renvoie faux.
Refus alternatif | NAND Renvoie faux si toutes les entrées sont vraies. Sinon renvoie vrai.
Refus conjoint ↓ NOR Renvoie faux si une entrée est vraie. Sinon renvoie vrai.
Biconditionnel ↔ XNOR Renvoie faux si une seule entrée est vraie. Sinon renvoie vrai.
Implication (Gauche) → - Renvoie faux si l’entrée gauche est vraie et que l’entrée droite est fausse. Sinon renvoie vrai.
Implication (Droite) ← - Renvoie faux si l’entrée gauche est fausse et que l’entrée droite est vraie. Sinon renvoie vrai.
