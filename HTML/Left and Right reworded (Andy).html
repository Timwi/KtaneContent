On the Subject of Left and Right

Three rights make one left.

To solve the module, first find a decimal number using the table below, then modify the sequence using the second table.

First digit(Last digit of serial)^3 % 8
Second digitSame number of ports and platesNumber of D batteries followed by the number of letters in the serial number.*
Else(Ports + port plates) % 2.
Third digitBatteries = holders(lit indicators - unlit indicators)^2 % 6.
ElseSum of batteries and holders.*
*These can be multiple digits, which messes up the digit counter.

Convert this decimal number to binary. This will be the unaltered sequence.

To alter the sequence, find the first two rows where the conditions apply. The associated numbers refer to green and blue respectively. When a colored button has been pressed this many times, invert the remaining sequence.

If, however, you specifically have all of: lit FRK, unlit NSA, PS/2, Parallel, Serial, RJ and DVI ports and no RCA port - no switches occur.

To submit the sequence, use 0 and 1 as the left and right buttons respectively.

If you were to strike, the input would not reset.
