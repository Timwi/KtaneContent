Keep Talking and Nobody Explodes Mod ＞И＜

О ＞И＜

Модные имена нынче в моде.

На модуле имеется три экрана и панель с цифрами от 0 до 9. На среднем экране отображаются два пятизначных двоичных числа. На верхнем левом экране отображаются символы, а на верхнем правом экране отображается номер этапа.

Данный модуль можно обезвредить только после обезвреживания всех небоссовых модулей. После обезвреживания каждого модуля* вам необходимо записать информацию с модуля и узнать ответ, который нужно будет ввести в самом конце.

Обезвреживание модуля:

Для каждого этапа возьмите два двоичных числа. Примените к каждым двум битам в соответствующей позиции логическую операцию, указанную на экране сверху слева (верхний бит считается первым, нижний - вторым). Однако, если указан символ ¬ (НЕ), тогда инвертируйте каждый бит в верхнем числе и переведите полученное число в десятичное и используйте его.

Например, если верхнее число равно 10101, а нижнее 00110, для логического оператора ИЛИ ответ будет 10111.

Далее, переведите двоичные ответы в десятичные числа. Если число этапа является чётным, ответом на данный этап является последняя цифра десятичного числа. Иначе, ответом на данный этап является цифровой корень из десятичного числа.

Если при вводе ответа вам будет зачтена ошибка, тогда модуль отобразит исходные числа и оператор для неверного этапа. Нажмите верную цифру для продолжения.

*Некоторые модули игнорируются ＞И＜.

Логические операторы, используемые в ＞И＜:

Символ и имя Значение
∧ (И) Возвращает истину если оба входа истины. Иначе возвращает ложь.
∨ (ИЛИ) Возвращает истину если хотя бы один вход истиный. Иначе возвращает ложь.
⊻ (ИСК.ИЛИ) Возвращает истину если только один вход истиный. Иначе возвращает ложь
| (НЕ И) Возвращает ложь если оба входа истины. Иначе возвращает истину.
↓ (НЕ ИЛИ) Возвращает ложь если хотя бы один вход истиный. Иначе возвращает истину.
↔ (НЕ ИСК.ИЛИ) Возвращает ложь если только один вход истиный. Иначе возвращает истину.
→ (Следствие) Возвращает ложь если первый вход истиный, а второй ложный. Иначе возвращает истину.
← (Обратное Следствие) Возвращает ложь если первый вход ложный, а второй истиный. Иначе возвращает истину.
