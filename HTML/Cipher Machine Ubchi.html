On the Subject of the Ubchi CM

A machine that knows no bounds when it comes to ciphers.

Take the keyword from screen 1 and assign each letter in alphabetical order where 1 is a letter that occurs earliest from the alphabet. If there is a tie, assign each tie the SAME number. This will be referred to as the Ubchi Key.

If the Submit Button is white, follow the instructions under Encrypt Instructions. Otherwise follow the instructions under Decrypt Instructions.

Encrypt Instructions

  • 1: Write the encrypted word underneath the Ubchi Key so that it fits in the columns of the key in reading order.
  • 2: Read each letter in ascending numerical order. In the case of duplicate numbers, read them from left to right, top to bottom for the current duplicate number.
  • 3: Append the letter displayed on screen A to the encrypted word.
  • 4: Repeat steps 1 - 2. Then remove the last letter
  • 5: The resulting letters should form a decrypted word.

Decrypt Instructions

  • 1: Append the letter shown on screen A to the encrypted word.
  • 2: Write X dashes underneath the Ubchi Key so that it fits in the columns of the key in reading order. X is equal to the length of the encrypted word.
  • 3: Place each letter of the encrypted word in ascending numerical order, replacing each dash underneath the number. In the case of duplicate numbers, place them from left to right, top to bottom for the current duplicate number, until each of the dashes underneath the duplicate numbers have been replaced.
  • 4: Read the letters in reading order.
  • 5: Remove the last letter. Then repeat steps 2 - 4.
  • 6: The resulting letters should form a decrypted word.
