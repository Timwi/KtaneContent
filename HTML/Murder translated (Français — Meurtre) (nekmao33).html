Keep Talking and Nobody Explodes Mod Meurtre

À propos du Meurtre

Ce module est alimenté par l’âme agitée d’une victime de meurtre. La seule façon de la désarmer est de résoudre l’affaire pour que la victime puisse passer paisiblement dans l’au-delà.

Voir l’annexe A pour la référence d’identification des indicateurs.
Voir l’annexe B pour la référence d’identification des piles.
Voir l’annexe C pour la référence d’identification des ports.
  • Sélectionner le meurtrier, l’arme du crime et l’emplacement sur les panneaux d’affichage et appuyer sur « ACCUSE » pour désarmer le module.
  • Le module affiche un emplacement en rouge - c’est la pièce dans laquelle le corps a été trouvé. Ce n’est pas nécessairement la pièce où le crime a été commis.
  • Le tableau ci-dessous montre la localisation des suspects et les armes potentielles au moment du meurtre. Le meurtrier devait se trouver au même endroit que l’arme du crime.
  • Certains suspects et armes potentielles ont déjà été éliminés de l’enquête - ils ne figurent pas dans le module.

Suspects :

  • S’il y a un voyant allumé avec l’étiquette TRN, utiliser la ligne 5 pour localiser les suspects.
  • Sinon, si le corps a été retrouvé dans le Dining Room, utiliser la ligne 7.
  • Sinon, si la bombe a 2 ports RCA stéréo ou plus, utiliser la ligne 8.
  • Sinon, s’il n’y a pas de piles D sur la bombe, utiliser la ligne 2.
  • Sinon, si le corps a été trouvé dans le Study, utiliser la ligne 4.
  • Sinon, s’il y a 5 piles ou plus, utiliser la ligne 9.
  • Sinon, s’il existe un indicateur éteint avec l’étiquette FRQ, utiliser la ligne 1.
  • Sinon, si le corps a été trouvé dans le Conservatory, utiliser la ligne 3.
  • Sinon, les suspects peuvent être localisés à l’aide de la ligne 6.

Armes :

  • Si le corps a été trouvé dans le Lounge, utiliser la ligne 3 pour localiser les armes.
  • Sinon, s’il y a 5 piles ou plus, utiliser la ligne 1.
  • Sinon, si la bombe a un port Série, utiliser la ligne 9.
  • Sinon, si le corps a été trouvé dans le Billard Room, utiliser la ligne 4.
  • Sinon, s’il n’y a pas de piles sur la bombe, utiliser la ligne 6.
  • Sinon, s’il n’y a pas d’indicateurs allumés sur la bombe, utiliser la ligne 5.
  • Sinon, si le corps a été trouvé dans le Hall, utiliser la ligne 7.
  • Sinon, si la bombe a 2 ports RCA stéréo ou plus, utiliser la ligne 2.
  • Sinon, les armes peuvent être localisées en utilisant la ligne 8.
