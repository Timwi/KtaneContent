Based on the symbol on the button with the flashing light, and the colour that it is flashing, determine which one of the four buttons to press using the correct chart below.
The original button will flash, followed by another. Repeat this process in order for each flash.
The sequence will lengthen by one each time you correctly enter a sequence until the module is disarmed. There are always five stages.
