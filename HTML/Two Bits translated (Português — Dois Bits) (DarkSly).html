Keep Talking and Nobody Explodes Mod Dois Bits

No tópico de: Dois Bits

Este dispositivo de pesquisa fracamente programado, é tão cruel com as suas respostas lentas como é imperdoável com as suas inserções inoportunas. É necessária paciencia.

Consulta uma série de códigos de duas letras para descobrir qual a resposta correta antes de a submeter. Esta máquina primordial de pesquisa é intolerante a inserções incompletas e excessivas, e também a inserções enquanto estiver ocupada.

Passo 1: Determinar o Código Inicial

Se o Número de Série possuir uma letra, usa o valor numérico no alfabeto da letra mais à esquerda como o teu valor inicial (Ex: A=1, B=2). Para um Número de Série sem letras, usa 0.

Adiciona o último dígito do Número de Série multiplicado pela quantidade de baterias presentes.

Se uma porta Stereo RCA for presente, dublica o valor atual.*

Este valor** será o teu código atual.

* Nota: Ignora este passo se existir também uma porta RJ45 presente.

** Nota: Usa os últimos dois dígitos se o valor for maior que 99. Adiciona um zero ao final se o valor for menor que 10.

Passo 2: Determinar o par de caracteres e executar consultas

Usando o código atual, procura na tabela o par de caracteres. Insere o par adquirido no módulo e pressiona Consultar [EN=Query] para realizar uma consulta.

Passo 3: Repete e Submete

O código de resposta da consulta executada no Passo 2 é agora o teu código atual. Executa o Passo 2 mais 2 vezes, usando sempre o novo código em cada consulta executada.

Depois de receber o código de resposta da última consulta, procura na tabela o par de caracteres correspondente, insere o par adquirido e pressiona Submeter [EN=Submit].
