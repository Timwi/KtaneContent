A Respeito do Cadeado de Combinação

Este módulo parece um cadeado de combinação. Eu achei que eu estava desarmando essa bomba, e não abrindo ela?

Veja o Apêndice B para a referência de identificação de pilhas.
Veja o Apêndice Dois Fatores para a referência de identificação de dois fatores.

Como um típico cadeado de combinações, este módulo precisa de três números para desarmar. Vire o disco para a direita até chegar no primeiro número. Então vire-o para a esquerda para chegar no segundo número. Por fim, vire-o para a direita para chegar no último número. Isso irá desarmá-lo!
Se dois números consecutivos no código forem os mesmos, gire o disco completamente até chegar no mesmo número.

Como decodificar a combinação:

Cada número varia entre 0 e 19. Verifique a tabela para determinar cada número. Tome cuidado caso existam números de dois fatores porque eles mudam periodicamente!

Primeiro número:
  1. Adicione o dígito menos significativo de todos os números de dois fatores. Se não houver números de dois fatores presentes, use o último dígito do número serial adicionado ao número de módulos já desarmados.
  2. Adicione o número de pilhas.
  3. Se o resultado for maior que 19, subtraia de 20 em 20 até o resultado estar entre 0 e 19.
Segundo número:
  1. Adicione o dígito mais significativo de todos os números de dois fatores. Se não houver números de dois fatores na bomba, use o número de módulos na bomba (inclusive módulos carentes).
  2. Adicione o número de módulos desarmados.
  3. Se o resultado for maior que 19, subtraia de 20 em 20 até o resultado estar entre 0 e 19.
Terceiro número:
  1. Adicione os dois primeiros números.
  2. Se o resultado for maior que 19, subtraia de 20 em 20 até o resultado estar entre 0 e 19.
