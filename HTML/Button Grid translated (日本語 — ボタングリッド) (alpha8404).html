Keep Talking and Nobody Explodes Mod ボタングリッド

モジュール詳細：ボタングリッド

「そんなに難しくないよね...?」

  • このモジュールは、赤・青・緑・黄の４色からなる８×５のボタンと、ステージを表すディスプレイ、及びリセットボタンで構成されている。このモジュールを解除するには、特定の順番で各ボタンを１回ずつ押す必要がある。
  • 各色は１０個ずつある。
  • 注：同じボタンを複数回押してはならない。しかし、押したボタンの見た目は変わらないので、どのボタンを押しているのか記録する必要がある。既に押したボタンを押すと即座にミスが記録され、そのステージの入力がリセットされる。
  • このモジュールはステージが１０まであり、上部には突破したステージ数が表示される(LEDの点灯が３つのとき、現在はステージ４である)。
  • 各ステージで、特定の順序で各色のボタンを１つずつ押す必要がある。
  • 間違った順序でボタンを押すと、４色を押し終わった後にミスが記録される。
  • ルールは、上から順に最初に該当するものに従う。

ただし、点灯したBOBインジケーター、DVI-Dポート、バッテリーが一つあり、及びシリアルナンバーに「C,H,R,I,S」のいずれかの英字が含まれる場合は、ステージ１で青、赤、青、黄を押すことで即座に解除することが出来る。これを押す場合は、ステージ１で行わなければならない。

ステージ１

  1. SIGインジケーターがある場合、赤、青、黄、緑の順で押す。
  2. TRNインジケーターがある場合、緑、赤、黄、青の順で押す。
  3. FRQインジケーターがある場合、赤、黄、青、緑の順で押す。
  4. それ以外の場合、緑、黄、青、赤の順で押す。

ステージ２

  1. 少なくとも２０個のモジュールがある場合、青、赤、黄、緑の順で押す。
  2. 少なくとも１５個のモジュールがある場合、青、緑、黄、赤の順で押す。
  3. 少なくとも１０個のモジュールがある場合、緑、青、赤、黄の順で押す。
  4. それ以外の場合、緑、赤、青、黄の順で押す。

ステージ３

  1. シリアルナンバーに「B,T,N,G,W,D」のいずれかの英字がある場合、黄、青、赤、緑の順で押す。
  2. シリアルナンバーに「J,V,P,K,Z,F」のいずれかの英字がある場合、黄、緑、青、赤の順で押す。
  3. シリアルナンバーに母音が含まれている場合、青、緑、赤、黄の順で押す。
  4. それ以外の場合、青、赤、緑、黄の順で押す。

ステージ４

  1. バッテリーが６つ以上ある場合、赤、黄、緑、青の順で押す。
  2. バッテリーが４つ以上ある場合、赤、緑、青、黄の順で押す。
  3. バッテリーとバッテリーホルダーの数が同じ場合、黄、赤、緑、青の順に押す。
  4. それ以外の場合、黄、赤、青、緑の順で押す。

ステージ６

  1. パラレルポートがある場合、赤、青、緑、黄の順で押す。
  2. PS/2ポートがある場合、青、黄、赤、緑の順で押す。
  3. ポートがない場合、緑、黄、赤、青の順で押す。
  4. それ以外の場合、黄、青、緑、赤の順で押す。

ステージ６

  1. ４色全てが含まれた列がある場合、緑、黄、赤、青の順で押す。
  2. ４色のいずれかがない段がある場合、青、赤、緑、黄の順で押す。
  3. ４隅の色がすべて同じ場合、黄、緑、青、赤の順で押す。
  4. それ以外の場合、赤、青、黄、緑の順で押す。

ステージ７

  1. １段目に最もある色が青の場合、青、緑、黄、赤の順で押す。
  2. １段目に最もある色が緑の場合、緑、赤、青、黄の順で押す。
  3. １段目に最もある色が赤の場合、赤、黄、緑、青の順で押す。
  4. それ以外の場合、黄、青、赤、緑の順で押す。

ステージ８

  1. １列目に赤が２つある場合、赤、緑、青、黄の順で押す。
  2. １列目に黄が２つある場合、黄、赤、緑、青の順で押す、
  3. １列目に緑が２つある場合、緑、青、黄、赤の順で押す。
  4. それ以外の場合、青、黄、赤、緑の順で押す。

ステージ９

  1. 読み順の最後の色が青の場合、赤、緑、黄、青の順で押す。
  2. 読み順の最後の色が赤の場合、黄、青、緑、赤の順で押す。
  3. 読み順の最後の色が黄の場合、緑、青、赤、黄の順で押す。
  4. それ以外の場合、青、赤、黄、緑の順で押す。

ステージ１０

  • 残りのボタンを任意の順で押す。
