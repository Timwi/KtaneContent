Keep Talking and Nobody Explodes Mod 迷路信号

モジュール詳細：迷路信号

この電報が意思を持つことは「アメイジング」じゃないか！？発信しているものはナンセンスだ！

  • 点滅するライトの信号を、経路を形成する方角の集まりとして解釈する。
  • 壁を貫通したり180°回転せずに道を横断することができる、唯一の迷路を見つける。
  • 経路に対応するように、周波数を設定する。周波数の最後の3桁は次のように設定する必要がある。
    1. 迷路のID
    2. 経路の開始地点
    3. 経路の最終地点
  • 周波数を変更するには矢印を使用する。
    • 矢印を使用すると、現在の周波数が感度に応じて増減する。初期値は1hzである。
    • TXボタンを押すと、現在の感度が6倍になる。
    • TXボタンを5回押すと感度が1hzにリセットされ、現在表示されている周波数が送信される。
  • 周波数は三十六進数で表示される。
解釈方法
  1. 点滅と空白のパターンは、ある方角への1マス分の移動を示す。
  2. それぞれのパターンは異なる方向を示す。
  3. 方向の間には長い空白がある。
  4. 経路が繰り返される前に非常に長い空白がある。
