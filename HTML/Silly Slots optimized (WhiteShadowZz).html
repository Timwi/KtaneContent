On the Subject of Silly Slots

Sassy Sally said sorry since soggy Steven slurped soup.

Only press the KEEP button when the slots are in a LEGAL state. Only pull the lever when the slots are in an ILLEGAL state. The module will automatically defuse after 4 pulls of the lever.

The slots are in an ILLEGAL state if any of these statements are true:

Keyword: Sassy

  • There is a single Red Bomb.
  • There is a single Blue Cherry, unless the slot in the same position 2 stages ago was Green.
  • There are 2 or more Green Coins.
  • There are 3 Grapes, unless any of them are Blue.
  • There is a Bomb adjacent to a Cherry, unless every adjacent Cherry is Green.
  • There are exactly 2 Red slots, unless they are both Coin.
  • There is a single Green slot, unless the previous stage had any number of Bomb slots.
  • All 3 slots are the same symbol and color, unless there has been a Green Bomb in any previous stage.
  • All 3 slots are the same color, unless any of them are Cherry or there was a Red Coin in the last stage.
  • There are any number of Red Grapes, unless there has been a Blue Bomb in any previous stage.

Keyword: Silly

  • There is a single Green Grape.
  • There is a single Blue Coin, unless the slot in the same position 2 stages ago was Red.
  • There are 2 or more Red Cherrys.
  • There are 3 Bombs, unless any of them are Blue.
  • There is a Grape adjacent to a Coin, unless every adjacent Coin is Red.
  • There are exactly 2 Green slots, unless they are both Cherry.
  • There is a single Red slot, unless the previous stage had any number of Grape slots.
  • All 3 slots are the same symbol and color, unless there has been a Red Grape in any previous stage.
  • All 3 slots are the same color, unless any of them are Coin or there was a Green Cherry in the last stage.
  • There are any number of Green Bombs, unless there has been a Blue Grape in any previous stage.
