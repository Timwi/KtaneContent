À propos du Code d’Usine

La chaîne de production d’un produit peut être longue et complexe ; heureusement qu’il existe les mesures de contrôle qualité !

Utiliser différentes parties de la bombe pour trouver dans quel groupe elle a été construite et les caractères requis pour désarmer le module.

  1. Trouver le numéro de série de la bombe
    • Noter la première lettre du numéro de série.
  2. Observer les DEL allumées sur le module.
    • Utilisant le binaire, les DEL indiquent quelle lettre du code de désarmement (première, deuxième, etc.) est la même que la première lettre du numéro de série. Cette lettre n’apparaitra qu’une seule fois dans le code de désarmement.
