On the Subject of RGB Quiz

The chromatologist's nightmare.

Press the 'GO' button to activate the module. Upon doing so, music will start to play, and a 120 second timer will start.

You must complete three stages of the module, each at an increased difficulty of the previous, to disarm the module.

For each stage, note down the RGB color values of the cells in the grid.

Additionally, in stage 1, note down which color appears the most in this grid. Refer to this color as the index color. (This will only be red, green, or blue.)

For each subsequent stage, cycle the index color forward once in the sequence red, green, blue, wrapping around.

After noting down the grid, press the GO button. The cells will all go white and display question marks.

Toggle each cell in the grid in which the index color's value of that cell matches the index color's value of the color shown on the screen at the top left.

Entering an incorrect submission or letting the timer run out will incur a strike and reset the module.

Stage 1 will only show red, green, or blue.

Stage 2 will only show red, green, blue, cyan, magenta, or yellow.
