О Простом мастермайнде

Классическая игра по нахождению последовательности. Сможешь ли ты запомнить все?

  • Для решения этого модуля необходимо найти код из 5 правильных цветов.
  • Цвета, используемые на модуле: Белый, Маджента, Желтый, Зеленый, Красный и Синий.
  • На заметку: каждый цвет может использоваться несколько раз.
  • Цвет светодиода будет циклично изменяться при нажатии по светодиоду.
  • При нажатии на кнопку "Query" будет отображена информация о текущем коде.
    Левое число отображает количество верных цветов в верных позициях.
    Правое число отображает количество верных цветов в неверных позициях.
  • Для решения модуля необходимо ввести верный код и нажать на кнопку "Submit". При отправке неверного кода модуль зарегистрирует ошибку.
