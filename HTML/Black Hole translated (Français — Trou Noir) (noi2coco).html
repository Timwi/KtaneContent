À propos du Trou Noir

Contrôle au sol au Commandant Tom, vous me recevez ?

  • Pour désarmer tous les modules Trou Noir d’une bombe, il faut obtenir un code composé des chiffres 0 à 4. La longueur du code est égale à 7 fois le nombre total de modules Trou Noir sur la même bombe. À tout moment, vous pouvez entrer le chiffre suivant du code sur n’importe quel module Trou Noir non résolu.
  • Si vous résolvez un autre module de la bombe après avoir entré un chiffre du code, le code est raccourci de deux chiffres, sauf si votre chiffre a permis de résoudre un module Trou Noir. Ainsi, le code d’un seul Trou Noir peut être réduit de 7 à 3 chiffres en résolvant d’autres modules entre chaque chiffre. Le code pour deux Trous Noirs peut être réduit de 14 à 6, etc.
  • Observer la grille 10×10 de la page suivante. Calculer votre position de départ dans cette grille à partir du numéro de série de la bombe. Le troisième caractère est la coordonnée x, le sixième caractère la coordonnée y. Les colonnes et les lignes sont comptées de gauche à droite/de haut en bas à partir de 0.
  • Pour déterminer la direction initiale de votre mouvement, commencez par faire face au nord, puis tourner de 45° dans le sens horaire pour chaque port de la bombe.
  • Obtenir le premier chiffre du code de cet emplacement de la grille, se déplacer d’une case dans la direction initiale, puis tourner de 45° dans le sens horaire.
  • Le chiffre suivant est la somme, modulo 5*, de deux positions consécutives de la grille dans votre direction actuelle. Obtenir les deux chiffres, se déplacer d’une case au-delà, puis tourner de 45° dans le sens horaire.
  • Procéder de la même manière pour chaque chiffre suivant, en ajoutant à chaque fois une position de grille supplémentaire. Le n-ième chiffre du code est la somme (modulo 5*) de n positions consécutives sur la grille.
  • Chaque fois que vous dépassez le bord de la grille, faites le tour de l’autre côté.
  • La page suivante explique le geste à effectuer pour saisir chaque chiffre.
  • Le geste « C » peut être utilisé pour déterminer combien de chiffres corrects ont déjà été introduits, au cas où vous auriez perdu le fil.

* “modulo 5”: Ajouter/soustraire 5 jusqu’à ce que le nombre soit compris entre 0 et 4.
