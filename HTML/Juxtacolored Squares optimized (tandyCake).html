On the Subject of Juxtacolored Squares

Where's the colors? So drab...

  • See the original manual for further instructions.
  • Observe each color and convert it into a letter using the right diagram.
  • If colorblind mode is enabled, this letter will be displayed on the square. Black is the exception; its letter will not be displayed.
  • Use the below table to determine if a square should be pressed.
  • Squares will turn white when pressed, this does not count as changing their color.
