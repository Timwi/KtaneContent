Keep Talking and Nobody Explodes Mod Nokia

On the Subject of Nokia

Well you can’t call with anyone and this is not real Nokia. It just looks like Nokia...

Solving this module isn’t hard. You have to find out the right code to type in and press the green button.

  • Press numeric buttons to type the code
  • Press the green button to send the code
  • Press the red button to delete the typed code

Solving:

Look for... Then type...
1. Vowel in Serial number If FOUND, type 5.
If NOT FOUND, type 1.
2. Strikes If 0, type 0.
If 1, type 8.
If MORE, type 7.
3. The FIRST port found If PARALLEL, type 6.
If SERIAL, type 2.
If USB, type 1.
If OTHER OR NONE, type 4.
4. Batteries If 0, type 9.
If 1, type 5.
If 2, type 0.
If MORE, type 3.
5. Indicators BOB: (OFF - 3, ON - 8, NOT FOUND - 0
FRK: (OFF - 6, ON - 7, NOT FOUND - 2)

Notes:

  • Code resets every time you get a strike.
  • Ports are located in Appendix C (original bomb manual).
  • The code must contain only 6 digits!
