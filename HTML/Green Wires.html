On the Subject of Green Wires

Cut the green wire. Which one?

A “Green Wires” module has seven plugs on each side of the module, each of which can have a wire in it. There are always 7 wires with all the wires being shades of Green or there might be an error so there’s a purple wire.

There is a screen that shows a number in green. There is also a chance that a battery is present on the module (1/100). Only cut green wires unless told otherwise.

Use the first rule that applies to determine what wire needs to be cut to disarm the module. Wires are numbered 1 to 7 from left to right.

  1. If there is a battery on the module, cut any Brunswick Green, Green, or Olive wire.
  2. If there is a purple wire, you must cut the wire you get from Table A using the position of the purple wire as the row.
  3. If the number is 00 or 50, cut wire 2.
  4. If the number is 4, 8, 15, 16, 23, or 42, cut wire 3.
  5. If the number is higher than 100, cut wire 7.
  6. If there is an orange wire, cut it.
  7. If wire 3 or wire 6 is Olive, cut wire 4.
  8. If wire 5 is Brunswick Green, cut it. Otherwise, skip to rule 10.
  9. If the number is 00-99, cut wire 5.
  10. If the number is 69, cut wire 7.
  11. If wire 8 is yellow, cut it.
  12. If wire 5 or 6 is red, cut it.
  13. If wire 5 is Forest, cut wire 6.
  14. If the number is 7 or 14, cut wire 1.
  15. If wire 5 or wire 7 is Green, cut wire 2.
  16. If the module has two purple wires, cut the first one in reading order.
  17. If the number is a multiple of 17, cut wire 4.
  18. If the module has 10 wires, cut wire 9.
  19. If wire 1 is Brunswick Green or Jade, cut wire 3.
  20. If the number is 28, cut wire 7.
  21. If the number is 90 or higher, cut wire 5.
  22. If wire 7 is not green or purple, cut it.
  23. If wire 4 is Olive, cut wire 1.
  24. If the wires are neither shades of green nor purple, cut wire 5.
  25. If the number is 48, cut wire 6.
  26. If wire 4 is Emerald, cut wire 4. Otherwise skip to rule 28.
  27. If there are 7 wires, cut wire 6.
  28. Otherwise cut wire 7.
