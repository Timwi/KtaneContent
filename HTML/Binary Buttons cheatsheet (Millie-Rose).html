On the Subject of Basic Binary Buttons

Now it’s easier than 01, 10, 11.

On this module, you will see 5 digits of a binary number. Pressing each digit will toggle between a one and a zero. Use edgework around the bomb to determine which rule to use. Once you have found the correct rule, make that binary number with the digits on the module, then press submit. Submitting the wrong binary number will issue a strike.

Use the first rule that applies in the table below:

RuleInput
Lit BOB, 3+ batteries, parallel port00000
0 batteries, 4+ ports01101
Any lit indicators, PS2 port01011
