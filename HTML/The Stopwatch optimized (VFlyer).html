Exactly 3 Digits in the Serial Number

  1. Assume X being the 1st digit, Y being 2nd, Z being 3rd.
  2. If the number of batteries is odd and X * Y > Z * Y - X, +1 to all.

If there are:

  • 2 or 3 even digits: Base Time = 220 seconds (Even/Even cell based on the table)
  • 2 odd digits: Drop 1 of the lowest numbers. Use the table on the next page to grab the base time with the remaining numbers.
  • 3 odd digits: Base Time = 87 seconds. (Odd/Odd cell based on the table)
