En caso del código morse

¿Una anticuada forma de comunicación naval? ¿Qué será lo siguiente? Por lo menos es código morse genuino, así que presta atención y puede que aprendas algo.

  • Interpreta la señal de luz intermitente con la tabla de código morse para conseguir una de las palabras proporcionadas en la tabla de abajo.
  • La señal se repoducirá infinitamente, con una gran separación entre repeticiones.
  • Cuando la palabra sea identificada, pon la frecuencia correspondiente en el módulo y presiona el botón de transmisión "TX".
Cómo interpretarlo

1. Un parpadeo corto representa un punto.

2. Un parpadeo corto representa una raya/línea.

3. Hay una pausa grande entre letras.

4. Hay una pausa muy grande antes de que se repita la palabra.
