On the Subject of The Colored Fan

Another spin on things!

  • Use the colored buttons to set the fan to the goal speed presented on the display.
  • Each of the buttons modifies the speed of the fan by adding or subtracting some amount.
  • Setting the magnitude of the fan's speed to a number divisible by any factors presented on the display, will result in a strike.
  • Setting the fan to an invalid speed is valid if the speed is the goal speed.
  • If the fan spins too fast, it will explode.

Determining the Speeds

  • The display shows five letters (A–Z).
  • The goal speed of the fan is the numeric value of the letter with the same color as the button to the left or right of the button that has the color complementary to the color of the fan blades.
  • The initial speed of the fan is the numeric value of the letter that has color complementary* to that of the goal speed.
  • The remaining letters all refer to the factors to avoid. All of the factors have been modified according to table 1.
  • If none of the letters have a color matching the color of the initial or final speed, then that speed is zero and there is a fourth factor to avoid.
  • If the fan blades are see-through, the initial and final speeds swap.
  • The initial speed of the fan is always opposite in direction to the goal speed. 0 is treated as positive.
*complementary colors are red-green, yellow-purple, and blue-orange.

Identifying the Buttons

The standard order for the buttons is:

Red, Green, Yellow, Purple, Blue, Orange.

  1. Start with the order of the colors on the module (reading order)
  2. For each pair of colors, if the index of the color that appears first in reading order in the standard order for the buttons is higher than that of the second, that pair is an inversion.
  3. Find the index of the blue button in reading order where the top left button is 0 and the bottom right button is 5.
  4. Find (The number of inversions + the index of the blue button) % 6. This is the modification number.
  5. Using table 2, find the modification values for each button based on the index.

Each button’s effect corresponds to its position in the reading order. For example, if the modification number is 2, the first button adds 2, the second subtracts 2, etc.
