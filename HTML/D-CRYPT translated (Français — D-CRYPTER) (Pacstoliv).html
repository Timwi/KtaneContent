À propos de D-CRYPTER

SALUTATIONS, HUMAINS.

Ce module consiste en cinq hiéroglyphes extraterrestres.

Chaque hiéroglyphe correspond à une lettre, comme indiqué dans le tableau ci-dessous.

Ensemble, les cinq hiéroglyphes épellent un mot du Tableau des Mots dans l'ordre de lecture.

Si un hiéroglyphe est maintenu, il passera en revue tout l'alphabet des hiéroglyphes. Le relâcher bloquera le hiéroglyphe affiché.

Une fois que tous les hiéroglyphes sont bloqués, le module validera le mot épelé par les hiéroglyphes.

Pour déterminer le mot à valider, prendre le mot sur le module et avancer dans l'ordre de lecture dans le Tableau des Mots selon le nombre de piles + le dernier chiffre du numéro de série, en retournant au début de la liste si la fin est atteinte.

Si le nombre obtenu est 0, utiliser 7 à la place.
