Keep Talking and Nobody Explodes Mod Spongebob Birthday Identification

On the Subject of Spongebob Birthday Identification

All aboard at the Intracoastal Yatch Club!

  • The module consists of a keyboard, a text box, a display with an image of Needy Birthdays, 3 unlit light bulbs, and an image of a crying Spongebob.
  • To initiate a stage, press the entrée key on the keyboard. The crying Spongebob will then reveal a Spongebob Birthday Invitation with the name blurred out.
  • Type the exact name of the child shown in the module including punctuation if needed. Submit your answer by pressing the Entree button on the keyboard. The submission is case-sensitive.
  • If the input given was correct, a light bulb beside the border will light up. Light up all the light bulbs given to solve the module.
  • Reference images for the birthday invitations can be found in the table on the next page.
