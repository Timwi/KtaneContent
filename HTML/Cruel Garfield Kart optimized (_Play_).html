In the table above, each track/round pair has a string. Words in [square brackets] indicate actions with items and words in (round brackets) indicate other actions. The first square brackets in the string indicate the starting items, with the brackets being empty for no items. The number after the starting items is the starting placement of the player. All numbers after that are the player’s placement changes. The string goes in a sequence from the start of the video to the end. If multiple numbers are not separated by spaces, that means that the placement switches very quickly, and might not be instantly noticeable.

After determining the track:

  • If the track is Play Misty for Me: your final code is 666661
  • If there is exactly 24 batteries in 12 holders: your final code is 111111

If any of the above rules applied, skip to submission phase. Otherwise, keep reading the manual.
