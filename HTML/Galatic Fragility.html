On the Subject of Galatic Fragility

Houston, we may have a number.

This module is an alien authentication device. The device’s largest feature is a two-digit seven-segment display timer which counts down cycling through all numbers. Use the objects present in the four spaces between timer segments to determine a number that, when displayed on the next timer cycle as you press the button, passes authentication. Each object corresponds to a single constraint, and may change on a new timer cycle. Valid numbers abide by all four constraints. Pressing the button on an invalid number results in a strike.

Compass

The compass’ constraint only applies to the segments the compass’ needle is pointing to. If the needle is red, all the relevant segments are lit. If the needle is black, all the relevant segments are unlit.

Domino

Count the number of pips on each half of the domino. The absolute difference between the two digits ranges between these two numbers. If the pips are white, this range in inclusive. If the pips are black, this range is exclusive.

Gear

Look at how the gear is spinning. If the gear is spinning clockwise, the digit on the right is the greater of the two. If the gear is spinning counter-clockwise, the digit on the left is greater. If the gear is stationary, the two digits match.

Matrix

The matrix’s constraint only applies to the digit the matrix is inside of. Count the number of squares that are lit at any single moment. If the matrix is present on the top half of the digit, this number is an upper bound for what the digit can be. If the matrix is present on the bottom half, this number is a lower bound.

PCB

The PCB’s constraint only applies to the segments next to each end of the trace or traces present on the board. If the chip is square relative to the board, the segments are all lit or all unlit. If the chip is at 45° relative to the board, exactly one segment is lit.

Screws

The displayed number must be divisible by the number of screws that are screwed into this space. If there are Phillips screws (), the display is read normally. If there are Torx screws (), read the display with the tens digit on the right and the ones digit on the left.

Stamp Markings

If a marking says “CLASSIFIED”, the number of lit segments is prime. If a marking says “DECLASSIFIED”, the displayed number is prime. Multiple markings may be present. If only one marking is present on the module, the other quantity is not prime.

Sticker

At least one of the four object spaces are surrounded by lit segments which match that of the letter present on the sticker encoded using a Pigpen cipher.

Thermo

Label the segments of a seven-segment display A through G in reading order. The number of lit segments labelled with the letter shown next to the thermo equals that of the temperature reading.

Vectorscope

Label the boxes on the vectorscope 0 through 9, starting at north and going clockwise. The line from the center will go through one box. Neither of the digits can be this box's number.
