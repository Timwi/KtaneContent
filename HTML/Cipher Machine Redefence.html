On the Subject of the Redefence CM

A machine that knows no bounds when it comes to ciphers.

The numbers on screen 1 will be referred to as the Redefence Key.

If the Submit Button is white, follow the instructions under Encrypt Instructions. Otherwise follow the instructions under Decrypt Instructions.

Encrypt Instructions

  • Write the encrypted word in a zig zag pattern with the number of rows equal to the length of the Redefence Key.
  • Read the letters in the order of the Redefence Key, treating each row as if it is labeled 1-X in top to bottom order to get your encrypted word.

Decrypt Instructions

  • Write an X amount of asterisks in a zig zag pattern with the number of rows equal to the length of the Redefence Key. X is equal to the encrypted word's length.
  • Replace each asterisk with the letters of the encrypted word in the order of the Redefence Key, treating each row as if it is labeled 1-X in top to bottom order.
  • Read the letters in a zig zag patterns to get your decrypted word.

Example 1

Encrypted Word: ASPOLRAT
Redefence Key: 32415
Using Encrypt Rules

1: A               
            2:   S                       T
        3:       P               A
    4:          O       R
5:               L

Decrypted Word: PASTORAL
