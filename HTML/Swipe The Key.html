On the Subject of Swipe The Key

*beep*

A bomb with this module will ship with a Keycard Folder.

To solve this module, scan the correct keycard in the correct direction.

Each keycard has an 8-digit ID number. You must modify this number to determine which card to use. Find the symbol on the module in the table below, then align each digit with a column in order. Move any digits to the left of the symbol all the way to the right of the number, keeping them in order. You need to scan the keycard with the greatest or least modified ID number. Use the greatest if the first digit in the bomb’s serial number is greater than the last, and the least otherwise.

If the symbol on the module is in the top row of the table below, scan the correct keycard from the top of the module to the bottom. Otherwise, the symbol will be in the bottom row, and you must scan the keycard from the bottom of the module to the top.
