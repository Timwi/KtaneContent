On the Subject of I Got No Legs D:

Meatgrinder would be a very funny addition...

Use the arrow buttons to scroll through the five pages.

Each page has a count (the number of animals of the animal type) on the top left screen, an animal type on the top right screen (sometimes with a unique quirk written with small text font) and a quirk on the bottom screen.

For each animal type, per one animal, first apply the unique quirk (if present) then the quirk on the bottom of the screen.

Afterwards, using numbers of animal types present, count how many legs all animals have in total and input it using the number buttons.

The button labeled S submits the number entered, and the button labeled C clears the current input.

If the final answer is less than zero, submit zero instead.
