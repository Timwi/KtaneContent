On the Subject of the Malespín CM

A machine that knows no bounds when it comes to ciphers.

This cipher uses 2 pages.

Concatenate the 6 keywords in this order: page 1 screen 1, page 1 screen 2, page 1 screen 3, page 1 screen 4, page 2 screen 1, page 2 screen 2. Remove any duplicate letters while keeping the first occurrences.

Split the key in half and write the 2nd half of the key underneath the 1st half. This will be referred to as the Malespín Key.

For each letter of the encrypted word, do the following:

  • Find the Nth letter of the encrypted word in the Malespín Key.
  • Replace it with the letter above/below it in the Malespín Key.
  • Do this for each letter to get an decrypted word.

Example

Encrypted Word: SBDNPMI
Keywords: VEXING, SQUEEZE, FORCIBLY, INSTEAD, HAWK, JUMPS

CBLYTADHWKJMP

SBDNPMI → DESTROY
