1 R +--

Get the color that appears the most at the start. This is known as the reference index.

Take notes of the color displayed currently and press "GO!"

Where the first column is the current stage and the second column is the current reference index, obtain the row corresponding to the color displayed in the TL and select all colors previously displayed that are in the same row.

Shift the reference index 1 right in RGB order (R → G, G → B, B → R), and repeat the previous 2 steps, 2 more times, until the module is disarmed.

And do all this in 120 seconds or less.
