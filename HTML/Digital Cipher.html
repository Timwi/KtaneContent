Keep Talking and Nobody Explodes Mod Digital Cipher

On the Subject of Digital Cipher

Let’s digitize everything...

This module consists of 2 big screens. The upper one is the input screen and the bottom one the output screen.

In order to solve the module, take the string displayed on the input screen. Call it String A. Take String A and reverse it; call this String B. Take the numerical value of the first letter in String A and the first letter in String B and add them up. Take the digital root of the sum and then subtract 1. Click the button which corresponds to this digit. Repeat for the whole sequence.

In order to get the numerical value of a letter follow this sequence: A=0, B=1, C=2 and so on.

If the digital root before subtracting 1 is already 0, use 0 instead.
