À propos des Boutons Compliqués

Ces boutons ne sont pas comme les autres. La bonne nouvelle c’est qu’on a trouvé des instructions concises sur quoi faire ! Peut-être trop concises...

  1. Chaque bouton a une ou deux couleurs, et un libellé.
  2. Pour chaque bouton, utiliser le diagramme de Venn ci-dessous pour déterminer si le bouton doit être pressé ou pas.
  3. Presser les boutons dans l’ordre indiqué dans le tableau ci-dessous. Si aucun bouton ne doit être pressé, presser le second bouton dans la liste.
  4. Si une erreur est enregistrée, le module se réinitialisera.
