LED 암호화에 대하여

오옹! 빤짝이 빛이랑 버튼이 있어요. 만지작만지작!

  • 모듈의 상단에 설치된 2~5개의 LED가 각 단계를 의미합니다. 모듈을 해제하려면 이 단계를 순서대로 완료해야 합니다.
  • 4개의 서로 다른 로마자가 쓰인 4개의 버튼이 있습니다. 각 단계마다 로마자는 바뀝니다.
  • 모듈 왼쪽 위에 현재 단계가 숫자로 표시됩니다.
  • 현재 단계에 해당하는 LED의 색깔을 이용해 다음과 같이 현재 단계의 곱수를 계산합니다.
    붉은색 초록색 파란색 노란색 보라색 주황색
    2 3 4 5 6 7
  • 로마자 각각을 0–25 범위의 수로 변환합니다. (A = 0, B = 1, C = 2 등.)
  • 다음을 만족하는 올바른 버튼을 찾으세요: "버튼의 로마자의 변환값과 현재 단계 곱수의 곱을 26으로 나눈 나머지가, 그 버튼에서 대각선으로 반대쪽에 있는 로마자의 변환값과 같음."
  • 각 단계마다 올바른 버튼 중 하나를 누릅니다. 올바른 버튼이 여러 개일 수도 있습니다.
