On the Subject of Five Letter Words

Rules are quite basic.

The module contains 3 five-letter words. Select the word with the highest value when the seconds in the bomb’s timer equal the total score of the lowest scoring word shown in the module.

Use the table below to calculate the value of the words. Add the values of the letters to calculate the value of the word.
