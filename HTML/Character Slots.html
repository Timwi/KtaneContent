Keep Talking and Nobody Explodes Mod Character Slots

On the Subject of Character Slots

“WHO ARE YOU PEOPLE?”

This module resembles a video poker slot machine, with three screens representing character stock icons from the game Super Smash Bros. Ultimate. Each screen also has a “keep” button below it and a status light above it, being red if the respective slot is “kept”.

At the top of the module are three status lights, representing the 3 possible stages. If you have completed a stage, you can press the corresponding light to see this stage once again, in case you forgot something.

For each stage, identify each character and decide if you have to keep the slot or not, according to the conditions below. If the obtained score is strictly positive, keep this slot. Otherwise, do not.

Press the “crank” button when you think you’re done. If you’ve kept a slot when you didn’t have to or vice-versa, a strike will occur. This won't reset the stage.

This module is solved when :

  • You correctly pressed the “crank” button on the third stage.
  • You correctly pressed the “crank” button on any stage, and you kept every slot correctly.
  • You pressed the “crank” button while keeping all 3 slots displaying the same character.

About relationships

Check which other characters have appeared, INCLUDING FROM OTHER STAGES. If a character appears in the “liked” category, add 1 point. If they appear in the “disliked” category, subtract one point.

Note: Do not count this instance of this character, even if it is at a previous stage. Count instances of other characters only once. However, if you rolled the character again, count it.

About other modules

Each character has liked and disliked modules. If a module is present on this bomb, then it is worth 1 point. If one of these module is solved, then it is worth 2. If it belongs to the “liked” category, add those points to the character's score. If it is in the “disliked” category however, subtract them instead.

Having multiple instances of a module does not affect the score. Also, the module ignores itself, but not other instances of it.

What about Mii Fighter?

The Mii Fighter counts as a joker. You are allowed to keep them for 1 stage only. Trying to keep the same Mii Fighter during two stages in a row will result in a strike.

Shoot! I forgot what was on a previous stage!!!

In case you forget which characters have appeared in an earlier stage, you can press the top bulb corresponding to that stage to make them reappear for a brief moment.
