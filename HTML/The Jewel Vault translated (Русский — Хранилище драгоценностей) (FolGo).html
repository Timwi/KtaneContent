О Хранилище драгоценностей

Как древние греки придумали такой сложный кодовый замок? Это должно быть пришельцы!

  • Модуль содержит: пяти дисковую дверь, компас, кнопка обнуления (слева снизу) и кнопка отправки решение (справа снизу).
  • На внешнем неуправляемом диске изображены восемь греческих символов. В остальные диски вставлены по четыре разных цветных драгоценных камней, которые можно вращать.
  • Всего можно встретить восемь драг. камней. Каждый драг. камень может появиться только раз на одном колесе:
    • Аметист (фиолетовый)
    • Изумруд (зелёный)
    • Стекло (серебряный/прозрачный)
    • Оникс (чёрный)
    • Пудреттеит (розовый)
    • Рубин (красный)
    • Сапфир (синий)
    • Скаполит (жёлтый)
  • Для решения модуля, установите положение драг. камней в правильной ориентации и нажмите кнопку отправки ответа.
  • Отправка неверного положения камней зарегистрирует ошибку, перекручивает диски и меняет ориентацию компаса.
  • Если сумма поворотов дисков достигнет тринадцати раз без обнуления, тогда диски перекрутятся и изменится ориентация компаса.
  • При нажатии кнопки обнуления, диски вернутся в свое начальное положение.
  • Центральная сфера на двери меняется в зависимости от текущего действия:
    • Белый = неподвижное
    • Желтый = вращение диска
    • Синий = обнуление
    • Розовый = достигнут лимит вращений
    • Красный = ошибка
    • Зелёный = решён

Как работают диски

  • Четыре диска будут номероваться одним из двух способов:
    • Диски 1–4 относятся к физическому положению (1 внешнее, 4 внутреннее).
    • Диски A–D относятся к механике, как диски влияют друг на друга и они расположены случайно.
  • Механика буквенных дисков таковы:
    • Диск A вращается по часовой.
    • Диск B вращается по часовой и диск A против часовой.
    • Диск C вращается по часовой и диск B против часовой.
    • Диск D вращается по часовой и диск C против часовой.

Греческие буквы

  • Восемь букв определяют правильное положение драг. камней на каждом диске.
  • К каждому диску присвоены 2 буквы. Используйте таблицу снизу для определения приоритетов драг. камней.
  • Самый высокий приоритет имеет тот, который имеется на диске.
  • Заметка: таблица снизу относится к дискам 1–4.

Целевая орентация

  • Целевая орентация четырёх драг. камней определена самым ценным камнем.
  • Если более одного драг. камня имеют одинаково высокую ценность, возьмите последнею цифру серийного номера и используйте его в таблице приоритетностей для определения какой самый ценный драг. камень. Если последняя цифра серийного 0, то используйте 10.
