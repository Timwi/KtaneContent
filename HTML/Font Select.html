On the Subject of Font Select

Until now, you’ve only had to deal with the font Special Elite and whatever font that appears in the module, and usually it wasn’t important, That changes today.

This Module will present a display, two arrow keys and a submit button. The display will show a phrase and the arrow keys will shift the font. Locate the phrase in Table A and its corresponding number, locate the row with the same number in Table B and submit the first font out of the list that’s present on the module. Font names, as well as an example phrase for in each font, can be found on the following page.
