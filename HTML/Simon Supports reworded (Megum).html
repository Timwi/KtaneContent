On the Subject of Simon Supports Tax Evasion

We couldn't agree on a module idea so we decided to commit fraud.

This module will display five colored ties surrounding a large square button.

There are going to be 3 flashes, multiple colors each, in a cycle. The cycle restarts each time the square button flashes.

Each flash has one descriptor associated with it. To find said descriptor, examine the flashing colors. The desired descriptor will be present in all of the flashing colors' lists (Table A) but will be absent from the non-flashing colors' lists.

To disarm the module, press the square button within one cycle on each positive descriptor's (Table B) flash. If no present descriptors are positive, press the button during the cycle restart.
