Keep Talking and Nobody Explodes Mod 残忍マスターマインド

モジュール詳細：残忍マスターマインド

昔からあるゲームだ。少し難しくなっただけ。あと覚えにくくなった。あと前のような簡単な解き方ではなくなった・・・

  • モジュールを解除するには正しい5つのLEDの配色を見つける必要がある。
  • LEDの色は白、マゼンタ、黄色、緑、赤、青のうちのどれか。
  • 答えに同じ色が複数回使われることがある。
  • LEDをクリックすることでその色を変えることができる。
  • "Query"(質問) ボタンを押すことで、現在の配色の状態が液晶に表示される。
  • 正しい配色を入力して"Submit"(送信)ボタンを押せば解除される。間違った配色はミスとなる。

"Query"(質問) ボタンを押す度に二つの数字が液晶に表示される。これらの数は下の表に該当する二つの数の合計を表している。

A = 正しい色が正しい位置にある数。
B = 正しい色が間違った位置にある数。
C = 間違った色がある数。

液晶の数字の色: 左の数値 右の数値
白 左の数字の色: A B
右の数字の色: バッテリーの数 解除したモジュール数
マゼンタ 左の数字の色: C A
右の数字の色: 点灯したインジケーターの数 シリアルナンバーの最後の数字
黄色 左の数字の色: B C
右の数字の色: シリアルナンバーの数字の合計 ポートの数
緑 左の数字の色: B A
右の数字の色: 爆弾のモジュール数 点灯していないインジケーターの数
赤 左の数字の色: C B
右の数字の色: 爆弾にあるポートの種類の数 ミスした回数
青 左の数字の色: A C
右の数字の色: シリアルナンバーの最初の数字 バッテリーホルダーの数
