On the Subject of Blackjack

We are legally required to say that this module is not endorsing gambling and that you should gamble like there’s no tomorrow at your own risk.

  • Your cards will be notated as C1, C2, C3, C4, C5.
  • Your score is the sum of all of your cards. J = Q = K = 10. A = 1.
  • To solve the module, win. To win, either:
    • Get to 21.
    • Not go lower than the dealer while going below 21.
    • Make the dealer go over 21.
  • Unicorn: If C1 = A, C2 = 10/J/Q/K, Check Blackjack.

Determine C2

If C1 = A♠,

  • lit BOB, C2 = K♥.
  • SN vowel, C2 = 5♦.
  • SN sum > 7, C2 = 7♠.
  • Otherwise, C2 = 2♣.

If C1 = K♦,

  • has CLR, CAR, FRQ, SIG, NSA, MSA, TRN, BOB, FRK indicator (GAMBLER), C2 = Q♥.
  • D batt > 1, C2 = 9♠.
  • Serial port, C2 = 3♦.
  • Otherwise, C2 = 4♣.

If C1 = 2♥,

  • ports > battery, C2 = A♦.
  • NSA indicator, C2 = 3♥.
  • batt = 0, C2 = 7♣.
  • Otherwise, C2 = 4♠.

If C1 = 10♣,

  • SN has any letter of "CASINO", C2 = 5♣.
  • AA batt > 3, C2 = 3♥.
  • Para port, C2 = 6♦.
  • Otherwise, C2 = A♠.

See instructions on betting and determining C4 and C5 on the next page.
