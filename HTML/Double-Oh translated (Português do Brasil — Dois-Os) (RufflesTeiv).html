A Respeito dos Dois-Os

Um módulo, por favor. No jeitinho. Que lixo inútil; display quebrado, botões quebrados...

  • O módulo possui um display que mostra um dígito de dois números, assim como cinco botões. Usando os botões, mude o display para 00 (localizado no centro da tabela abaixo), e então aperte o botão de enviar.
  • Quatro dos cinco botões mudam o número no display. Baseando-se na tabela abaixo, que consiste em uma matriz 3×3 de matrizes 3×3 menores, os botões irão se comportar da seguinte maneira:
  • O botão “↕” move para a próxima posição para cima ou para baixo dentro da matriz menor atual, fazendo um loop se ultrapassar a beira da matriz (exemplo: 00 para 85 para 14 para 00).
  • O botão “↔” move para a próxima posição para a esquerda ou para a direita dentro da matriz menor atual, fazendo um loop se ultrapassar a beira da matriz (exemplo: 00 para 56 para 21 para 00).
  • O botão “⇕” move para a próxima posição da próxima matriz 3×3 grande para cima ou para baixo, fazendo um loop se ultrapassar a beira da matriz (exemplo: 00 para 22 para 58 para 00).
  • O botão “⇔” move para a próxima posição da próxima matriz 3×3 grande para a esquerda ou para a direita, fazendo um loop se ultrapassar a beira da matriz (exemplo: 00 para 44 para 65 para 00).
  • O botão “▣” é o botão de enviar. Apertá-lo irá desarmar o módulo se o número 00 estiver sendo mostrado.

NOTA: Este módulo é velho, e o último dígito buga quando o primeiro dígito for zero. Além disso, o circuito dos botões está frito, então as funções dos botões podem trocar entre si. Felizmente, você só terá problemas ao apertar o botão de enviar se o número mostrado for menor que 10.

NOTE: This module is old, and the last digit glitches out when the first digit is zero. In addition, the wiring for the buttons is acting up, so the functions of the buttons may be swapped. Fortunately, the strikes from the submit button are only registered by the bomb if the displayed number is less than 10.
