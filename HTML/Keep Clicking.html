On the Subject of Keep Clicking

“Oh, this is keypads!” No, not really. Good luck

Each of the three buttons will have a symbol. Clicking a button changes its symbol. For every button, do the following:

  1. Identify the type of the button as follows:
    1. If the button is in the middle and there are more than 2 batteries on the bomb, it is a pN button.
    2. Otherwise, if the button is on the left and there is a lit indicator with label SIG, it is an aK button.
    3. Otherwise, if there is a lit indicator that ends with the letter "R," it is a gX button.
    4. Otherwise, if there is a vowel in the serial number, it is an aK button.
    5. Otherwise, it is a gX button.
  2. Click the button until it displays any of the valid stop symbols for that button:
