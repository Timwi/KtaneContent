Keep Talking and Nobody Explodes Mod Base Tierce

A Propos de la Base Tierce

Ce module est similaire à -Who's On First-, mais avec 4 caractères par touche au lieu des mots et phrases. Ca devrait être facile... pas vrai?

  1. Comparer le code de l'afficheur avec le tableau de l'étape 1 pour déterminer la touche dont il faut lire le code.
  2. Se référer à l'étape 2 pour déterminer la touche à presser en se basant sur le code lu.
  3. Répéter ces étapes jusqu'à désarmer le module.
