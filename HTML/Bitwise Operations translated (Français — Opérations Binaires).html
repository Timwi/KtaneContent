À propos des Opérations Binaires

La moins appréciée des mathématiques. D’ailleurs, qui aime les maths, hein ?

  • 2 afficheurs sont présents sur ce module :
    1. L’opérateur binaire (AND, OR, XOR, NOT)
    2. La saisie de la résultante
  • Utiliser les deux octets obtenus dans le tableau ci-dessous et l’opérateur binaire de l’afficheur pour déterminer la réponse. Dans ce tableau, MSB est le bit le plus significatif, LSB le bit le moins significatif.

Voici le détail de chaque opérateur binaire :

Info AND OR XOR NOT
COMMENT En allant bit par bit, si les deux bits sont à 1, le résultat est 1. Sinon le résultat est 0. En allant bit par bit, si l’un des deux bits est à 1 (ou les 2), le résultat est 1. Sinon le résultat est 0. En allant bit par bit, si l’un des deux bits est à 1 (pas les 2), le résultat est 1. Sinon le résultat est 0. Ignorer le second opérande. En allant bit par bit, le résultat est son opposé.
