On the Subject of GTA Cheats

You have 5 stars. You’ve got to run. No Car. No Weapons ... Just Cheat!

  • To disarm the module you have to use the correct cheats to reach exactly 0 stars.
  • Follow the instructions in the table below from top to bottom based on the number of stars present on the module.
  • Once you find the correct instruction, use the CheatSheet on the next page to find the correct input sequence to press.
  • If you press a wrong button, a strike will be incurred and 1 star will be added.
  • If you remove more stars than those present on the module, the exceeding stars will be added back.

    (Example: If you have 2 stars and you remove 3 stars, you will have 1 star remaining)

Attention!

Every cheat removes a fixed number of stars.

That number of stars will always increase by 1 if the first button of the input is pressed when the seconds timer contains the number obtain from the following operation:

highest # in the serial + ( # batteries × current # of stars ) - # battery holders

(If this yields a two-digit number, sum up the digits until you have a one-digit number)

Instruction Table

5 stars:

  • If there is a lit/unlit BOB indicator and a unlit CAR indicator: PAINKILLER
  • Otherwise, if there is an empty port plate: HOTHANDS
  • Otherwise, if the serial number contains an odd number: BUZZOFF
  • Otherwise: LAWYERUP

4 stars:

  • If there is a DVI-D port and a PS/2 port: TOOLUP
  • Otherwise, if there is a parallel port or at least 2 D batteries: HIGHEX
  • Otherwise, if there are 2 or more lit indicators: RAPIDGT
  • Otherwise: LAWYERUP

3 stars:

  • If there are 3 or more port plates: TURTLE
  • Otherwise, if there are more lit indicators than unlit: GOTGILLS-JUMP
  • Otherwise, if the are no indicators: GOTGILLS-SWIM
  • Otherwise: LAWYERUP

2 or fewer stars:

First Occurrence

  • If there are more digits than letters in the serial number: CATCHME
  • Otherwise, if there are 2 or more pairs of AA batteries: INCENDIARY
  • Otherwise, if there is a Stereo RCA port: BANDIT
  • Otherwise: LAWYERUP

Second Occurrence

  • If any indicator has a matching letter with the serial number: BANDIT
  • Otherwise, if there is a vowel in any indicator: CATCHME
  • Otherwise, if there are more letters than digits in the serial number: INCENDIARY
  • Otherwise: LAWYERUP

Three or more occurrences

  • If there is an equal number of digits and letters in the serial number: INCENDIARY
  • Otherwise, if there is a RJ-45 port: BANDIT
  • Otherwise, if there is a serial port: CATCHME
  • Otherwise: LAWYERUP
