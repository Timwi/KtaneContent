INCOMING TRANSMISSION...

The module presents you with three alien glyphs.

Each glyph corresponds to a digit via the table below.

Press each glyph when the last digit of the bomb’s timer is that glyph’s digit associated.

Pressing a glyph at an incorrect time will record a strike, but not reset the module.
