À propos de Tourner les Clés

L’ordre, c’est tout.

Ce module dispose de deux clés et d’un afficheur. L’afficheur indique la priorité de ce module.

CLÉ GAUCHE

Tourner la clé gauche après avoir effectué tout ce qui suit :

  • Tourner la clé droite sur tous les modules « Tourner les Clés »
  • Tourner toutes les clés gauches de priorité inférieure.
  • Résoudre tous les modules Mot de Passe.
  • Résoudre tous les modules Jeux de Mots.
  • Résoudre tous les modules Conversation de Fous.
  • Résoudre tous les modules Clavier.
  • Résoudre tous les modules Écoute.
  • Résoudre tous les modules Cube d’Orientation.

Mais avant d’avoir fait l’une des choses suivantes :

  • Tourner toutes les clés gauches de priorité supérieure.
  • Résoudre tous les modules Labyrinthe.
  • Résoudre tous les modules Memory.
  • Résoudre tous les modules Fils Compliqués.
  • Résoudre tous les modules Séquences de Fils.
  • Résoudre tous les modules Cryptographie.

CLÉ DROITE

Tourner la clé droite après avoir effectué tout ce qui suit :

  • Tourner toutes les clés droites de priorité supérieure.
  • Résoudre tous les modules Code Morse.
  • Résoudre tous les modules Fils.
  • Résoudre tous les modules Deux Bouts.
  • Résoudre tous les modules Bouton.
  • Résoudre tous les modules Flash de Couleurs.
  • Résoudre tous les modules Clavier Rond.

Mais avant d’avoir fait l’une des choses suivantes :

  • Tourner toutes les clés gauches.
  • Tourner toutes les clés droites de priorité inférieure.
  • Résoudre tous les modules Sémaphore.
  • Résoudre tous les modules Serrure à Combinaison.
  • Résoudre tous les modules Simon.
  • Résoudre tous les modules Astrologie.
  • Résoudre tous les modules Interrupteurs.
  • Résoudre tous les modules Plomberie.
