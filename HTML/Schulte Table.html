Keep Talking and Nobody Explodes Mod Schulte Table

On the Subject of Schulte Table

Just press all the numbers in order. Easy enough, right? Right?

The module presents a 5×5 grid and a screen.

Press any square in the grid to initiate a 30-second timer. The grid will also display rotating numbers from 1 to 25 in each grid in a random order.

To solve the module, press each number in ascending order if the last digit of the serial number is even, otherwise press each number in descending order.

Pressing a wrong number will yield a strike and reset the module. Letting the timer run out will not yield a strike, but still reset the module.

Note that the number 6 on the module has a tail curvier than 9 on the module.
