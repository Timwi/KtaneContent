On the Subject of Minecraft Cipher

Creeper? AWW MAN! So we back in the mine...

Translate the first two SGA symbols into English letters (see Appendix: SGA). Convert each letter to its alphabetic position (A=1…Z=26).

For each of the two letters, apply the first matching priority rule: first-letter rules for the first letter, second-letter rules for the second. Add the modifier of the rule to the value of the letter, and turn the result back into a letter.

First-letter priorities (first matching)
  • If the bomb has a lit IND indicator, apply -3.
  • Else, if the bomb has no batteries, apply -2.
  • Else, if the bomb has no indicators, apply +3.
  • Else, if the bomb has The Crafting Table module, apply +2.
  • Otherwise, add the number of ports.
Second-letter priorities (first matching)
  • If the bomb has a parallel port and exactly four battery holders, apply +4.
  • Else, if the first letter after modification is a multiple of 3, apply +2.
  • Otherwise, subtract the smallest digit of the serial number.

If either number is out of the range of 1-26, repeatedly add or subtract 26 until it is.

Find the row that begins with the number of letters in the encrypted word, followed by your two letters.

In each column with an operation token, perform that operation (see Operations table) to the letter above it. Empty cells mean the letter is used as is.

Perform all operations and then convert the letters back to the Standard Galactic Alphabet (Appendix SGA) and submit it.
