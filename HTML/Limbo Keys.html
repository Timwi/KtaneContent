Keep Talking and Nobody Explodes Mod Limbo Keys

On the Subject of Limbo Keys

IT’S BLUE!!! IT’S BLUE!!! IT’S BLUE!!! IT’S BLUE!!!

The module presents a black screen with one red key on it. Tap the screen to start the sequence.

The word “FOCUS” will appear, followed by eight keys. One of these keys will flash green for a brief moment — this is the target key.

The keys will then rotate and shuffle rapidly, set to music. Once the keys have finished shuffling, each key will change to a different color and start flashing, individually. Tap the screen again when the target key is flashing, to solve the module.

Selecting an incorrect key will result in a strike and reset the module.

There are three options available in this module’s mod settings:

  • BossMode: If enabled, this module will be inactive, until all non-ignored modules have been solved. If you strike while this mode is active, the bomb will detonate.
    • While the module is inactive, there will be an eye in place of the red key, and the display will not be selectable.
  • FocusMode: If enabled, each key will only flash once, during submission. If no key is selected, the module will strike.
    • The border of the module will be red, if this setting is enabled.
    • If BossMode is enabled, this setting is also enabled.
    • If Twitch Plays is active, this setting is disabled, no matter what.
  • MuteMusic: If enabled, the music that plays while the keys are swapping does not play.
    • If either FocusMode or BossMode are enabled, this setting is disabled.

In mission descriptions, you may add [Limbo Keys] , followed by one of music, focus or boss, followed by either on or off, to enable / disable music, enable / disable focus mode, and enable / disable boss mode. This overrides mod settings.
