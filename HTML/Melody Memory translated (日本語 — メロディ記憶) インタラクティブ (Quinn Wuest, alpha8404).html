Keep Talking and Nobody Explodes Mod メロディ記憶

モジュール詳細：メロディ記憶

音楽はあなたの制御を失わせる！

  • このモジュールは4つの鍵盤、再生ボタン(Play)、録音ボタン(Record)、及びモジュールの進行状況を示すLEDで構成されている。モジュールを解除するには、5つのステージ全てを完了する必要がある。
  • 4つの鍵盤を押すと、それぞれが異なる楽器を使用して4つの異なる音を鳴らす。通常は鍵盤を押しても入力に影響されない。ただし、録音ボタンが押された後に一度だけ入力が記録される。
  • 再生ボタンはモジュールにある1つの音を再生するが、4つの鍵盤に存在しない楽器を使用している。この音を鳴らす楽器を段に、この音と同じ音程を再生した鍵盤の色を列に使用して文字を取得する。

Keep Talking and Nobody Explodes Mod メロディ記憶

取得した文字に対応するリストと現在のステージ数を見て、押す必要がある鍵盤を特定する。

A:
  1. 赤の鍵盤
  2. ステージ1で緑の鍵盤があった位置の鍵盤
  3. ステージ2で青の鍵盤だった音程の鍵盤
  4. ステージ3で押した鍵盤の色の鍵盤
  5. ステージ1で黄の鍵盤と同じ音程だったステージ4の鍵盤の色の鍵盤
C:
  1. 存在する楽器のうち、表の上から3番目にある楽器の鍵盤
  2. ステージ1で押された鍵盤の楽器に、表において最も近い楽器の鍵盤
  3. ステージ2の赤の鍵盤の楽器に、表において最も近い楽器の鍵盤
  4. ステージ3の2番目に低かった音程の鍵盤の楽器に、表において最も近い楽器の鍵盤
  5. ステージ4の2番目にあった鍵盤の楽器に、表において最も近い楽器の鍵盤
B:
  1. 2番目に高い音程の鍵盤
  2. ステージ1の最も低い音程の鍵盤の色の鍵盤
  3. ステージ2で押された鍵盤の音程の鍵盤
  4. ステージ3の青の鍵盤の音程と反対の音程の鍵盤
  5. ステージ3で押された鍵盤と同じ色だったステージ4の鍵盤の位置にある鍵盤
D:
  1. 4番目の鍵盤
  2. ステージ1の3番目にあった鍵盤の色の鍵盤
  3. ステージ2の1番目にあった鍵盤の音程の鍵盤
  4. ステージ1で押された鍵盤と同じ色だったステージ3の鍵盤と同じ位置にある鍵盤
  5. ステージ3で押された鍵盤と同じ色だったステージ2の鍵盤と同じ音程だったステージ4の鍵盤と同じ位置にある鍵盤

注：表における楽器の距離については、同点がある場合はそのうち最も左にある鍵盤を使用する。
