En caso de la creación

¡Que se haga la vida!

  • ¡Combina elementos para crear un nuevo ser!
  • El módulo contiene una pantalla mostrando unos elementos, otra para mostrar el número de día, y otra para decir el tiempo.
  • Cuando vayas creando nuevos elementos, aparecerán nuevos iconos.
  • No obstante, ¡ten cuidado, porque la temperatura afecta a los elementos que vas a combinar, y requerirá a otro para contrarrestarlo!
  • El combinar un par de elementos erróneamente te dará un fallo y además el módulo se reiniciará.

Usando la tabla de abajo, determina el elemento inicial basado en el tiempo del día 1:

Tiempo Elemento
Lluvia Agua
Viento Aire
Ola de calor Fuego
Lluvia de meteoritos Tierra

Basado en la posición del elemento inicial, encuentra el número de permutación pertinente con esta tabla:

Reglas del tiempo:
Mira el pronóstico del tiempo en la parte de arriba a la izquierda del módulo, al lado de "Day #"
Si está lloviendo, toda "Agua" deberá ser sustituida por "Fuego."
Si hace viento, todo "Aire" deberá ser sustituido por "Tierra."
Si hay una ola de calor, todo "Fuego" deberá ser sustituido por "Agua."
Si hay una lluvia de meteoritos, toda "Tierra" deberá ser sustituida por "Aire."
Si está despejado, entonces no hace falta hacer nada. ¡Disfruta del sol!
