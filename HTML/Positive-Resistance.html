On the Subject of Positive-Resistance

What do you mean? I’m totally fine!

There is a number on the module. Pressing the module will decrement the number. When the number reaches 0, press the status light to solve the module.

The module will not solve using the previously shown instructions. Pressing the status light when the number is negative will result in the module going into Reading State.

A timer of 180 seconds will start. Every two seconds, some number X will be added to a number A, where A is the 20-digit number in the middle of the module. X and A are in base-94, and are both truncated to their 20 least significant digits.

Note down all needed information before the timer runs out and the module enters Submitting State. Alternatively, the status light can be pressed to enter this state prematurely.

Convert X from base-94 to base-36 and submit it via Tap Code using the following table, with rows first and columns second. Note that the module will show only the 7 last digits of the full submitted number.

After sending a digit, a timer of 2.5 seconds will start and the text will gradually turn red. If the timer runs out, the last inputted digit will be removed.

To submit the inputted number, press the status light. If the submitted number was correct, the module will solve. If the input was blank, the module will give a fake strike, otherwise a real one. If the input was wrong, the module will fully reset.
