On the Subject of Passwords (Translated)

Unfortunately these passwords may have some special characters in them. That's at least one point towards meeting standard government security requirements.

This module is identical to Passwords, except the possible passwords are in a different language. Check if there is a barcode sticker on the module with label "XX V#", where XX is a two or three letter language code, and # is a digit. If there is no sticker, you are not looking at a translated Passwords module.

Perform the instructions as found on the manual page for Passwords, using the tables below to help find the right password for the right language.
