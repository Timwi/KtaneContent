À propos de l’Alphabet

Vous parlez français ? Vous connaissez l’alphabet ? Alors ça devrait aller.

  1. Utiliser les quatre lettres pour épeler un mot de la banque de mots ci-dessous.
  2. Épeler le mot le plus long avec les lettres données en appuyant sur les boutons correspondants.
  3. Une lettre ne peut être utilisée qu’une seule fois pour épeler un mot.
  4. Si plusieurs mots peuvent être épelés, épeler le mot qui vient en premier dans l’ordre alphabétique, puis le suivant s’il y a suffisamment de lettres restantes.
  5. Si plus aucun mot ne peut être épelé, appuyer sur les boutons restants dans l’ordre alphabétique.
