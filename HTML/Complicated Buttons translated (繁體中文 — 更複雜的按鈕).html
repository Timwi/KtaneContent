關於更複雜的按鈕Complicated Buttons

這些按鈕看起來很奇怪，好消息:我們有說明書!

  1. 每個按鈕都有一、兩個顏色和一個標籤。
  2. 運用下面的文氏圖查找每一個按鈕該不該按。
  3. 按照下面的表單看按鈕該按的順序。如果沒有按鈕需要被按下，請按下順序中第二個按鈕。
  4. 每得到一次Strike，整個模組會重置。
字母說明
P按下按鈕
D不要按按鈕
R如果機身編號有二個或更多重複字母，按下按鈕
S如果炸彈上有串列埠，按下按鈕
B如果電池槽有兩個或更多，按下按鈕

第一個 按鈕的標籤		 0–1 個電池 2–3 個電池 4–5 個電池 6+ 個電池
按按鈕 順序:Press1, 2, 32, 3, 13, 1, 21, 2, 3
Hold2, 1, 33, 2, 11, 3, 22, 3, 1
Detonate3, 1, 21, 2, 32, 1, 33, 1, 2
