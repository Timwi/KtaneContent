On Suiting the Optimization of Simon

Simon has never looked more well suited than he has ever been.

Original Manual

  • Complete 3 stages to disarm the module.
  • For each stage, apply the left button’s color value with the operator of the bowtie that flashed for this stage and the right buttons’s color value, and find its corresponding button.
  • After the first stage, input the answer for all previous stages, then the current stage.
  • Entering an incorrect answer will incur a strike and reset inputs for this stage.
  • If top is G, C, or B, 0 = left, 1 = top, 2 = right
  • Otherwise, 0 = right, 1 = top, 2 = left
