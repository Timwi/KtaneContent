Keep Talking and Nobody Explodes Mod Binaire Cassé

À propos du Binaire Cassé

Ou les Touches Coordonnées, si tu préfères.

Ce module consiste en un écran et cinq boutons. Chacun des trois petits boutons vont répéter une série de 8 chiffres, correspondant à une lettre en code binaire ASCII. Les boutons du bas ont aussi une lettre écrite en code binaire ASCII. Trouver ces lettres en utilisant le tableau ci-dessous pour obtenir une anagramme d’un mot anglais. Déchiffrer le mot et l’entrer dans le module en appuyant sur les boutons correspondant.

Si le mot validé est incorrect, une erreur sera enregistrée et de nouvelles lettres seront affichées.

Si l’écran affiche un mot, il s’agit d’un module différent.
