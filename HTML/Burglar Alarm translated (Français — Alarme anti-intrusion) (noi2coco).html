À propos de l’Alarme anti-intrusion

Pourquoi voudriez-vous voler une bombe ?

Ce module affiche des boutons de 0 à 9, un bouton de désarmement, un bouton de validation et un numéro de module. Pour désarmer le module, trouver le long code à 8 chiffres dans le tableau ci-dessous, appuyer sur le bouton de désarmement (✗) et entrer le code, puis appuyer sur le bouton de validation (✓). Mais il y a une limite de temps. Après avoir appuyé sur le bouton de désarmement, vous disposez de 15 secondes pour saisir le code. Si le temps s’écoule avant que le bon code ne soit saisi, le module génèrera une erreur. Si le mauvais code est saisi, le module génèrera une erreur.
