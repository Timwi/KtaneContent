Keep Talking and Nobody Explodes Mod Button Sequence

CIR: Circle, HEX: Hexagon, SQR: Square.

If both conditions match, hold down the button. Then, the colored LED will light up next to the button. Based on its color you must release the button at a specific point in time:

  • Blue light: release when the countdown timer has a 2 in any position.
  • White light: release when the countdown timer has a 7 in any position.
  • Yellow light: release when the countdown timer has a 3 in any position.
  • Magenta light: release when the countdown timer has a 4 in any position.
  • Cyan light: release when the countdown timer has a 0 in any position.
