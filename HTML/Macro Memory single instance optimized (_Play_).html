On the Subject of Macro Memory

This just feels like Wide Memory now...

  • WARNING: Only use this manual if this is the only Macro Memory module on this bomb. If there are two or more Macro Memory modules on the bomb, use the original manual.
  • The module will activate, showing a number in the display.
  • Use the displayed digit to determine the correct button, and press it to advance the stage of the module.
  • Complete all stages of the module to disarm it.
  • An incorrect press resets the module back to stage 1.
